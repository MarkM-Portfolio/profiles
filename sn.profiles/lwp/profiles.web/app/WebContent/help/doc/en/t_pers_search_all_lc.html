Searching across Lotus Connections

Use the search control in the title bar to search for information across all the IBM® Lotus® Connections features.

About this task
To search all the Lotus Connections features from Profiles, complete the following steps:
  1. From any view in Profiles, enter the term that you want to search for in the Search field in the title bar.
  2. Define the scope of the search by clicking the down arrow next to the Search field and selecting one of these options:
    Table 1. Search options
    Option Description
    All Connections Searches for the text string in all the Lotus Connections features.
    <feature_name> Searches for the text string in an individual feature.
    Advanced Opens the Advanced Search form. Use the form to search for a single term or multiple terms in one, all, or a subset of features.
  3. Click Search.
