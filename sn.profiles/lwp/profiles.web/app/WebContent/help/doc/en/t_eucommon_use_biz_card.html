Using the business card

Use the business card to view people's contact details and get in touch with them.

About this task

The business card provides a useful snapshot of a user's profile information. In addition to displaying a job title and contact details, if your administrator has enabled Sametime® awareness, the card displays the user's current status information.

The business card provides links to the Lotus® Connections features that are used by the person and allows you to perform a number of actions. For example, if your administrator has configured e-mail addresses to display, you can send an e-mail, or you can start an instant chat with them if you have installed the Lotus Connections plug-in for Sametime.

You can access a person's business card by hovering over their name in any of the features and clicking the link that is displayed.

Use the business card to perform the following tasks:

Procedure

  • Access the features associated with a person by clicking the links at the top of the business card. For example, you can access the person's public file library and files that they have shared with you by clicking Files.
  • Add the person as a colleague by selecting More Actions > Invite to My Network, typing an invitation message, and clicking Send invitation.
  • Download and save the person's vCard by selecting More Actions > Download vCard, clicking Save, browsing to a file location, and clicking Save again.
  • Send the person an e-mail by clicking Send Mail.
    Note: This option is not available if your administrator has configured Lotus Connections to prevent e-mail addresses from being displayed.
  • Start an instant chat with the person by clicking Chat.
    Note: You must install the Lotus Connections plug-in for Sametime for this option to display.
  • Start a telephone conference call with the person by clicking Call.
    Note: You must install Sametime Advanced Telephony for this option to display.
