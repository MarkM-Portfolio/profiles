Using the Things in Common widget

Use the Things in Common widget to find out what you have in common with a person. The widget displays when you are viewing other people's profiles.

Before you begin

You must be logged in to see content in the Things in Common widget.

About this task

When you are viewing another person's profile, the Things in Common widget lists the things that you have in common with that person. The widget displays a list of content from IBM® Lotus® Connections organized according to feature. For example, the widget might display a list of the communities that you and the profile owner both belong to, or a list of the tags that you have both used. You can use this information to help you decide if you want to follow that person or add them to your network.

Procedure

Use the Things in Common widget in the following ways.

  • Expand the feature twisties to display content from a particular feature. For example, this content might include communities, discussions (forum threads), blogs, activities, files, bookmarks, tags, or wikis that you and the profile owner have in common.
  • When a content listing interests you, click the title to open it and take a closer look.
