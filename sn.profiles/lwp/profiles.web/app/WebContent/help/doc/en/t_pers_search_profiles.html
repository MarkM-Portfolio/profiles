Performing an advanced profiles search

Perform an advanced search to find profiles by contact information, location, tag, experience, technical knowledge, training, or interest. You can perform an advanced search from any page in Profiles.

About this task

An advanced search filters the results by letting you search multiple profile fields at the same time. It is a more targeted search than a quick search, which searches against a single field at a time. The more values that you provide in these fields, the narrower the search results become.

To perform an advanced profiles search, complete the following steps:

Procedure

  1. Select Profiles Advanced Search from the search control in the title bar.
  2. The administrator can add or remove fields from the Advanced Profiles Search form as needed. Because of this capability, the fields that display in the form can vary. To search for people based on their contact information, type a value into one or more of the following fields:
    Table 1. Advanced search fields
    Field name Words to enter
    Display Name Type the full name of the person that you want to find.
    First Name Type a given name.
    Last Name Type a family name.
    Tags Enter keywords to search for people based on their tags.
    Job Title Type a job title to find the profiles of people who perform that job function.
    About Me Type keywords that describe the type of skills and experience that you are looking for.
    Background Type keywords that describe the kind of background that you are interested in.
    Organization or Company Type the name of an organization or company to find the profiles of employees.
    City Enter a city to find the profiles of people who work at that location.
    State Enter a state to find the profiles of people who work at that location.
    Country Type a country to find the profiles of people who work at that location.
    E-mail address Type a business e-mail address. This option is not available when the configuration prevents e-mail addresses from being displayed.
    Phone Number Type a telephone number. This field searches all five phone number fields in a person's profile.
  3. Click Search.
