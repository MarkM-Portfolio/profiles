Viewing network contacts

Want to see who's connected to who? Use Profiles to view a person's network contacts or check if someone is already part of your network.

About this task

You can view the people in your network and in other people's networks from the Network Contacts page.

To view network contacts for your profile or someone else's profile, complete the following steps:

Procedure

  1. Open the profile page of the person whose network contacts you want to view.
  2. Click Show All in the Network area of the page.
  3. Sort the display using the following options available from the Sort by list:
    Table 1. Display options
    Option Description
    Display Name Lists colleagues in alphabetic order.
    Recent Lists colleagues in order of most recent.
