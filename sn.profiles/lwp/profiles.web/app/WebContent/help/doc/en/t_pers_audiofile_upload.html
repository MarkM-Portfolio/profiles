Uploading an audio file

Edit your profile to upload a recording of your name being pronounced correctly.

About this task

Enhance your profile by adding a recording of yourself saying your name so that others can hear it pronounced correctly.

To upload an audio file to your profile, perform the following steps:

Procedure

  1. From the Edit My Profile page, click Pronunciation.
  2. Click Browse to browse for an audio file recording of your name being clearly spoken with the correct pronunciation. The file must be in WAV or WMA format, and it cannot be larger than 100 KB in size.
  3. Select the file that you want to upload, then click Open.
  4. Click Update to apply your changes.
