Searching profiles

You can search profiles to find people by their contact information, such as name and job title, or by more advanced information, such as their interests, experience, and work projects.

There are a number of different types of search that you can perform in Profiles:
  • Use the search control to perform a basic search based on a single search field. This control is available at the top of every page in Profiles.
  • Search for a person by name or keyword using the Profiles by Name and Profiles by Keyword options available from the quick search control.
  • Use your company directory to perform a more targeted search based on multiple search fields. You can search the company directory from the Directory page, or by selecting Directory Search from the search control at the top of the page.
  • The tag clouds on the My Network and Directory pages allow you to search the profile tags for your network or the entire organization. Click a tag to display a list of people with that tag and show related tags to help you refine your search. To search for profiles that contain a specific tag, click Find a tag, enter a tag in the field provided, and click Search.

For all types of search, the wildcard character (*) is supported. For example, if you enter Ethan * as a search string for names, the results display all profiles for people with the first name Ethan. If you enter 555* as the search string for telephone numbers, the results display all numbers beginning with 555.

  • Performing a quick search
    Use the search control at the top of any page in Profiles for quick searches.
  • Performing an advanced profiles search
    Perform an advanced search to find profiles by contact information, location, tag, experience, technical knowledge, training, or interest. You can perform an advanced search from any page in Profiles.
