Viewing profiles

View profiles to find contact and work-related information for people.

In addition to contact details and work background, a profile provides rich, social information about a person that can help you decide if you want to connect with them. For example, you can see how you are related to the person, the top things you have in common, their latest status updates, and what they've been doing lately in the other applications.

View a person's profile to find the following information:

The Board
The latest posts to the profile owner's board. This information includes the profile owner's recent status messages and comments that other users have posted to the profile owner's board.
Contact information
Typically found on a business card, such as name, job title, company, location, telephone numbers, and e-mail addresses, the profile identifies the person's local time, and, if applicable, an assistant's contact information. Click the business card icon to export the person's contact information as a virtual card file. A virtual card (vCard) is a file that stores basic contact information following the Internet Mail Consortium specification. You can add new contacts by using your e-mail system to import vCard files.
Note: Your administrator can configure IBM® Lotus® Connections to prevent e-mail addresses from being displayed, in which case e-mail addresses do not display in a person's profile or business card.
When Sametime® awareness is enabled, the person's online status is displayed. The Lotus Connections Plug-in for Sametime must be installed for this information to display.
Background
Current® work experience, technical skills, languages spoken, interests, past work experience, and education. This information is a rich-text field composed by the profile owner and includes any information that the profile owner wants to share.
Recent Posts
This section uses tabbed views to display the latest updates that the profile owner has made to other Lotus Connections applications.
Pronunciation
The pronunciation link is displayed when the profile owner has created an audio file and uploaded it to the profile. Click the link to play a recording of the person's name with the correct pronunciation.
What connects us?
A list of the people that link you and the profile owner in Lotus Connections. Viewing the information in the What Connects Us widget is a useful way to see how you are connected to a particular person. For more information, see Using the What Connects Us widget.
Note: You must be logged in to see information about the social path that exists between you and the profile owner.
Reporting structure
Choose one of the following report-to chain view options:
Table 1. Reporting structure views
View Description
Full Report-to Chain Shows the profile owner's position from the profile owner up to the top of the organizational chart.
Same Manager Shows the people who also report to the profile owner's manager.
People Managed Shows the profile owner's position within the full organizational chart, including people that the profile owner manages.
Profile tags
Short keywords used to identify the profile owner's skills, areas of interest, or work activities. These tags can be added by the profile owner or by another person.
Collaborative tools
When profiles are integrated with other collaborative tools, such as IBM Lotus Sametime, you can see the profile owner's presence information indicating whether the person is available to chat.
Network
This area shows a selection of the people in the profile owner's network. You can click See all to display a list of everyone in the person's network.
My Links
External links that profile owners include as part of their profile.
