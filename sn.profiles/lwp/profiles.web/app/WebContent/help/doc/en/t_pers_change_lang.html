Changing the display language

If your administrator has enabled it, you can change the language used in the user interface so that all of the IBM® Lotus® Connections features display in your preferred language.

Before you begin
The feature that allows you to change the language setting in Lotus Connections is disabled by default. Your administrator must enable it. To find out if the feature is enabled, look for your name in the navigation bar. If a language name is displayed beside your name, then the feature is enabled.
Note: Changing the display language used in the Profiles feature does not affect the language of the content that displays in Profiles. For example, if the About Me area of your profile is written in English and you change the Profiles display language to French, the content in the About Me area of your profile still displays in English.
About this task
To change the display language, complete the following steps:
  1. Click the arrow next to the language displayed beside your name in the navigation bar.
  2. Select a language from the list.
    Note: If the list does not include your preferred language, and your preferred language is defined in the locale setting of the Web browser that you are using, select Browser Setting.
Results
The page is refreshed and the user interface is displayed in the selected language. If you do not see a change, refresh the page.
