Searching across Lotus Connections

Use the search control in the title bar to search all the Lotus® Connections features.

Procedure

  1. From any view, type the word or term that you want to search for into the Search field on the title bar.
  2. Define the scope of the search by clicking the Down Arrow next to the Search field, and then selecting one of the following options:
    Option Description
    All Connections Searches for the text string in all the Lotus Connections features.
    <feature_name> Searches for the text string in an individual feature.
    Advanced Opens the Advanced Search page from which you can search for the text string in one, all, or a subset of features and to which you can apply the following filters to the search:
    • Select to search all content or just the content relating to you.
    • Type one or more keywords into the Keywords field to search by keyword.
    • Type one or more tags in the Tags field to search by tag.
    • Type a name or names in the Person field to find content associated with the specified person or people.
    Note: Use commas to separate multiple search terms.
  3. Click the Submit search icon.

    The search results are displayed in the current page. Click the title of a result to open and edit it.
