ביצוע חיפוש מתקדם

בצעו חיפוש מתקדם כדי לאתר פרופילים לפי פרטי קשר, תג, נסיון, ידע טכני, הכשרה או תחומי עניין. תוכלו לבצע חיפוש מתקדם מכל דף ברכיב הפרופילים.

אודות משימה זו

חיפוש מתקדם מסנן את החיפוש באמצעות סריקת שדות מרובים בפרופיל בעת ובעונה אחת. חיפוש זה ממוקד יותר מחיפוש מהיר, המחפש רק בשדה אחד בכל פעם. ככל שתספקו ערכים רבים יותר בשדות אלה, תוצאות החיפוש יהיו ממוקדות יותר.

כדי לבצע חיפוש מתקדם בפרופילים, בצעו את הצעדים האלה:

  1. בחרו חיפוש מתקדם בפרופילים מפקד החיפוש בפס הכותרת.
  2. המנהלן יכול להוסיף או לסלק שדות מהטופס חיפוש מתקדם בפרופילים לפי הצורך. לפיכך, השדות המוצגים יכולים להשתנות. כדי לחפש אדם לפי פרטי הקשר שלו, הקלידו ערך באחד או יותר מהשדות האלה:
    טבלה 1. שדות חיפוש מתקדם
    שם שדה מילים לציון
    שם להצגה הקלידו את השם המלא של האדם שברצונכם לאתר.
    שם פרטי הקלידו שם פרטי.
    שם משפחה הקלידו שם משפחה.
    תגים כדי לחפש אדם לפי פרטי התגים שלו, הקלידו מילות מפתח.
    תיאור תפקיד הקלידו תיאור תפקיד כדי לאתר פרופילים של אנשים המבצעים תפקיד זה.
    מחלקה כדי לחפש אדם העובדה במחלקה מסוימת, הקלידו מילו מפתח.
    אודותי הקלידו מילות מפתח המתארות את סוג הכישורים והנסיון שאתם מחפשים.
    רקע הקלידו מילות מפתח המתארות את סוג הרקע שאתם מחפשים.
    ארגון או חברה הקלידו את שם הארגון או החברו כדי לאתר פרופילים של עובדים.
    עיר הקלידו עיר כדי לאתר פרופילים של אנשים העובדים במיקום זה.
    מדינה הקלידו מדינה (למשל מדינה בארצות הברית) כדי לאתר פרופילים של אנשים העובדים במיקום זה.
    ארץ הקלידו ארץ כדי לאתר פרופילים של אנשים העובדים במיקום זה.
    כתובת דואל ציינו כתובת דואל עסקית. אפשרות זו אינה זמינה אם התצורה מונעת הצגה של כתובות דואל.
    מספר טלפון הקלידו מספר טלפון. שדה זה מחפש בכל חמשת השדות של מספרי הטלפון בפרופיל של אדם.
  3. לחצו על חיפוש.
