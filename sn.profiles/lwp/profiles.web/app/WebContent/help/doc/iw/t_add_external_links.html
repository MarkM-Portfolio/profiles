פרופילים - עזרה

הוספת קישורים חיצוניים לפרופיל שלכם

כדי להרחיב את הפרופיל שלכם, הוסיפו את קישורי הרשת החברתית המועדפים שלכם לדף הפרופיל.

אודות משימה זו
תוכלו להוסיף את קישורי הרשת החברתית המועדפים שלכם לאזור הקישורים שלי בפרופיל שלכם. לדוגמה, אולי תרצו לכלול קישור לפרופיל שלכם באתרי Facebook או LinkedIn כדי שאתם ועמיתיכם תוכלו לגשת בקלות לנתונים החברתיים ממיקום אחד.

כדי להוסיף קישור אל החלק 'הקישורים שלי' בפרופיל שלכם, בצעו את הצעדים האלה:

  1. מהדף הפרופיל שלי, לחצו על הוספת קישור בחלק הקישורים שלי.
  2. הקלידו שם עבור הקישור בשדה שם. תמליל זה מזהה את הקישור בחלק הקישורים שלי.
  3. הקלידו את URL הקישור בשדה קישור.
  4. לחצו על שמירה.
