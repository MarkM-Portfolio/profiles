مساعدة عن ملف المواصفات

ارسال تفاصيل جهة الاتصال

ارسال vCard لشخص لاضافته كجهة اتصال جديدة.

نبذة عن هذه المهمة
وتعد البطاقة الظاهرية Virtual Card (vCard)‎ ملف يقوم بتخزين معلومات الاتصال الأساسية لمستخدم، وهذه المعلومات تتبع مواصفات Internet Mail Consortium. يمكنك اضافة جهات اتصال جديدة باستخدام نظام البريد الالكتروني الخاص بك لاستقبال ملفات vCard. يمكنك أيضا تحميل vCard من بطاقة التعارف لملفات المواصفات.

عند القيام بتحميل vCard، يجب أن تحدد اختيار تكويد الحروف المناسب لوحدة قراءة دليل العناوين الخاص بك. الاختيار المفترض هو Internationalized (UTF-8)، الذي يعد مناسبا لمستخدمي IBM Lotus Notes وMicrosoft Outlook. لكن، يمكن أن يتطلب الأمر تحديد تكويد مختلف اذا كانت الوحدة التابعة للبريد الالكتروني هي Microsoft Outlook Express أو Microsoft Windows Address Book.

لحفظ vCard الخاص بأحد زملاء العمل بسجلاتك الشخصية، اتبع الخطوات التالية:
  1. قم بفتح صفحة ملف المواصفات للشخص الذي تريد اضافته كجهة اتصال.
  2. اضغط تحميل vCard.
  3. حدد تكويد الحروف الذي تريد استخدامه.
  4. اضغط تحميل لحفظ بطاقة vCard الخاصة بالشخص في المكان المحدد.
