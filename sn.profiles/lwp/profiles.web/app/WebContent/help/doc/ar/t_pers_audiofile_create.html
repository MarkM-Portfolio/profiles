تكوين ملف صوتي

يمكنك تكوين تسجيل لاسمك بحيث يستطيع الآخرين سماعه بالنطق الصحيح.

نبذة عن هذه المهمة
لتكوين ملف صوتي على نظام Microsoft Windows 2000 أو Microsoft Windows XP، قم بتنفيذ الخطوات التالية:
  1. حدد Start > All Programs > Accessories > Entertainment > Sound Recorder.
  2. اضغط على الاختيار Record، قم بنطق اسمك بالكامل، بطريقة واضحة كلما أمكن، ثم اضغط على الاختيار Stop. اضغط على الاختيار Play للاستماع الى التسجيل.
  3. تحقق من حجم التسجيل بالضغط على File > Properties. يجب أن يكون حجم البيانات أقل من 100 كيلوبايت.
  4. لاعادة تسجيل الملف الصوتي، اضغط على File > New، وحدد No عند طلب حفظ التسجيل. كرر الخطوة 2 لاعادة التسجيل. اتبع هذه الملاحظات لتحسين جودة التسجيل:
    • اذا لم يمكنك سماع التسجيل، تأكد من أن الميكروفون متاح. حدد Start > All Programs > Accessories > Entertainment > Volume Control، ثم حدد Options > Properties. اختر Recording، وحدد Microphone ثم اضغط OK. قم بسحب مؤشر الصوت الخاص بالميكروفون لأعلى حتى أقصى مستوى له.
    • اذا لم يكن التسجيل واضحا أو اذا كان الملف الناتج كبير جدا (أكثر من 100 كيلوبايت)، قم بتوصيف وحدة تسجيل الصوت. حدد Start > All Programs > Accessories > Entertainment > Sound Recorder. وحدد File > Properties ثم اضغط Convert Now. في المجال Format، حدد PCM وفي المجال Attributes، حدد ‎8.000 kHz‎،‏ ‎8 bit، ‏Mono. اضغط OK ثم OK لاغلاق نافذة Properties.
  5. بعد الحصول على تسجيل واضح ويمكن سماعه وأصغر من 100 كيلوبايت، حدد File > Save As لحفظه. أدخل اسم للملف واختر دليل لحفظ الملف به. قم بحفظ اسم ومكان الملف في مذكرة، بحيث يمكنك استرجاعه عند تحميله بملف المواصفات الخاص بك. يكون نوع الملف المفترض في المجال Save as type هو ‏‎.wav‎‏. قم بقبول الاختيار المفترض. اضغط حفظ.
ما تريد القيام به بعد الآن
لتكوين ملف صوتي بنظام Linux، استخدم أمر التسجيل أو استخدم وظيفة تسجيل الصوت.
