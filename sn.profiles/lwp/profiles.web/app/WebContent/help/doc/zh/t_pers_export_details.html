个人档案帮助

导出详细联系信息

导出某个人的电子名片，以将其添加为新联系人。

关于此任务
电子名片（vCard）是存储用户的基本联系信息的文件，遵循 Internet 邮件协会规范。 您可以通过使用电子邮件系统导入电子名片文件的方式来添加新联系人。您还可以从“个人档案”名片下载电子名片。

下载电子名片时，需要选择适合您的通讯录阅读器的字符编码选项。缺省选项是国际化（UTF-8），该选项适合于 IBM® Lotus Notes® 和 Microsoft® Outlook 的用户。 但是，如果您的电子邮件客户端是 Microsoft Outlook Express™ 或 Microsoft Windows® 通讯簿，那么您可能需要选择其他编码。

要将某个同事的电子名片保存到您的个人记录中，请执行以下步骤。
  1. 打开您想添加为联系人的人员的“个人档案”页面。
  2. 单击下载电子名片。
  3. 选择您要使用的字符编码。
  4. 单击下载将某人的电子名片保存到指定位置。
