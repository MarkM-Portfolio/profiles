个人档案帮助

更改背景信息

编辑个人档案的自我介绍区域，可让他人了解您的技能和以前的工作经历。

关于此任务

使用个人档案的自我介绍区域说明您的工作。提供关于您正在从事的项目、正在使用的技术、正碰到和正解决的问题的详细信息。包括您可能掌握的任何相关技术技能、语言技能或您的兴趣爱好的信息。描述您以前的工作经历、教育和培训情况。

注： 最初，管理员使用贵组织用户目录中的信息创建您的个人档案。由于这些信息来自于组织的人员目录，且不同的目录存储不同的信息，因此您的个人档案中显示的字段可能不同于缺省字段。此外，管理员可能自动填写了某字段以使您不必填写。如果某个字段收集的信息与贵组织无关，管理员还可以将该字段设置为只读。
要更新自我介绍信息，请完成以下步骤：
  1. 从“编辑我的个人档案”页面，单击自我介绍。
  2. 在提供的字段中输入您自己的相关信息。 这些是富文本格式字段，即您可以编排这些字段中文本的格式。您还可以将从其他地方拷贝的文本粘贴到这些字段中，或者添加指向相关 Web 站点的超文本链接。每个字段的内容不可超过 1,500 个字符。
  3. 单击更新。
