个人档案帮助

使用标记

标记是您分配给您的个人档案以标识工作活动、技能或兴趣的描述性关键字。如果您的系统管理员已启用该功能，您就可以向其他人的个人档案分配标记。

分配给您的个人档案的标记显示在“我的个人档案”页面上的标记区域中。 标记描述了您是做什么的以及您关心什么。创建标记时，请使用能代表您的兴趣或专长领域的关键字。标记必须以单个单词的形式输入，或是由连字符或下划线连接的多个词。例如，payroll、event-planning 和 human_resources 都是有效的标记。

为什么应该为个人档案指定标记？

公布您的技能

通过为个人档案指定标记，可以向贵组织中的其他人描述您的技能、工作职位、兴趣和教育情况。

查找人员

单击您自己的个人档案标记中的一个，可以快速查找与您有共同兴趣或者做类似工作的其他人。与您共享相同标记的人员对您来说可能是很好的资源。您能从他们的个人档案获得联系信息，并开始与他们协作。

查找对您进行了标记的人员。单击特定标记的频率数以显示使用该关键字对您进行了标记的人员列表。 再次单击该频率可隐藏此信息。

为什么应该标记别人的个人档案？

定义您的联系

为同事的个人档案指定标记可以帮助您记住您和他有何联系。例如，如果您与他们共同从事某个特定项目，可以选择用该项目的名称来标记他们。 如果您标记别人的个人档案，查看此个人档案的任何人都可看到该标记。

通过标记您自己和他人的个人档案，您与同事可以在组织内更轻松地找到专长。在“个人档案”中添加的所有标记都显示在“个人档案主页”上的组织标记区域中。您可以通过浏览标记云，然后单击您感兴趣的标记，快速找到特定领域内的专家。通过单击查找标记，输入您要查找的标记并单击搜索包含此标记的个人档案可搜索包含特定标记的个人档案。
