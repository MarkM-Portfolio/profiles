个人档案帮助

更改个人档案图像

从“编辑我的个人档案”页面，为自己的个人档案上载最新的照片。

关于此任务

不时地更新图像可以充实自己的个人档案。确保上载适用于业务的图像。

要更新个人档案图像，请执行以下步骤：
  1. 从“编辑我的个人档案”页面，单击照片。
  2. 要编辑个人档案图像，请执行下面的一个操作：
    • 要除去现有照片，请选择除去图像。
    • 要上载新的图像，请执行以下操作：
      1. 单击浏览以查找包含要显示在您的个人档案中的图片的 .jpeg 或 .gif 文件。
        注： 没有文件大小限制。但是，由于图像会在“个人档案”服务器上自动调整大小以适合用户界面中分配的空间，因此请注意，上载的文件越大，服务器处理图像所耗费的时间就越长。
      2. 选择要显示的图像文件，然后单击打开。
      3. 单击更新以保存更改。
    • 要预览图像，请执行以下操作：
      1. 单击浏览以查找要预览的 .jpeg 或 .gif 文件，然后单击打开。
        注： 没有文件大小限制。但是，由于“个人档案”服务器会在预览图像之前对其进行病毒扫描和调整大小，因此请注意，所选的文件越大，服务器预览图像所耗费的时间就越长。
      2. 单击预览图像以查看图像将在个人档案页面上显示的情况。
      3. 单击选择框内部，并拖动它以选择要在个人档案中显示的图像部分。如果想要修改选择区域的大小，可以拖动选择框的边缘以调整其大小。
      4. 单击更新以保存更改。
