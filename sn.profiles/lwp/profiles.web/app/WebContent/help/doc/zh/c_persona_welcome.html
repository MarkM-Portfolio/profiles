个人档案帮助

欢迎使用“个人档案”

“个人档案”提供了组织中人员的目录：如何联系他们，如何了解他们的项目、技能和兴趣。

通过帮助您查找并快速与正确的人取得联系，“个人档案”可促进协作。

提高您的可视性。使用“个人档案”可让组织中的人员知道您是谁、做什么工作、正在做什么项目、正使用什么工具和技术、当前试图解决的问题、正在阅读或撰写什么材料。首先请填写您的个人档案。通过添加丰富的社交信息（例如您的兴趣、爱好以及可表明您专长领域的标记），来帮助他人加深对您的了解。

参与进来。使用“个人档案”可发现组织中谁正在做和您类似的项目，或者谁正在查找类似问题的解决方案，以及他们正用什么资源来简化工作。查明您的合作者是谁、他们的兴趣是什么、他们有什么技能、他们正在发布或者阅读什么材料，并和他们取得联系。查看他们的联系人，使您能够进一步扩展您的网络。

构建您自己的网络。使用个人档案进行有用的联系。找到具有您所需技能和专长的人员，并将他们添加至您的网络。当您建立好网络后，您可以登录“个人档案”主页并查看联系人的状态更新以及时了解其近况。

“个人档案”能帮助您利用贵组织中人员的集体知识，并和其他人协作。

访问“个人档案”功能部件

要登录“个人档案”功能部件，请转至由配置该产品的管理员指定为该功能部件主页的 Web 站点。提供管理员为您创建的用户名和密码。如果要将此功能部件的访问权授予其他人，请与管理员联系；必须首先将此人添加到相应的用户目录，登录才能成功。
