搜索个人档案

您可以搜索个人档案，以根据联系信息（例如姓名和职务）或更高级的信息（例如兴趣、专长和工作项目）来查找人员。

您可在“个人档案”中执行多种不同类型的搜索：
  • 使用搜索控件可根据单个搜索字段执行基本搜索。“个人档案”中每个页面的右上方都提供了此控件。
  • 使用快速搜索控件提供的按姓名搜索个人档案和按关键字搜索个人档案选项，可按姓名或关键字来搜索人员。
  • 单击个人档案高级搜索可使用高级搜索选项，根据多个搜索字段执行更有针对性的搜索。
  • 使用“个人档案主页”上的标记云可搜索组织的所有个人档案标记。单击某个标记可显示具有该标记的人员列表，并显示相关标记以帮助您优化搜索。要搜索包括特定标记的概要文件，请单击查找标记，在提供的字段中输入一个标记并单击搜索包含此标记的概要文件。

所有类型的搜索都支持通配符（*）。 例如，如果输入 Ethan * 作为姓名的搜索字符串，结果将显示名为 Ethan 的人员的所有个人档案。如果输入 555* 作为电话号码的搜索字符串，结果将显示所有以 555 开头的号码。
