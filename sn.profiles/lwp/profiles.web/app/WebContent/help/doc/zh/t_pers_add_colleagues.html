个人档案帮助

将人员添加到您的网络

使用个人档案可创建工作同事和有用联系人的网络。

关于此任务
可以通过向人员发送邀请来邀请他们加入您的网络。被邀请者在接收到消息时，可以接受或忽略该消息。
注： 您无需在“个人档案”中，就可以将某人添加到您的网络中。您可以从任一 IBM® Lotus® Connections 功能部件中的个人档案名片直接联系人员。例如，如果某人的个人档案作为搜索结果的一部分返回，或者如果您与某人是同一个社区的成员，而您想将其添加到您的网络，可以通过使用名片完成此操作。

要将某人添加到您的网络中：

  1. 打开您要添加到网络的人员的个人档案页面。
  2. 单击邀请加入我的网络。 如果尚未登录到“个人档案”功能部件，系统会指引您登录。
  3. 可选： 输入一条邀请该人员加入您网络的消息。
  4. 单击发送邀请。
