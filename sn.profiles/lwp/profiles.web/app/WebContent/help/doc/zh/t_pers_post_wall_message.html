个人档案帮助

向公告板发布消息

您可以在其他人的个人档案公告板区域内发布消息或评论，通过这种方式与您网络中的人员直接沟通。

开始之前
您必须登录“个人档案”才能对您自己或其他人的个人档案发布消息。
关于此任务
所有“个人档案”用户的个人档案页面中都有一个公告板区域，该区域用于显示其 IBM® Lotus® Connections 功能部件中最新的状态消息和更新。您可以通过在同事的个人档案中直接发布消息来与之沟通。

要向某人员的公告板发布消息，请完成以下步骤：

  1. 在该人员的个人档案页面上，单击公告板区域顶部的消息字段的内部（其中显示在此个人档案公告板上撰写消息）。
  2. 输入消息。
  3. 单击发布消息。
结果
您的消息将显示在公告板区域顶部，个人档案所有者和其他用户可在该处进行回复。个人档案所有者将收到一封电子邮件通知，告知其个人档案中有新发布的消息。
