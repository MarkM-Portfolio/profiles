将外部链接添加到个人档案

通过将您收藏的 Web 链接添加到个人档案页面，可扩充您的个人档案。

关于此任务
您可以将收藏的社交网络 Web 链接添加到个人档案中的我的链接区域。例如，您可能想要在 Facebook 或 LinkedIn Web 站点上包含到您的个人档案的链接，以便您和您的同事能够方便地从单个位置访问您的社交信息。

要将链接添加到个人档案中的“我的链接”部分，请完成以下步骤：

  1. 从“我的个人档案”页面，单击我的链接区域中的添加链接。
  2. 在名称字段中输入链接的名称。 此文本标识我的链接区域中的链接。
  3. 在链接字段输入链接 URL。
  4. 单击保存。
