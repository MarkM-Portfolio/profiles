“人员”帮助

本发行版的新增功能

了解有关新增和更新功能的信息，这些功能旨在使您更加方便地使用基本信息来联系他人和介绍自己。

Lotus® Connections 2.0.1

  • 安全方面的改进意味着管理员现在可以在基本信息名片中禁止显示电子邮件地址。

Lotus Connections 2.0

  • 增强了上载图形功能，使您能够将更大的图形上载到您的基本信息，并可将其剪切为所需的外观。
  • 基本信息名片中的改进使您能通过多种方式与同事互动，例如检查他们的 Sametime® 状态、将他们添加到您的网络以及开始与他们进行即时交谈。
  • 改进的标记允许您 使用关键字来标记其他人员，以定义他们与您的联系和使其易于查找。
  • 报告信息现在可告诉您哪些人隶属于同一个经理，谁受谁管理。
  • 改进的搜索功能使您能够按照名字或姓氏的顺序进行搜索。
