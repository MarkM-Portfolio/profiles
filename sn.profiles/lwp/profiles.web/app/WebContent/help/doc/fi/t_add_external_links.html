Profiilit-ohje

Ulkoisten linkkien lisääminen omaan profiiliin

Voit laajentaa profiiliasi lisäämällä profiilisivullesi linkkejä haluamiisi Web-sivuihin.

Voit lisätä sosiaaliseen verkostoitumiseen liittyviä Web-linkkejä profiilisi Omat linkit -alueeseen. Voit esimerkiksi halutessasi lisätä linkin profiiliisi Facebook- tai LinkedIn-sivustossa, jotta sinä ja kollegasi voitte helposti käyttää sosiaalisia tietojasi yhdestä paikasta käsin.

Voit lisätä linkin profiilisi Omat linkit -osaan toimimalla seuraavasti:

  1. Valitse Oma profiili -sivun Omat linkit -alueesta Lisää linkki.
  2. Kirjoita linkin nimi Nimi-kenttään. Tämä teksti näkyy linkissä Omat linkit -alueessa.
  3. Kirjoita linkin URL-osoite Linkki-kenttään.
  4. Napsauta Tallenna-painiketta.
