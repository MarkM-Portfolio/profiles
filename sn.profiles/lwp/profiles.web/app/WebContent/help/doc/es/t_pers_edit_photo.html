Cómo cambiar la imagen del perfil

Puede subir una fotografía actualizada de su perfil desde la página Editar Mi perfil.

Por qué y cuándo se efectúa esta tarea

Puede mejorar el perfil actualizando su imagen periódicamente. Asegúrese de subir una imagen apropiada para la empresa.

Para actualizar la imagen del perfil, realice los siguientes pasos:
  1. En la página Editar mi perfil, pulse Fotografía.
  2. Para editar la imagen del perfil, haga lo siguiente:
    • Para eliminar la fotografía existente, seleccione Eliminar imagen.
    • Para subir una nueva imagen:
      1. Pulse Examinar para buscar un archivo .jpeg o .gif que contenga una fotografía suya para mostrar en el perfil.
        Nota: No hay restricciones de tamaño de archivo. No obstante, como se cambia automáticamente el tamaño de la imagen en el servidor de perfiles para que se ajuste al espacio asignado en la interfaz de usuario, tenga en cuenta que cuanto mayor sea el archivo subido, más tiempo tardará el servidor en procesar la imagen.
      2. Seleccione el archivo de imagen que desea mostrar y, a continuación, pulse Abrir.
      3. Pulse Actualizar para guardar los cambios.
    • Para ver una visualización previa de una imagen:
      1. Pulse Examinar para ubicar el archivo .jpeg o .gif que desea previsualizar y, a continuación, pulse Aceptar.
        Nota: No hay restricciones de tamaño de archivo. No obstante, como el servidor de perfiles cambia el tamaño de la imagen al previsualizarla, tenga en cuenta que cuanto mayor sea el archivo subido, más tiempo tardará el servidor en procesar la imagen.
      2. Pulse Imagen de visualización previa para ver la imagen como aparecerá en la página del perfil.
      3. Pulse y arrastre el recuadro de selección para seleccionar la sección de la imagen que desea mostrar en el perfil. Puede arrastrar el borde del recuadro de selección, si desea cambiar el tamaño del área seleccionada.
      4. Pulse Actualizar para guardar los cambios.
