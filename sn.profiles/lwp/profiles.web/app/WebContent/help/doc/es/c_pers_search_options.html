Búsqueda en los perfiles

Puede buscar en los perfiles para encontrar personas por su información de contacto, como el nombre y el cargo, o por información más avanzada, como sus intereses, experiencia y proyectos de trabajo.

Hay varios tipos de búsqueda diferentes que puede ejecutar en Perfiles:
  • Utilice el control de búsqueda para realizar una búsqueda básica basada en un campo de búsqueda individual. Este control está disponible en la parte superior derecha de todas las páginas de Perfiles.
  • Busque una persona por nombre o palabra clave mediante las opciones Perfiles por nombre y Perfiles por palabra clave disponibles desde el control de búsqueda rápida.
  • Pulse Búsqueda avanzada de perfiles para utilizar las opciones de búsqueda avanzada y realizar una búsqueda más específica basada en varios campos de búsqueda.
  • Utilice el grupo de etiquetas de la página de inicio Perfiles para buscar todas las etiquetas de perfiles de su organización. Pulse una etiqueta para mostrar una lista de personas con esa etiqueta y mostrar etiquetas relacionadas que le ayudarán a perfeccionar su búsqueda. Para buscar perfiles que contengan una etiqueta determinada, pulse Buscar etiquetas, indique una en el campo proporcionado y pulse Buscar perfiles con esta etiqueta.

El carácter comodín (*) está soportado en todos los tipos de búsqueda. Por ejemplo, si escribe Juan * como cadena de búsqueda de nombres, los resultados mostrarán todos los perfiles de personas con Juan como primer nombre. Si especifica 555* como cadena de búsqueda de números de teléfono, los resultados mostrarán todos los números que empiecen por 555.
