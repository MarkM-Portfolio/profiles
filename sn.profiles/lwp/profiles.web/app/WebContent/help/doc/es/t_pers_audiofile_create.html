Creación de un archivo de audio

Cree una grabación de usted mismo diciendo su nombre para que los demás lo oigan pronunciado correctamente.

Por qué y cuándo se efectúa esta tarea
Para crear un archivo de audio en un sistema Microsoft® Windows® 2000 o Microsoft Windows XP, efectúe los pasos siguientes:
  1. Seleccione Inicio > Todos los programas > Accesorios > Entretenimiento > Grabadora de sonidos.
  2. Pulse el botón Grabar, diga su nombre completo, hablando lo más claro posible, y, a continuación, pulse el botón Parar. Pulse el botón Reproducir para escuchar la grabación.
  3. Para comprobar el tamaño de la grabación, pulse Archivo > Propiedades. El tamaño de datos debe ser inferior a 100 KB.
  4. Para volver a grabar el archivo de audio, pulse Archivo > Nuevo y seleccione No cuando se le solicite si desea guardar la grabación. Repita el paso 2 para volver a grabar. Siga estos consejos para mejorar la calidad de la grabación:
    • Si la grabación es inaudible, compruebe que el micrófono está habilitado. Seleccione Inicio > Todos los programas > Accesorios > Entretenimiento > Control de volumen y, a continuación, seleccione Opciones > Propiedades. Elija Grabación, seleccione Micrófono y, a continuación, pulse Aceptar. Arrastre el graduador de volumen del micrófono hasta el nivel máximo.
    • Si la grabación no es clara o si el archivo es demasiado largo (superior a 100 KB), configure la grabadora de sonido. Seleccione Inicio > Todos los programas > Accesorios > Entretenimiento > Grabadora de sonidos. Seleccione Archivo > Propiedades y, a continuación, pulse Convertir ahora. En el campo Formato, seleccione PCM y en el campo Atributos, seleccione 8,000 kHz, 8 bit, Mono. Pulse Aceptar y, a continuación, pulse Aceptar para cerrar la ventana Propiedades.
  5. Cuando haya creado una grabación que sea clara, audible e inferior a 100 KB, seleccione Archivo > Guardar como para guardarlo. Escriba un nombre para el archivo y especifique un directorio donde guardarlo. Anote el nombre y la ubicación, para recuperarlo cuando suba el archivo a su perfil. El tipo de archivo predeterminado en el campo Guardar como tipo es .wav. Acepte el valor predeterminado. Pulse Guardar.
Qué hacer a continuación
Para crear un archivo de audio en un sistema Linux®, utilice el mandato record o utilice un programa de utilidad de grabación de sonido.
