Ayuda de Perfiles

Adición de enlaces externos a su perfil

Amplíe el perfil añadiendo sus enlaces web preferidos a la página del perfil.

Por qué y cuándo se efectúa esta tarea
Puede añadir sus enlaces web de redes sociales preferidos al área Mis enlaces del perfil. Por ejemplo, puede incluir un enlace a su perfil en los sitios web Facebook o LinkedIn para que usted y sus compañeros de trabajo puedan acceder fácilmente a sus datos sociales desde una ubicación individual.

Para añadir un enlace a la sección Mis enlaces del perfil, siga estos pasos:

  1. En la página Mi perfil, pulse Añadir enlace en el área Mis enlaces.
  2. Escriba un nombre para el enlace en el campo Nombre. Este texto identifica el enlace en el área Mis enlaces.
  3. Escriba el URL del enlace en el campo Enlace.
  4. Pulse Guardar.
