Comprobación de las actualizaciones de estado

Para mantener actualizada su red compruebe regularmente las actualizaciones de estado.

Por qué y cuándo se efectúa esta tarea
El área Actualizaciones de estado en la página de inicio Perfiles muestra las últimas actualizaciones de estado desde diferentes perfiles. Puede filtrar la muestra para ver las actualizaciones de estado de personas conectadas con usted o sus propias actualizaciones de estado. También puede actualizar su mensaje de estado personal desde esta página y añadir un comentario a los mensajes de estado de otras personas.

Para comprobar las actualizaciones de estado, siga estos pasos:

  1. Abra la página de inicio Perfiles.
  2. En el área Actualizaciones de estado, realice una de las siguientes acciones:
    • Pulse Actualizaciones de estado en Mi red para mostrar la última lista de actualizaciones de las personas de su red.
    • Pulse Mis actualizaciones de estado para mostrar sus propias actualizaciones de estado.
  3. Opcional: Pulse Añadir comentario para hacer un comentario a un mensaje de estado determinado.
