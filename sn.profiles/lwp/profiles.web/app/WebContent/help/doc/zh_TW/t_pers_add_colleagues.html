人員資訊說明

新增人員至您的網路

使用人員資訊，可以建立同事及實用聯絡人的網路。

執行這項作業的原因和時機
您可以傳送邀請給人員來邀請他們加入您的網路。當受邀者收到訊息時，可以接受或忽略它。
註: 您不需要位於「人員資訊」，即可將某人新增至網路。 您可以在任何 IBM® Lotus® Connections 功能中，從人員資訊名片直接連接至人員。例如，如果傳回人員的人員資訊作為搜尋結果的一部分，或者如果您與某人都是相同社群的成員，而且想要將這些人新增至您的網路，則您可以使用名片來進行這個動作。

若要將某人新增至網路，請執行下列動作：

  1. 開啟您要新增至網路之人員的人員資訊頁面。
  2. 按一下邀請加入我的網路。 如果您尚未登入「人員資訊」功能，將指引您登入。
  3. 選擇性的: 鍵入邀請人員加入網路的訊息。
  4. 按一下傳送邀請。
