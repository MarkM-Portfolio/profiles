人員資訊說明

本版本新增的功能？

提供讓您更容易使用人員資訊以聯繫人員，並為了讓大家認識您而設計的新增及更新功能。

Lotus® Connections 2.5

  • 「佈告欄」功能可讓您與網路中的人員保持同步，並且不需要離開人員資訊即可與他們溝通。
  • 您可以張貼個人狀態訊息，讓網路中的人員隨時掌握您的動態。
  • 「人員資訊」首頁上的組織標籤區域，會顯示「人員資訊」中新增的所有標籤。瀏覽標籤雲並按一下感興趣的標籤，找出具有特定專業的人員。
  • 變更顯示語言可讓您置換瀏覽器的語言設定，並以您選擇的語言來檢視人員資訊。

Lotus Connections 2.0.1

  • 安全改良項目表示管理者現在可以避免電子郵件位址顯示在「人員資訊」名片上。

Lotus Connections 2.0

  • 上傳圖形的加強功能可讓您將更大的影像檔上傳至人員資訊，並將它裁剪為您想要的樣子。
  • 人員資訊名片的改良項目可讓您透過多種方式與同事互動，例如，檢查其 Sametime® 狀態、將同事新增至網路，以及啟動與他們的即時會談。
  • 改良的標示可讓您利用關鍵字來標示其他人員，以定義他們與您的連線，並使他們易於搜尋。
  • 報告目標資訊現在會告訴您誰共用管理程式，以及誰由誰管理。
  • 改良的搜尋功能可讓您依名字或姓氏的順序來進行搜尋。
