人員資訊示範 script

這是「人員資訊」示範文字。

執行這項作業的原因和時機

使用「人員資訊」，在組織中搜尋技能及專業人員。建置同事社交網路，並追蹤其最新張貼內容。

本示範會告訴您如何進行下列動作：
  • 按名稱搜尋人員
  • 新增人員至您的網路
  • 在某人的佈告欄張貼訊息
  • 對佈告欄訊息進行評論
  • 更新狀態訊息
  • 編輯人員資訊
  1. 使用搜尋控制項來迅速地搜尋某人。
  2. 傳回的搜尋結果具有如位置、工作職稱及電話號碼等資訊。
  3. 若要進一步瞭解人員，請將游標移至其名稱上方，並按一下以檢視其名片。
  4. 即現式名片會顯示照片及有關人員的其他資訊鏈結。
  5. 使用名片可以使用電子郵件來聯絡人員。
  6. 按一下其他動作以存取聯絡人員的其他選項。
  7. 按一下人員資訊鏈結，以顯示人員的完整人員資訊。
  8. 人員資訊頁面會顯示人員的相關資訊，包括工作職稱、辦公室位置及詳細的聯絡資訊。
  9. 如果可以使用報告目標鏈，請使用它來取得較佳的組織結構。
  10. 按一下邀請加入我的網路，將人員新增至您的網路。
  11. 您也可以儲存人員的 vCard，將他們新增為您的聯絡人。
  12. 在人員的佈告欄上張貼訊息，即可直接與該人員溝通。
  13. 輸入訊息，然後按一下張貼訊息。
  14. 向下捲動至最近張貼內容區域，檢視人員在其他功能中所做的最近更新項目。
  15. 按一下我的人員資訊，您就可以隨時存取自己的人員資訊。
  16. 花一些時間檢查您的資訊是否保持最新，並檢視佈告欄上的最新訊息。
  17. 按一下新增評論，以回應訊息。
  18. 更新您的狀態訊息，讓人員掌握您的動態。
  19. 按一下您正在做什麼？，輸入訊息，然後按一下儲存。
  20. 若要更新人員資訊詳細資料，請按一下編輯我的人員資訊標籤。
  21. 修改不同標籤上的欄位內容，然後按一下更新以儲存變更。
  22. 現在您已有一些基本概念，請登入並連線！
  23. 謝謝您觀賞本示範。如需相關資訊，請查看「人員資訊」說明頁面。
