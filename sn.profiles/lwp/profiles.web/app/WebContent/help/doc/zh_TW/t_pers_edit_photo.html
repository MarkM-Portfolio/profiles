人員資訊說明

變更您的人員資訊影像檔

從編輯我的人員資訊頁面中，上傳您的人員資訊的最新照片。

執行這項作業的原因和時機

不時更新影像來加強您的人員資訊。請務必上傳適合公司的影像檔。

若要更新您的人員資訊影像檔，請執行下列步驟：
  1. 從編輯我的人員資訊頁面中，按一下照片。
  2. 若要編輯您的人員資訊影像檔，請執行下列其中一項：
    • 若要移除現有的照片，請選取移除影像檔。
    • 若要上傳新的影像檔：
      1. 按一下瀏覽，找出 .jpeg 或 .gif 檔案，其中包含您要在人員資訊中顯示的圖片。
        註: 沒有檔案大小限制。不過，因為影像會在「人員資訊」伺服器上自動調整大小，以符合使用者介面中配置的空間，因此請注意，您上傳的檔案愈大，伺服器處理影像所花費的時間就會愈長。
      2. 選取您要顯示的影像檔，然後按一下開啟。
      3. 按一下更新以儲存您的變更。
    • 若要預覽影像檔：
      1. 按一下瀏覽，找出您要預覽的 .jpeg 或 .gif 檔案，然後按一下開啟。
        註: 沒有檔案大小限制。不過，因為影像會由「人員資訊」伺服器進行病毒掃描及調整大小以進行預覽，因此請記住，您選取的檔案越大，伺服器預覽影像所花費的時間就會越長。
      2. 按一下預覽影像以檢視影像，如同它將出現在您的人員資訊頁面一般。
      3. 在選擇框內按一下，然後拖曳該選擇框，以選取您要在人員資訊中顯示的影像區段。如果要修改選擇區域的大小，您可以拖曳選擇框的邊緣來調整其大小。
      4. 按一下更新以儲存您的變更。
