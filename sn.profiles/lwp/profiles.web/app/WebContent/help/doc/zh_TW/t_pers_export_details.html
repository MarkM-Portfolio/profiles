人員資訊說明

匯出聯絡詳細資料

匯出人員的 vCard，將他們新增為新的聯絡人。

執行這項作業的原因和時機
虛擬卡片 (vCard) 是一個檔案，它依循「網際網路郵件聯盟 (Internet Mail Consortium)」規格，來儲存使用者的基本聯絡資訊。 您可以使用電子郵件系統來匯入 vCard 檔案，以新增聯絡人。您也可以從「人員資訊」名片中下載 vCard。

下載 vCard 時，您需要選取通訊錄讀取器適用的字元編碼選項。 預設選項是國際化 (UTF-8)，該選項適用於 IBM® Lotus Notes® 及 Microsoft® Outlook 的使用者。 然而，如果您的電子郵件用戶端是 Microsoft Outlook Express™ 或 Microsoft Windows® Address Book，則可能需要選取不同的編碼。

若要將同事的 vCard 儲存至您的個人記錄，請執行下列步驟。
  1. 開啟您要新增為聯絡人之人員的人員資訊頁面。
  2. 按一下下載 vCard。
  3. 選取您要使用的字元編碼。
  4. 按一下下載，將人員的 vCard 儲存至指定的位置。
