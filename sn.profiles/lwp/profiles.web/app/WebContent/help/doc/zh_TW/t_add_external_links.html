人員資訊說明

新增外部連結至您的人員資訊

將我的最愛 Web 鏈結新增至您的人員資訊頁面來延伸您的人員資訊。

執行這項作業的原因和時機
您可以將我的最愛社交網路 Web 鏈結新增至人員資訊的我的鏈結區域。例如， 您可能想要包括 Facebook 或 LinkedIn 網站上您的人員資訊的鏈結，讓您及您的同事可以從單一位置輕易地存取您的社交資料。

若要將鏈結新增至您的人員資訊的「我的鏈結」區段，請完成下列步驟：

  1. 從我的人員資訊頁面中，按一下我的鏈結區域中的新增鏈結。
  2. 在名稱欄位中，鍵入鏈結的名稱。 這個文字會識別我的鏈結區域中的鏈結。
  3. 在鏈結欄位中，鍵入鏈結 URL。
  4. 按一下儲存。
