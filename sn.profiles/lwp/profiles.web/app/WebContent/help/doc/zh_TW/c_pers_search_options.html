人員資訊說明

搜尋人員資訊

您可以搜尋人員資訊，按人員的聯絡資訊 (例如名稱和工作職稱) 或按更進階的資訊 (例如興趣、經驗和工作專案) 來搜尋人員。

有一些不同類型的搜尋，您可以在「人員資訊」中執行它們：
  • 使用搜尋控制項，根據單一搜尋欄位來執行基本搜尋。您可以在「人員資訊」中每一個頁面的右上方使用這個控制項。
  • 使用快速搜尋控制項中可用的依名稱的人員資訊及依關鍵字的人員資訊選項，依名稱或關鍵字搜尋人員。
  • 按一下人員資訊進階搜尋，使用進階搜尋選項，並根據多個搜尋欄位來執行更接近目標的搜尋。
  • 使用人員資訊首頁上的標籤雲，搜尋組織中的所有人員資訊標籤。 按一下標籤以顯示與該標籤相關聯的人員清單，並顯示相關的標籤，協助您精簡搜尋。若要搜尋含有特定標籤的人員資訊，按一下尋找標籤、在提供的欄位中輸入標籤，然後按一下搜尋含有此標籤的人員資訊。

所有類型的搜尋均支援萬用字元 (*)。例如，如果您輸入 Ethan * 作為名稱的搜尋字串，結果會顯示名字為 Ethan 的人員的所有人員資訊。 如果您輸入 555* 作為電話號碼的搜尋字串，結果會顯示開頭為 555 的所有號碼。
