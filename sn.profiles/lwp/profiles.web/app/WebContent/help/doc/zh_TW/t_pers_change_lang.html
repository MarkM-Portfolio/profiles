人員資訊說明

變更顯示語言

如果管理者已啟用此功能，您就可以變更使用者介面中使用的語言，讓所有 IBM® Lotus® Connections 功能都以您偏好的語言顯示。

開始之前
預設會停用允許您變更 Lotus Connections 語言設定的功能。 您的管理者必須啟用它。若要瞭解是否已啟用功能，請在導覽列中尋找您的名稱。如果語言名稱顯示在您的名稱旁邊，則表示已啟用功能。
註: 變更「人員資訊」功能中使用的顯示語言，並不會影響「人員資訊」中顯示內容的語言。 例如，如果人員資訊的關於我區域是以英文撰寫，且將「人員資訊」顯示語言變更為法文，人員資訊的關於我區域仍會以英文顯示。
執行這項作業的原因和時機
若要變更顯示語言，請完成下列步驟：
  1. 在導覽列中，按一下在您名稱旁邊顯示之語言旁邊的箭頭。
  2. 從清單中選取語言。
    註: 如果清單沒有包含您偏好的語言，且您偏好的語言定義於正在使用的 Web 瀏覽器語言環境設定，請選取瀏覽器設定。
結果
頁面會重新整理，而且使用者介面會以選取的語言顯示。如果沒有看到變更，請重新整理頁面。
