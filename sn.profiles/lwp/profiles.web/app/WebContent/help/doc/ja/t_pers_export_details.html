連絡先の詳細のエクスポート

ある人の vCard をエクスポートして、新しい連絡先として追加できます。

このタスクについて
仮想カード (vCard) とは、IMC (Internet Mail Consortium) の仕様に従い、基本的な連絡先情報を保管するファイルのことです。E メール・システムを使用して vCard ファイルをインポートすることにより、新規連絡先を追加することができます。vCard をプロフィールのビジネス・カードからダウンロードすることもできます。

vCard をダウンロードする場合、アドレス帳リーダーに適した文字エンコード・オプションを選択する必要があります。 デフォルトのオプションは、「国際化 (UTF-8)」です。IBM® Lotus Notes® や Microsoft® Outlook を使用する場合は、このオプションを選択します。 ただし、E メール・クライアントとして Microsoft Outlook Express™ や Microsoft Windows® Address Book を使用する場合は、別のエンコードを選択する必要があります。

ワーク・メンバーの vCard を個人レコードに保存するには、以下のステップを実行します。
  1. 連絡先として追加するユーザーのプロフィール・ページを開きます。
  2. 「vCard のダウンロード」をクリックします。
  3. 使用する文字エンコードを選択します。
  4. 「ダウンロード」をクリックして、このユーザーの vCard を指定の場所に保存します。
