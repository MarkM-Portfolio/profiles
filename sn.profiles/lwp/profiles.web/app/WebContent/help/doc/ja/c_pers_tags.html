プロフィールのヘルプ

タグの使用

タグは、アクティビティー、スキル、または関心分野を特定するために、自分のプロフィールに割り当てる説明的なキーワードです。システム管理者がタグ付けを許可した場合、ユーザーは他の人のプロフィールにタグを割り当てることができます。

自分のプロフィールに割り当てたタグは、「マイ・プロフィール」ページの「タグ」領域に表示されます。 タグは自分の作業や自分が関心を寄せているものを記述したものです。タグを作成するときは、自分の関心分野や専門領域を示すキーワードを使用します。タグは単一の単語として入力するか、ハイフンまたは下線で接続された複数の単語として入力する必要があります。例えば、payroll、event-planning、および human_resources はすべて有効なタグです。

なぜプロフィールにタグをつける必要があるのでしょうか。

自分のスキルを公開する

プロフィールにタグを割り当てることで、自分のスキル、職務上の役割、関心分野、および受けた教育について、組織内の人に紹介します。

人を検索する

自分のプロフィール・タグをクリックすれば、関心分野を共有する人や、似た作業を行っている人を素早く検索することができます。同じタグを共有する人は、多くの場合、自分にとってよいリソースとなります。その人のプロフィールから連絡先情報を取得して、コラボレーションを始めることができます。

タグに表示されている数字をクリックすると、自分のプロフィールに対してそのタグを割り当てている人のリストが表示されます。 この数字をもう一度クリックすると、リストが非表示になります。

なぜ他の人のプロフィールにタグをつける必要があるのでしょうか。

関係を定義する

ワーク・メンバーのプロフィールにタグを付けることで、その人との関係を明確にできます。例えば、特定のプロジェクトに一緒に取り組んでいるワーク・メンバーに、そのプロジェクト名のタグをつけることができます。他の人のプロフィールにタグをつけると、誰がその人のプロフィールを表示してもそのタグが見えます。

自分のプロフィールおよび他の人のプロフィールにタグを付けると、自分またはワーク・メンバーが組織内で専門知識を持つ人を探し出すのが簡単になります。プロフィールに追加されたすべてのタグは、「プロフィール・ホーム」ページの「組織タグ」領域に表示されます。タグ・クラウドを参照し、自分が関心を持つタグをクリックすることにより、その領域の専門知識を持つ人を素早く見つけることができます。また、特定のタグが定義されているプロフィールを検索することもできます。その場合は「検索タグ」 をクリックし、検索したいタグを入力して 「このタグを含むプロフィールの検索」をクリックします。
