プロフィールのヘルプ

経歴情報の変更

自分のプロフィールの「私について」領域を編集すると、自分のスキルやこれまでの職務経歴を他のユーザーに知らせることができます。

このタスクについて

自分のプロフィールの「私について」領域を使用して、自分に関する情報を編集します。作業中のプロジェクト、使用中のテクノロジー、および検討中または解決中の問題について、詳細を記述します。また、関連する技術スキル、言語スキル、あるいは関心分野についての情報を編集することもできます。以前の職務経験、教育、および研修について記述します。

注: 管理者は最初に、組織のユーザー・ディレクトリーの情報からプロフィールを作成します。この情報は組織の職員名簿から取り出したもので、名簿が異なれば格納される情報の種類も異なるため、プロフィールに表示されるフィールドは、デフォルトのフィールドと異なる場合があります。さらに、管理者は、各自が入力しなくても済むようにフィールドに自動入力できます。また、組織に関係ない情報を収集する場合は、フィールドを読み取り専用に設定することもできます。
「私について」領域の情報を更新するには、以下のステップを実行します。
  1. 「マイ・プロフィールの編集」ページから、「私について」をクリックします。
  2. 表示されたフィールドに、自分に関する情報を入力します。 これらのフィールドはリッチ・テキスト・フィールドであるため、入力したテキストの書式を設定することができます。また、別の場所からコピーしたテキストをフィールドに貼り付けたり、関連する Web サイトのハイパーテキスト・リンクを追加することもできます。各フィールドのテキストは、1,500 文字以内で入力する必要があります。
  3. 「更新」をクリックします。
