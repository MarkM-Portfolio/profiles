プロフィールのヘルプ

プロフィールへようこそ

プロフィールは、組織内の人と連絡を取る方法や、その人のプロジェクト、スキル、および関心分野を検索する方法を示す、組織のユーザー・ディレクトリー (名簿) を提供するものです。

プロフィールがあれば、適切な人を素早く検索して連絡を取ることが容易になり、コラボレーション (共同作業) を促進します。

可視性の向上。 プロフィールを使用することで、自分がどんな人で、何をしているのか、また作業中のプロジェクト、使用中のツールおよびテクノロジー、現在解決しようとしている問題、および現在読んでいたり書いていたりするものについて、組織内の人に知らせることができます。まず自分のプロフィールを作成してください。自分の関心分野、趣味、専門分野を示すタグなどの豊富なソーシャル情報を追加すると、他の人があなたのことをよく知ることができます。

関与を高める。 プロフィールを使用することで、自分が関わっているプロジェクトと似たプロジェクトに組織内の誰が取り組んでいるのか、または誰が同様の問題の解決策を見つけようとしているか、および作業を簡素化するためにどんなリソースを使用しているかを発見することができます。自分のワーク・メンバーは誰なのか、彼らの関心分野やスキル、何を公開して何を読んでいるのかを調べて、彼らと連絡を取ることができます。彼らが誰と関連しているかがわかると、自分のネットワークをさらに広げることができます。

独自のネットワークを構築する。 プロフィールを使用することで、有益な関係を築くことができます。あなたが求めるスキルと専門知識を持つ人を探し出して、ネットワークに追加できます。ネットワークを確立したら、「プロフィール・ホーム」ページにログインして自分と関連がある人の状況の更新情報を参照することで、その人についての最新情報を把握できます。

プロフィールによって、組織内の人が持つ集合的な知識を活用できるようになり、他人との共同作業が可能になります。

プロフィール機能へのアクセス

プロフィール機能にログインするには、製品の構成を行った管理者によってその機能のホーム・ページとして指定された Web サイトにアクセスします。管理者がユーザー向けに作成したユーザー名とパスワードを指定します。その機能へのアクセス権限を他の人にも与える場合は、管理者に連絡してください。まずその人を適切なユーザー・ディレクトリーに追加して、正常にログインできるようにする必要があります。
