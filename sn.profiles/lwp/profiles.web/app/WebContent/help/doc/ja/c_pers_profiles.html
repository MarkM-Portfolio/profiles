プロフィールのヘルプ

プロフィールの表示

プロフィールには、ある個人についての連絡先および業務関連の情報が記載されています。

個人のプロフィールを表示すると、以下の情報を参照することができます。

連絡先情報
ビジネス・カードに通常よく見られる、名前、職務上の肩書き、会社、住所、電話番号、および E メール・アドレスなどの情報です。プロフィールによって、その人のがいる場所の現地時間と、場合によってはアシスタントの連絡先情報も特定できます。ビジネス・カード・アイコンをクリックして、その個人の連絡先情報を仮想カード・ファイルとしてエクスポートします。仮想カード (vCard) は、IMC (Internet Mail Consortium) の仕様に従って、基本的な連絡先情報を保管するファイルのことです。E メール・システムを使用して vCard ファイルをインポートすることにより、新規連絡先を追加することができます。
注: 管理者は、E メール・アドレスが表示されないように IBM® Lotus® Connections を構成することができます。これにより、E メール・アドレスは個人のプロフィールまたはビジネス・カードに表示されなくなります。
Lotus Sametime® の認識機能が有効に設定されている場合は、ユーザーのオンライン状況が表示されます。この情報を表示するには、Lotus Sametime 用の Lotus Connections プラグインをインストールする必要があります。
発音
「発音」リンクは、プロフィールの所有者が音声ファイルを作成して、プロフィールにアップロードした場合に表示されます。このリンクをクリックすると、その人の名前が正しい発音で再生されます。
レポート・チェーン
以下のビュー・オプションから 1 つ選択します。

ビュー 説明
フル・レポート・チェーン プロフィール所有者から組織図のトップまでの中で、プロフィール所有者の位置を示します。
同じ組織 プロフィール所有者の管理者に対して報告する別の人を表示します。

私について
現在の職務内容、技術スキル、話すことができる言語、関心のある分野、これまでの職務経歴、および受けた教育についての情報です。この情報はプロフィール所有者によって作成されるリッチ・テキスト・フィールドであり、プロフィール所有者が共有を望む任意の情報が含まれます。
ボード
プロフィール所有者のボードへの最新の投稿です。この情報には、プロフィール所有者の最近の状況メッセージと、他のユーザーがプロフィール所有者のボードに投稿したコメントが含まれます。
最近の投稿
このセクションでは、Lotus Connections のその他の機能に対してプロフィール所有者が行った最新の更新内容を、タブ付きのビューを使用して表示します。
プロフィール・タグ
プロフィール所有者のスキル、関心分野、または業務活動を特定するために使用する短いキーワードです。これらのタグは、プロフィール所有者または他のユーザーが追加できます。
コラボレーション・ツール
プロフィールが IBM Lotus Sametime などの別のコラボレーション・ツールと統合されている場合、プロフィール所有者がチャットできるかどうかを示す在席情報を確認できます。
マイ・ネットワーク
このセクションには、プロフィール所有者のネットワーク内にいる、選択された一部の人が表示されます。「すべて表示」をクリックすると、そのユーザーのネットワーク内にいるすべての人のリストが表示されます。
マイ・リンク
プロフィール所有者が自身のプロフィールの一部として組み込む外部リンクです。
