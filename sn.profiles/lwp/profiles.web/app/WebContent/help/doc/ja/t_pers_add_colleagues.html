プロフィールのヘルプ

ネットワークにユーザーを追加

プロフィールを使用して、有益な連絡先とワーク・メンバーのネットワークを作成できます。

このタスクについて
自分のネットワークへの参加を呼びかけるには、その相手に招待を送信します。招待メッセージを受け取った人は、それを受け入れるか無視するかを選択できます。
注: 誰かをネットワークに追加する場合、自分がプロフィール内にいなくてもかまいません。IBM® Lotus® Connections の各機能で、プロフィール・ビジネス・カードから他のユーザーに直接連絡を取ることができます。例えば、ある人のプロフィールが検索結果の一部として返された場合、または自分が誰かと同じコミュニティーのメンバーである場合、ビジネス・カードを使用してその人を自分のネットワークに追加することができます。

他の人を自分のネットワークに追加するには、以下の手順を実行します。

  1. ネットワークに追加したい人のプロフィール・ページを開きます。
  2. 「マイ・ネットワークに招待」をクリックします。 プロフィールの機能にログインしていない場合は、ログインするように促されます。
  3. オプション: その人に自分のネットワークに参加することを勧誘するメッセージを入力します。
  4. 「招待の送信」をクリックします。
