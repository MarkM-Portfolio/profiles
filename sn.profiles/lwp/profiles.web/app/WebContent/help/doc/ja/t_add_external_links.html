プロフィールに外部リンクを追加する

プロフィール・ページにお気に入りの Web リンクを追加することにより、プロフィールを拡張できます。

このタスクについて
プロフィールの「マイ・リンク」領域に、お気に入りのソーシャル・ネットワーキングの Web リンクを追加できます。例えば、Facebook や LinkedIn などの Web サイトに自分のプロフィールへのリンクを組み込むことにより、自分やワーク・メンバーが 1 つの場所から容易にソーシャル・データにアクセスできるようになります。

自分のプロフィールの「マイ・リンク」セクションにリンクを追加するには、以下のステップを実行します。

  1. 「マイ・プロフィール」ページの「マイ・リンク」領域で、「リンクの追加」をクリックします。
  2. 「名前」フィールドにリンクの名前を入力します。 入力したリンク名は、「マイ・リンク」領域に表示されます。
  3. 「リンク」フィールドにリンク URL を入力します。
  4. 「保存」をクリックします。
