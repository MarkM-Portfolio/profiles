Eksportowanie danych kontaktowych

Użytkownik może wyeksportować plik vCard osoby, aby dodać ją jako nowy kontakt.

O tym zadaniu
Karta wirtualna (vCard) jest plikiem, w którym zapisano podstawowe informacje kontaktowe użytkownika zgodnie ze specyfikacją Internet Mail Consortium. Nowe kontakty można dodać, używając systemu poczty elektronicznej w celu zaimportowania plików vCard. Plik vCard można także pobrać z wizytówki komponentu Profile.

Podczas pobierania pliku vCard należy wybrać opcję kodowania znaków odpowiednią dla czytnika książki adresowej. Domyślną opcją jest Międzynarodowe (UTF-8). Jest to opcja odpowiednia dla użytkowników programów IBM Lotus Notes i Microsoft Outlook. Jednak w przypadku, gdy klientem poczty elektronicznej jest program Microsoft Outlook Express lub Microsoft Windows Address Book, być może będzie konieczne wybranie innego kodowania.

Aby zapisać plik vCard znajomej osoby u siebie, wykonaj następujące kroki:
  1. Otwórz stronę profilu osoby, która ma zostać dodana jako kontakt.
  2. Kliknij opcję Pobierz plik vCard.
  3. Wybierz pożądane kodowanie znaków.
  4. Kliknij opcję Pobierz, aby zapisać plik vCard tej osoby w podanym miejscu.
