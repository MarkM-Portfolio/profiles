Βοήθεια για τα Προφίλ

Αλλαγή των πληροφοριών ιστορικού

Τροποποιήστε την περιοχή Προσωπικές πληροφορίες του προφίλ σας για να ενημερώσετε τις πληροφορίες σχετικά με τις δεξιότητές σας και την επαγγελματική εμπειρία σας.

Σχετικά με αυτή την εργασία

Χρησιμοποιήστε την περιοχή Προσωπικές πληροφορίες του προφίλ σας για να περιγράψετε τις ασχολίες και τα ενδιαφέροντά σας. Καταχωρήστε πληροφορίες σχετικά με τα έργα στα οποία συμμετέχετε, τις τεχνολογίες που χρησιμοποιείτε και τα προβλήματα που έχετε συναντήσει ή προσπαθείτε να επιλύσετε. Συμπεριλάβετε πληροφορίες που περιγράφουν τις δεξιότητες και την τεχνική κατάρτιση που διαθέτετε, τις γλώσσες που μιλάτε ή τα ενδιαφέροντά σας. Περιγράψτε την επαγγελματική εμπειρία σας, τις σπουδές σας και τα εκπαιδευτικά προγράμματα που έχετε παρακολουθήσει.

Σημείωση: Ένας διαχειριστής αρχικά δημιουργεί το προφίλ σας χρησιμοποιώντας πληροφορίες από τον κατάλογο χρηστών του οργανισμού σας. Επειδή οι πληροφορίες προέρχονται από τον κατάλογο προσωπικού του οργανισμού και διαφορετικοί κατάλογοι αποθηκεύουν διαφορετικές πληροφορίες, τα πεδία που εμφανίζονται στο προφίλ σας πιθανόν να διαφέρουν από τα προεπιλεγμένα πεδία. Επιπλέον, ο διαχειριστής μπορεί να συμπληρώσει αυτόματα ένα πεδίο ώστε να μη χρειαστεί να το συμπληρώσετε εσείς. Επίσης, ο διαχειριστής μπορεί να ορίσει ένα πεδίο ως μόνο για ανάγνωση αν συλλέγει πληροφορίες που δεν είναι σχετικές με τον οργανισμό σας.
Για να ενημερώσετε τις προσωπικές σας πληροφορίες, εκτελέστε την παρακάτω διαδικασία:
  1. Από τη σελίδα Τροποποίηση του προφίλ μου, επιλέξτε Προσωπικές πληροφορίες.
  2. Καταχωρήστε τις πληροφορίες που θέλετε στα αντίστοιχα πεδία. Τα πεδία προσωπικών πληροφοριών είναι πεδία εμπλουτισμένου κειμένου. Ως εκ τούτου, μπορείτε να μορφοποιήσετε το κείμενο που θα καταχωρήσετε, να επικολλήσετε κείμενο που έχετε αντιγράψει από κάποια άλλη πηγή ή να προσθέσετε διασυνδέσεις υπερκειμένου που παραπέμπουν σε σχετικούς δικτυακούς τόπους. Το περιεχόμενο αυτών των πεδίων δεν μπορεί να υπερβαίνει τους 1500 χαρακτήρες.
  3. Πατήστε Ενημέρωση.
