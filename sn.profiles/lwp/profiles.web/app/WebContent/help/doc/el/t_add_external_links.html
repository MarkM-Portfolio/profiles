Βοήθεια για τα Προφίλ

Προσθήκη εξωτερικών διασυνδέσεων στο προφίλ σας

Επεκτείνετε το προφίλ σας προσθέτοντας τις προτιμώμενες διαδικτυακές διασυνδέσεις σας στη σελίδα του προφίλ σας.

Σχετικά με αυτή την εργασία
Μπορείτε να προσθέσετε τις αγαπημένες σας διαδικτυακές διασυνδέσεις στην περιοχή Οι διασυνδέσεις μου στο προφίλ σας. Για παράδειγμα, μπορείτε να συμπεριλάβετε στο προφίλ σας μια διασύνδεση για το δικτυακό τόπο Facebook ή LinkedIn ώστε να είναι εύκολη η πρόσβαση στα δεδομένα σας από μία τοποθεσία.

Για να προσθέσετε μια διασύνδεση στην ενότητα Οι διασυνδέσεις μου, εκτελέστε την παρακάτω διαδικασία:

  1. Μεταβείτε στη σελίδα Το προφίλ μου και επιλέξτε Προσθήκη διασύνδεσης στην περιοχή Οι διασυνδέσεις μου.
  2. Στο πεδίο Όνομα πληκτρολογήστε ένα όνομα για τη διασύνδεση. Αυτό το κείμενο προσδιορίζει τη διασύνδεση στην περιοχή Οι διασυνδέσεις μου.
  3. Στο πεδίο Διασύνδεση πληκτρολογήστε τη διεύθυνση URL της διασύνδεσης.
  4. Επιλέξτε Αποθήκευση.
