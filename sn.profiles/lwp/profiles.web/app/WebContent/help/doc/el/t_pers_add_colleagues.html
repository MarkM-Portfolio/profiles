Βοήθεια για τα Προφίλ

Προσθήκη προσώπων στο δίκτυό σας

Χρησιμοποιήστε την υπηρεσία Προφίλ για να δημιουργήσετε ένα δίκτυο συναδέλφων και χρήσιμων επαφών.

Σχετικά με αυτή την εργασία
Μπορείτε να στείλετε προσκλήσεις σε άλλους χρήστες για να τους καλέσετε να συμμετάσχουν στο δίκτυό σας. Όταν οι χρήστες λάβουν την πρόσκληση, μπορούν να την αποδεχτούν ή να την αγνοήσουν.
Σημείωση: Δεν χρειάζεται να εργάζεστε στην υπηρεσία Προφίλ για να προσθέσετε κάποιον στο δίκτυό σας. Μπορείτε να συνδεθείτε με κάποιο πρόσωπο απευθείας από την επιχειρηματική κάρτα προφίλ σε οποιαδήποτε λειτουργία του IBM Lotus Connections. Για παράδειγμα, αν θέλετε να προσθέσετε στο δίκτυό σας ένα πρόσωπο το προφίλ του οποίου συμπεριλαμβάνεται στα αποτελέσματα μιας αναζήτησης ή αν ανήκετε στην ίδια κοινότητα με ένα πρόσωπο που θέλετε να προσθέσετε στο δίκτυό σας, μπορείτε να το κάνετε χρησιμοποιώντας την επαγγελματική κάρτα.

Για να προσθέσετε ένα πρόσωπο στο δίκτυό σας, εκτελέστε την παρακάτω διαδικασία:

  1. Ανοίξτε τη σελίδα προφίλ του προσώπου που θέλετε να προσθέσετε στο δίκτυό σας.
  2. Πατήστε Πρόσκληση στο δίκτυό μου. Αν δεν είστε συνδεδεμένοι στην υπηρεσία Προφίλ, θα σας ζητηθεί να συνδεθείτε.
  3. Προαιρετικά: Καταχωρήστε ένα μήνυμα με το οποίο θα προσκαλείτε το πρόσωπο να γίνει μέλος του δικτύου σας.
  4. Πατήστε Αποστολή πρόσκλησης.
