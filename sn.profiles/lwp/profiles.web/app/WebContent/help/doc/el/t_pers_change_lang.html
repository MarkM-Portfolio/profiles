Βοήθεια για τα Προφίλ

Αλλαγή γλώσσας περιβάλλοντος χρήστη

Αν ο διαχειριστής έχει ενεργοποιήσει αυτή τη δυνατότητα, μπορείτε να αλλάξετε τη γλώσσα του περιβάλλοντος χρήστη ώστε όλες οι λειτουργίες του IBM Lotus Connections να εμφανίζονται στη γλώσσα που προτιμάτε.

Πριν ξεκινήσετε
Η δυνατότητα αλλαγής της γλώσσας εμφάνισης στο Lotus Connections είναι απενεργοποιημένη από προεπιλογή. Η δυνατότητα πρέπει να ενεργοποιηθεί από το διαχειριστή. Μπορείτε να διαπιστώσετε αν αυτή η δυνατότητα είναι ενεργοποιημένη κοιτάζοντας το όνομά σας στη γραμμή πλοήγησης. Αν δίπλα στο όνομά σας εμφανίζεται η τρέχουσα γλώσσα, η δυνατότητα αλλαγής της γλώσσας είναι ενεργοποιημένη.
Σημείωση: Η αλλαγή της γλώσσας του περιβάλλοντος χρήστη στην υπηρεσία Προφίλ δεν επηρεάζει τη γλώσσα του εμφανιζόμενου περιεχομένου. Για παράδειγμα, αν το περιεχόμενο της περιοχής Προσωπικές πληροφορίες στο προφίλ σας είναι γραμμένο στα Αγγλικά, η αλλαγή της γλώσσας του περιβάλλοντος χρήστη από Αγγλικά σε Γαλλικά δεν θα επηρεάσει τη γλώσσα του περιεχομένου της περιοχής Προσωπικές πληροφορίες που θα εξακολουθήσει να εμφανίζεται στα Αγγλικά.
Σχετικά με αυτή την εργασία
Για να αλλάξετε τη γλώσσα του περιβάλλοντος χρήστη, εκτελέστε τις ακόλουθες ενέργειες:
  1. Πατήστε στο βέλος δίπλα από την τρέχουσα επιλογή γλώσσας που εμφανίζεται δίπλα από το όνομά σας στη γραμμή πλοήγησης.
  2. Επιλέξτε την επιθυμητή γλώσσα από τη λίστα.
    Σημείωση: Αν η γλώσσα της προτίμησής σας δεν περιλαμβάνεται στη λίστα αλλά έχει οριστεί στις ρυθμίσεις γλώσσας του προγράμματος πλοήγησης, επιλέξτε Ρύθμιση προγράμματος πλοήγησης.
Αποτελέσματα
Η σελίδα θα ανανεωθεί ώστε το περιβάλλον χρήστη να εμφανίζεται στη γλώσσα που επιλέξατε. Αν η σελίδα δεν ανανεωθεί αυτόματα, ανανεώστε τη χρησιμοποιώντας την κατάλληλη επιλογή.
