Βοήθεια για τα Προφίλ

Αποδοχή προσκλήσεων

Αποδεχτείτε τις προσκλήσεις που λαμβάνετε για να γίνετε μέλος σε νέα δίκτυα και να προσθέσετε νέες επαφές στο προσωπικό σας δίκτυο.

Σχετικά με αυτή την εργασία

Αν ο διαχειριστής έχει ενεργοποιήσει τη δυνατότητα αποστολής ειδοποιήσεων μέσω e-mail για την υπηρεσία Προφίλ, θα λαμβάνετε μια πρόσκληση όποτε προστίθεστε στο δίκτυο ενός προσώπου. Το μήνυμα e-mail περιέχει μια διασύνδεση με την καρτέλα Προσκλήσεις. Μεταβείτε σε αυτή την καρτέλα για να αποδεχτείτε ή να απορρίψετε την πρόσκληση.

Αν η αποστολή ειδοποιήσεων μέσω e-mail δεν έχει ενεργοποιηθεί, μπορείτε να δείτε τις προσκλήσεις που λαμβάνετε μεταβαίνοντας απευθείας στην καρτέλα Προσκλήσεις στη σελίδα Επαφές δικτύου.
Σημείωση: Αν αγνοήσετε μια πρόσκληση, δεν θα σταλεί καμία ειδοποίηση στο πρόσωπο που σας κάλεσε.

Ανάλογα με την κατάσταση της λειτουργίας ειδοποίησης μέσω e-mail, εκτελέστε μία από τις ακόλουθες διαδικασίες για να αποδεχθείτε μια πρόσκληση:

  1. Αν η αποστολή ειδοποιήσεων μέσω e-mail είναι ενεργοποιημένη:
    1. Πατήστε στη διασύνδεση στο μήνυμα e-mail για να ανοίξετε την καρτέλα Προσκλήσεις στη σελίδα Επαφές δικτύου. Σε αυτή τη σελίδα εμφανίζονται όλες οι εκκρεμείς προσκλήσεις.
    2. Επιλέξτε Αποδοχή για να αποδεχθείτε την πρόσκληση και να συνδεθείτε στο δίκτυο του προσώπου που σας προσκάλεσε.
  2. Αν η αποστολή ειδοποιήσεων μέσω e-mail είναι απενεργοποιημένη:
    1. Στη σελίδα Το προφίλ μου, πατήστε Το δίκτυό μου.
    2. Μεταβείτε στην καρτέλα Προσκλήσεις για να δείτε όλες τις εκκρεμείς προσκλήσεις.
    3. Επιλέξτε Αποδοχή για να αποδεχθείτε μια πρόσκληση και να συνδεθείτε στο δίκτυο του προσώπου που σας προσκάλεσε.
