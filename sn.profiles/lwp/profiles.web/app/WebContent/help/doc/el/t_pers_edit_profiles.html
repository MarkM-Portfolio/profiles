Βοήθεια για τα Προφίλ

Τροποποίηση του προφίλ σας

Τροποποιήστε το προφίλ σας για να προσθέσετε ή να ενημερώσετε πληροφορίες για το εαυτό σας, για να αλλάξετε τη φωτογραφία σας ή για να προσθέσετε ένα αρχείο προφοράς.

Σχετικά με αυτή την εργασία

Ένας διαχειριστής αρχικά δημιουργεί το προφίλ σας χρησιμοποιώντας πληροφορίες από τον κατάλογο χρηστών του οργανισμού σας. Επειδή οι πληροφορίες προέρχονται από τον κατάλογο προσωπικού του οργανισμού και διαφορετικοί κατάλογοι αποθηκεύουν διαφορετικές πληροφορίες, τα πεδία που εμφανίζονται στο προφίλ σας πιθανόν να διαφέρουν από τα προεπιλεγμένα πεδία. Επιπλέον, ο διαχειριστής μπορεί να συμπληρώσει αυτόματα ένα πεδίο ώστε να μη χρειαστεί να το συμπληρώσετε εσείς. Επίσης, ο διαχειριστής μπορεί να ορίσει ένα πεδίο ως μόνο για ανάγνωση αν συλλέγει πληροφορίες που δεν είναι σχετικές με τον οργανισμό σας.

Για να τροποποιήσετε το προφίλ σας, εκτελέστε την παρακάτω διαδικασία:
  1. Επιλέξτε Τροποποίηση του προφίλ μου.
  2. Ανάλογα με το είδος των αλλαγών που θέλετε να κάνετε, επιλέξτε μία από τις παρακάτω καρτέλες:
    Πίνακας 1. Καρτέλες τροποποίησης προφίλ
    Καρτέλα Εργασία τροποποίησης
    Πληροφορίες επικοινωνίας Τροποποιήστε την τοποθεσία του γραφείου, τις λεπτομέρειες επικοινωνίας, τον επαγγελματικό τίτλο ή τη ζώνη ώρας. Προσθέστε ή αλλάξτε τη διασύνδεση του προσωπικού ιστολογίου σας.
    Προσωπικές πληροφορίες Ενημερώστε τις λεπτομέρειες του ιστορικού εργασίας ή του προσωπικού ιστορικού σας.
    Φωτογραφία Ενημερώστε ή αφαιρέστε την τρέχουσα φωτογραφία στο προφίλ σας.
    Προφορά Ενημερώστε ή αφαιρέστε ένα αρχείο ήχου στο οποίο το όνομά σας εκφωνείται με τη σωστή προφορά.
  3. Όταν κάνετε τις κατάλληλες αλλαγές, πατήστε Ενημέρωση για να τις εφαρμόσετε. Εάν ενημερώσετε τα περιεχόμενα ενός πεδίου και στη συνέχεια μεταβείτε σε άλλη σελίδα, θα σας ζητηθεί να αποθηκεύσετε τις αλλαγές σας ή να τις ακυρώσετε.
