Βοήθεια για τα Προφίλ

Εξαγωγή λεπτομερειών επικοινωνίας

Εξαγάγετε την κάρτα vCard ενός προσώπου για να τον προσθέσετε ως νέα επαφή.

Σχετικά με αυτή την εργασία
Μια εικονική κάρτα (vCard) είναι ένα αρχείο που αποθηκεύει βασικές πληροφορίες επικοινωνίας για ένα χρήστη σύμφωνα με τις προδιαγραφές του Internet Mail Consortium. Μπορείτε να προσθέσετε νέα πρόσωπα στις επαφές σας εισάγοντας αρχεία vCard με το σύστημα e-mail που χρησιμοποιείτε. Μπορείτε επίσης να μεταφορτώσετε την κάρτα vCard με τις πληροφορίες ενός προσώπου από την επαγγελματική κάρτα του προφίλ του.

Όταν πρόκειται να μεταφορτώσετε μια κάρτα vCard, πρέπει να επιλέξετε την κατάλληλη κωδικοποίηση χαρακτήρων για το πρόγραμμα ανάγνωσης βιβλίου διευθύνσεων που χρησιμοποιείτε. Η προεπιλεγμένη κωδικοποίηση Διεθνής (UTF-8) είναι κατάλληλη για τα προγράμματα IBM Lotus Notes και Microsoft Outlook. Ωστόσο, αν χρησιμοποιείτε το Microsoft Outlook Express ή το Microsoft Windows Address Book, μπορεί να απαιτείται η επιλογή διαφορετικής κωδικοποίησης.

Για να αποθηκεύσετε την κάρτα vCard ενός συναδέλφου σας στις προσωπικές επαφές σας, εκτελέστε την παρακάτω διαδικασία.
  1. Ανοίξτε τη σελίδα προφίλ του προσώπου που θέλετε να προσθέσετε ως επαφή.
  2. Πατήστε Μεταφόρτωση vCard.
  3. Επιλέξτε την κατάλληλη κωδικοποίηση χαρακτήρων.
  4. Πατήστε Μεταφόρτωση για να αποθηκεύσετε την κάρτα vCard του προσώπου στη θέση που θα ορίσετε.
