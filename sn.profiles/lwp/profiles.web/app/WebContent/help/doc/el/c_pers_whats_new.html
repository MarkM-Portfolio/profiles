Βοήθεια για τα Προφίλ

Νέες δυνατότητες και λειτουργίες στην τρέχουσα έκδοση

Η παρούσα έκδοση περιλαμβάνει νέες ή βελτιωμένες λειτουργίες και δυνατότητες που απλοποιούν τη χρήση της υπηρεσίας Προφίλ και την επικοινωνία μεταξύ των χρηστών της.

Lotus Connections 2.5

  • Ο Πίνακας σάς επιτρέπει να παρακολουθείτε τα πρόσωπα στο δίκτυό σας και να επικοινωνείτε μαζί τους χωρίς να κλείσετε το προφίλ σας.
  • Μπορείτε να δημοσιεύσετε ένα προσωπικό μήνυμα κατάστασης για να ενημερώσετε τα μέλη του δικτύου σας σχετικά με τις τρέχουσες ασχολίες σας.
  • Στην περιοχή Προσδιοριστικά οργανισμού στην αρχική σελίδα της υπηρεσίας Προφίλ εμφανίζονται όλα τα προσδιοριστικά που έχουν προστεθεί στην υπηρεσία Προφίλ. Εξερευνήστε τη συλλογή προσδιοριστικών και πατήστε σε ένα προσδιοριστικό που σας ενδιαφέρει για να δείτε όλα τα πρόσωπα που έχουν συσχετιστεί με αυτό το προσδιοριστικό.
  • Η δυνατότητα αλλαγής της γλώσσας του περιβάλλοντος χρήστη επιτρέπει την παράκαμψη των προεπιλεγμένων ρυθμίσεων γλώσσας του προγράμματος πλοήγησης και την επιλογή της γλώσσας στην οποία θα εμφανίζεται το περιβάλλον χρήστη της υπηρεσίας Προφίλ.

Lotus Connections 2.0.1

  • Βελτιώσεις στη λειτουργία ασφάλειας επιτρέπουν στο διαχειριστή να εμποδίζει την παρουσίαση διευθύνσεων e-mail σε επαγγελματικές κάρτες της υπηρεσίας Προφίλ.

Lotus Connections 2.0

  • Οι βελτιωμένες δυνατότητες μεταφόρτωσης γραφικών επιτρέπουν τη μεταφόρτωση μια μεγαλύτερης εικόνας στο προφίλ σας καθώς και την περικοπή της.
  • Βελτιώσεις στη λειτουργία της επαγγελματικής κάρτας της υπηρεσίας Προφίλ σάς επιτρέπουν να επικοινωνήσετε με τους συναδέλφους σας με διάφορους τρόπους. Για παράδειγμα, μπορείτε να ελέγξετε την κατάσταση σύνδεσής τους στο Sametime, να τους προσθέσετε στο δίκτυό σας ή να ξεκινήσετε μια συνομιλία μαζί τους.
  • Οι βελτιωμένες δυνατότητες ορισμού προσδιοριστικών επιτρέπουν τον ορισμό προσδιοριστικών για άλλα πρόσωπα με σκοπό τον ευκολότερο εντοπισμό τους και τον ορισμό της σχέσης σας μαζί τους.
  • Με τις πληροφορίες αλυσίδας αναφοράς μπορείτε να δείτε ποιοι αναφέρονται στον ίδιο διευθυντή καθώς και ποιοι αναφέρονται σε ποιους.
  • Οι βελτιωμένες λειτουργίες αναζήτησης επιτρέπουν την αναζήτηση με βάση το όνομα ή το επώνυμο.
