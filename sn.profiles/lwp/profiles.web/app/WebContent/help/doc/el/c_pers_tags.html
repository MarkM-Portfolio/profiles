Βοήθεια για τα Προφίλ

Χρήση προσδιοριστικών

Ένα προσδιοριστικό είναι μια περιγραφική λέξη-κλειδί την οποία μπορείτε να συσχετίσετε με το προφίλ σας για να δηλώσετε ένα ιδιαίτερο χαρακτηριστικό του (π.χ. εργασιακές δραστηριότητες, δεξιότητες ή ενδιαφέροντα). Αν ο διαχειριστής έχει ενεργοποιήσει τη δυνατότητα αυτή, μπορείτε να συσχετίσετε προσδιοριστικά με το προφίλ άλλων προσώπων.

Τα προσδιοριστικά τα οποία συσχετίζετε με το προφίλ σας, εμφανίζονται στην περιοχή Προσδιοριστικά στη σελίδα Το προφίλ μου. Τα προσδιοριστικά περιγράφουν με τι ασχολείστε και ποια είναι τα ενδιαφέροντά σας. Όταν δημιουργείτε προσδιοριστικά, χρησιμοποιήστε λέξεις-κλειδιά που περιγράφουν τα ενδιαφέροντά σας ή τους τομείς εξειδίκευσής σας. Τα προσδιοριστικά πρέπει να αποτελούνται από μία ή περισσότερες λέξεις που συνδέονται με παύλα ή υπογραμμή. Για παράδειγμα, τα προσδιοριστικά μισθοδοσία, προγραμματισμός-συμβάντος, και ανθρώπινο_δυναμικό θεωρούνται έγκυρα προσδιοριστικά.

Γιατί πρέπει να ορίσετε προσδιοριστικά για το προφίλ σας;

Δημοσιοποίηση των δεξιοτήτων σας

Αντιστοιχίζοντας προσδιοριστικά με το προφίλ σας, μπορείτε να περιγράψετε τις δεξιότητες, τον εργασιακό σας ρόλο, τα ενδιαφέροντα και την μόρφωσή σας.

Εύρεση προσώπων

Πατήστε σε ένα από τα προσδιοριστικά του προφίλ σας για να βρείτε γρήγορα άλλα πρόσωπα με τα οποία έχετε κοινά ενδιαφέροντα ή εργασία. Τα πρόσωπα με τα οποία έχετε κοινά προσδιοριστικά πιθανόν να αποτελέσουν καλές πηγές πληροφοριών. Μπορείτε να πάρετε τις πληροφορίες επικοινωνίας από τα προφίλ τους και να αρχίσετε τη συνεργασία μαζί τους.

Βρείτε ποια πρόσωπα σας έχουν προσθέσει προσδιοριστικά. Πατήστε στον αριθμό εμφανίσεων ενός προσδιοριστικού για να δείτε μια λίστα με τα πρόσωπα που έχουν χρησιμοποιήσει αυτό το προσδιοριστικό για σας. Πατήστε ξανά στον αριθμό εμφανίσεων για να κλείσετε τη λίστα.

Γιατί πρέπει να ορίσετε προσδιοριστικά για το προφίλ άλλου χρήστη;

Προσδιορισμός των σχέσεών σας

Με τον ορισμό προσδιοριστικών στο προφίλ ενός συναδέλφου μπορείτε να επισημάνετε τη σχέση σας με το συγκεκριμένο πρόσωπο. Για παράδειγμα, εάν συνεργαστήκατε μαζί του για ένα συγκεκριμένο έργο, μπορείτε να ορίσετε ένα προσδιοριστικό με το όνομα του έργου. Όταν ορίσετε ένα προσδιοριστικό στο προφίλ άλλου προσώπου, το προσδιοριστικό θα εμφανίζεται σε όλους τους χρήστες που ανατρέχουν στο προφίλ.

Η συσχέτιση προσδιοριστικών με το προφίλ σας και το προφίλ άλλων προσώπων διευκολύνει την αναζήτηση πληροφοριών στον οργανισμό σας και τον εντοπισμό προσώπων με εξειδικευμένες γνώσεις και δεξιότητες. Όλα τα προσδιοριστικά που προστίθενται μέσω της υπηρεσίας Προφίλ εμφανίζονται στην περιοχή Προσδιοριστικά οργανισμού στην αρχική σελίδα της υπηρεσίας. Μπορείτε να εντοπίσετε εύκολα πρόσωπα που εξειδικεύονται σε ένα συγκεκριμένο τομέα επιλέγοντας ένα αντίστοιχο προσδιοριστικό από τη συλλογή προσδιοριστικών. Μπορείτε επίσης να αναζητήσετε προφίλ που περιέχουν ένα συγκεκριμένο προσδιοριστικό. Για να αναζητήσετε προφίλ με βάση ένα προσδιοριστικό, πατήστε Εύρεση προσδιοριστικών, καταχωρήστε το προσδιοριστικό και πατήστε Αναζήτηση προφίλ που περιέχουν αυτό το προσδιοριστικό.
