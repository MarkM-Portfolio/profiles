Βοήθεια για τα Προφίλ

Ορισμός προσδιοριστικών σε πρόσωπα

Χρησιμοποιήστε λέξεις-κλειδιά για να ορίσετε προσδιοριστικά στα προφίλ των προσώπων που διαθέτουν δεξιότητες και εμπειρίες που σας ενδιαφέρουν.

Πριν ξεκινήσετε
Εάν ο διαχειριστής του συστήματος ενεργοποιήσει τη δυνατότητα ορισμού προσδιοριστικών σε πρόσωπα, μπορείτε να ορίσετε προσδιοριστικά τόσο για εσάς όσο και για άλλα πρόσωπα. Για να δημιουργήσετε ή να τροποποιήσετε προσδιοριστικά, πρέπει να είστε συνδεδεμένος.
Σχετικά με αυτή την εργασία
Μπορείτε να ορίσετε προσδιοριστικά για πρόσωπα χρησιμοποιώντας τη φόρμα καταχώρησης προσδιοριστικών στη σελίδα προφίλ. Δεν είναι δυνατή η απόκρυψη των προσδιοριστικών ή ο ορισμός τους ως ιδιωτικών. Οι χρήστες μπορούν να παρακολουθούν ποιοι ορίζουν προσδιοριστικά για αυτούς. Όταν ένας χρήστης πατήσει στον αριθμό που εμφανίζεται δίπλα σε ένα προσδιοριστικό, θα εμφανιστεί η λίστα με τα πρόσωπα που έχουν χρησιμοποιήσει αυτό το προσδιοριστικό για το συγκεκριμένο χρήστη.

Για να ορίσετε ένα προσδιοριστικό για ένα πρόσωπο:

  1. Ανοίξτε τη σελίδα προφίλ του προσώπου για το οποίο θέλετε να ορίσετε προσδιοριστικά.
  2. Καταχωρήστε ένα ή περισσότερα προσδιοριστικά στο πεδίο Προσδιοριστικά στην αριστερή πλαϊνή γραμμή. Η λειτουργία προ-πληκτρολόγησης προβλέπει το προσδιοριστικό που θέλετε να καταχωρήσετε με βάση την ομοιότητά του με τα υπάρχοντα προσδιοριστικά.
  3. Πατήστε Προσθήκη προσδιοριστικού(-ών) σε αυτό το προφίλ για να προσθέσετε τα προσδιοριστικά στο προφίλ του προσώπου.
