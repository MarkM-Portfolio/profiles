Βοήθεια για τα Προφίλ

Καλωσορίσατε στην υπηρεσία Προφίλ

Η υπηρεσία Προφίλ παρέχει έναν κατάλογο των προσώπων που ανήκουν σε έναν οργανισμό. Στο προφίλ κάθε προσώπου συγκεντρώνονται χρήσιμες πληροφορίες σχετικά με την ταυτότητά του, τον τρόπο επικοινωνίας μαζί του και τα έργα, τις δεξιότητες και τα ενδιαφέροντά του.

Η υπηρεσία Προφίλ διευκολύνει τη συνεργασία σε έναν οργανισμό αφού επιτρέπει στους χρήστες να εντοπίζουν τους κατάλληλους ανθρώπους και να επικοινωνούν μαζί τους εύκολα και γρήγορα. Χρησιμοποιήστε την υπηρεσία Προφίλ για να παρουσιάσετε τον εαυτό σας, να γνωρίσετε τους άλλους και να βρείτε τους συνεργάτες που χρειάζεστε.

Προβληθείτε. Δημιουργήστε το δικό σας προφίλ για να γνωστοποιήσετε στους άλλους χρήστες ποιος είστε, με τι ασχολείστε, για ποια έργα δουλεύετε, ποια εργαλεία και ποιες τεχνολογίες χρησιμοποιείτε, τι προβλήματα προσπαθείτε να λύσετε καθώς και τι διαβάζετε ή γράφετε. Ξεκινήστε συμπληρώνοντας το προφίλ σας. Βοηθήστε τους άλλους να σας γνωρίσουν καλύτερα προσθέτοντας στο προφίλ σας προσωπικές πληροφορίες για τις γνώσεις και τις δεξιότητές σας, τα χόμπι και τα ενδιαφέροντά σας.

Δραστηριοποιηθείτε. Εξερευνήστε τα προφίλ για να ανακαλύψετε ποιος ασχολείται με έργα παρόμοια με τα δικά σας, να βρείτε λύσεις σε παρόμοια προβλήματα και να εντοπίσετε τους πόρους που οι άλλοι χρήστες χρησιμοποιούν για την απλούστευση της εργασίας τους. Ανακαλύψτε ποιοι είναι οι συνάδελφοί σας, ποια είναι τα ενδιαφέροντά τους, τι δεξιότητες έχουν, τι διαβάζουν ή γράφουν και με ποιον τρόπο θα έρθετε σε επαφή μαζί τους. Δείτε με ποιους άλλους έχουν επαφές και προσπαθήστε να διευρύνετε τον κύκλο σας.

Δικτυωθείτε. Φτιάξτε το δικό σας δίκτυο. Χρησιμοποιήστε τα προφίλ για να κάνετε χρήσιμες επαφές. Βρείτε άτομα με τις δεξιότητες και την τεχνογνωσία που αναζητάτε και προσθέστε τους στο δίκτυό σας. Όταν φτιάξετε το δίκτυό σας, κρατήστε επαφή παρακολουθώντας τις δραστηριότητές των άλλων και ενημερώνοντας τους άλλους για τις δικές σας.

Η υπηρεσία Προφίλ μπορεί να σας βοηθήσει να αξιοποιήσετε τη συλλογική γνώση του ανθρώπινου δυναμικού του οργανισμού σας και να συνεργαστείτε αποτελεσματικά με τα κατάλληλά πρόσωπα.

Πρόσβαση στην υπηρεσία Προφίλ

Για να συνδεθείτε στην υπηρεσία Προφίλ, μεταβείτε στο δικτυακό τόπο που έχει οριστεί ως αρχική σελίδα της υπηρεσίας από το διαχειριστή που ρύθμισε το προϊόν. Καταχωρήστε το όνομα χρήστη και τον κωδικό πρόσβασης που δημιούργησε ο διαχειριστής για σας. Αν θέλετε να προσθέσετε κάποιο πρόσωπο στους χρήστες της υπηρεσίας, επικοινωνήστε με το διαχειριστή. Το συγκεκριμένο πρόσωπο θα πρέπει πρώτα να προστεθεί στον κατάλληλο κατάλογο χρηστών για να είναι δυνατή η σύνδεσή του στην υπηρεσία.
