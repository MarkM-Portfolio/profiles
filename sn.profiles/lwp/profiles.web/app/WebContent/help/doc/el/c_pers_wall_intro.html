Βοήθεια για τα Προφίλ

Τι είναι ο Πίνακας;

Η σελίδα προφίλ κάθε προσώπου περιλαμβάνει μια περιοχή στην οποία εμφανίζονται οι πιο πρόσφατες πληροφορίες σχετικά με τις αλλαγές στο δίκτυο του προσώπου. Η περιοχή αυτή ονομάζεται Πίνακας. Οποιοσδήποτε χρήστης έχει πρόσβαση στο προφίλ ενός προσώπου μπορεί να δημοσιεύει μηνύματα και σχόλια στον πίνακα του προσώπου. Με αυτό τον τρόπο, οι χρήστες της υπηρεσίας Προφίλ μπορούν να επικοινωνούν άμεσα μεταξύ τους χωρίς να χρειάζεται να διακόψουν την εργασία τους στην υπηρεσία Προφίλ.

Η περιοχή Πίνακας ενός προφίλ χρησιμοποιείται για την εμφάνιση πληροφοριών σχετικά με τις πιο πρόσφατες αλλαγές του κατόχου του προφίλ στο μήνυμα κατάστασής του και σε όλες τις λειτουργίες του IBM Lotus Connections. Αυτή η μορφή άμεσης επικοινωνίας είναι ιδιαίτερα χρήσιμη όταν πρέπει να είστε σε συνεχή επαφή με συγκεκριμένα πρόσωπα στο δίκτυό σας. Μπορείτε να επισκέπτεστε τις σελίδες προφίλ αυτών των προσώπων και να εξετάζετε τον πίνακα κάθε προσώπου για να δείτε μια σύνοψη των πιο πρόσφατων ενεργειών του στο Lotus Connections. Ο κάτοχος ενός προφίλ και οι άλλοι χρήστες μπορούν να δημοσιεύουν μηνύματα ως απάντηση σε οποιαδήποτε καταχώρηση πληροφοριών ενημέρωσης ή σε οποιοδήποτε μήνυμα.

Ο διαχειριστής της υπηρεσίας μπορεί να επιλέξει αν η λειτουργία Πίνακας θα είναι ενεργοποιημένη ή απενεργοποιημένη στον οργανισμό σας. Η λειτουργία Πίνακας είναι ενεργοποιημένη από προεπιλογή. Ο διαχειριστής μπορεί επίσης να ορίσει ρυθμίσεις σχετικά με τη δυνατότητα δημοσίευσης στην περιοχή Πίνακας των προφίλ. Για παράδειγμα, ο διαχειριστής μπορεί να περιορίσει την πρόσβαση στην περιοχή Πίνακας των προφίλ ώστε μόνο οι χρήστες που ανήκουν στο δίκτυο ενός προσώπου να μπορούν να δημοσιεύουν στον πίνακα του προφίλ του. Αυτές οι ρυθμίσεις έχουν καθολική ισχύ.
