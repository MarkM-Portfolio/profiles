Βοήθεια για τα Προφίλ

Δημοσίευση σχολίων στον πίνακα

Επικοινωνήστε με τα άλλα μέλη του δικτύου σας προσθέτοντας σχόλια στα μηνύματα κατάστασής τους ή στα μηνύματα που εμφανίζονται στην περιοχή Πίνακας του προφίλ τους.

Πριν ξεκινήσετε
Για να δημοσιεύσετε ένα σχόλιο στο δικό σας προφίλ ή στο προφίλ κάποιου άλλου προσώπου, πρέπει να έχετε συνδεθεί στην υπηρεσία Προφίλ.
Σχετικά με αυτή την εργασία
Η λειτουργία Πίνακας είναι ένα εργαλείο διαδραστικής επικοινωνίας που σας παρέχει τη δυνατότητα να επικοινωνείτε με τα άλλα μέλη του δικτύου σας δημοσιεύοντας μηνύματα στα προφίλ τους και σχολιάζοντας τα μηνύματα που τα άλλα μέλη δημοσιεύουν στο δικό σας προφίλ. Με αυτό τον τρόπο, μπορείτε να ανταλλάσσετε απόψεις και να συνεργάζεστε για τη διευθέτηση διαφόρων ζητημάτων, αναπτύσσοντας έτσι ένα νήμα συζήτησης που μπορεί να παρακολουθήσει ο οποιοσδήποτε.

Για να προσθέσετε ένα σχόλιο στην περιοχή Πίνακας στο προφίλ ενός προσώπου, εκτελέστε την παρακάτω διαδικασία:

  1. Εντοπίστε το μήνυμα ή το σχόλιο στο οποίο θέλετε να απαντήσετε και κάντε ένα από τα εξής:
    • Αν θέλετε να απαντήσετε σε ένα μήνυμα, πατήστε Προσθήκη σχολίου κάτω από το μήνυμα.
    • Αν θέλετε να απαντήσετε σε ένα σχόλιο, πατήστε Προσθήκη άλλου σχολίου κάτω από το σχόλιο.
  2. Καταχωρήστε το σχόλιό σας στο πεδίο και πατήστε Προσθήκη σχολίου.
Αποτελέσματα
Το σχόλιό σας θα εμφανιστεί ακριβώς κάτω από το μήνυμα για το οποίο προστέθηκε. Ο κάτοχος του προφίλ θα λάβει ένα μήνυμα e-mail που θα τον ενημερώνει ότι ένα νέο σχόλιο προστέθηκε στο προφίλ του.
