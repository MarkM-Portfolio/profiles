Βοήθεια για τα Προφίλ

Σύνθετη αναζήτηση

Χρησιμοποιήστε τη σύνθετη αναζήτηση για να εντοπίσετε προφίλ αναζητώντας πληροφορίες επικοινωνίας, στοιχεία τοποθεσίας, προσδιοριστικά, ενδιαφέροντα ή πληροφορίες εμπειρίας, τεχνικών γνώσεων ή εκπαίδευσης. Μπορείτε να εκτελέσετε σύνθετη αναζήτηση από οποιαδήποτε σελίδα της υπηρεσίας Προφίλ.

Σχετικά με αυτή την εργασία

Η σύνθετη αναζήτηση φιλτράρει τα αποτελέσματα παρέχοντας τη δυνατότητα ταυτόχρονης αναζήτησης σε περισσότερα από ένα πεδία των προφίλ. Αυτό το είδος αναζήτησης είναι πιο στοχευμένο σε σχέση με τη γρήγορη αναζήτηση που αναζητά πληροφορίες μόνο σε ένα πεδίο. Όσα περισσότερα πεδία συμπληρώσετε, τόσο πιο περιορισμένα θα είναι τα αποτελέσματα της αναζήτησης.

Για να πραγματοποιήσετε σύνθετη αναζήτηση, εκτελέστε την παρακάτω διαδικασία:

  1. Επιλέξτε Σύνθετη αναζήτηση στα Προφίλ από το μενού του πεδίου αναζήτησης στη γραμμή τίτλου.
  2. Ορίστε τα κριτήρια αναζήτησης συμπληρώνοντας ένα ή περισσότερα πεδία στη φόρμα σύνθετης αναζήτησης. Ο διαχειριστής μπορεί να έχει προσαρμόσει τη φόρμα σύνθετης αναζήτησης προσθέτοντας ή αφαιρώντας πεδία ανάλογα με τις ανάγκες του οργανισμού σας. Ως εκ τούτου, τα εμφανιζόμενα πεδία στη φόρμα σύνθετης αναζήτησης μπορεί να διαφέρουν από αυτά που παρουσιάζονται στον ακόλουθο πίνακα.
    Πίνακας 1. Πεδία σύνθετης αναζήτησης
    Όνομα πεδίου Περιγραφή
    Εμφανιζόμενο όνομα Καταχωρήστε το όνομα και το επώνυμο του προσώπου θέλετε να εντοπίσετε.
    Όνομα Καταχωρήστε ένα όνομα.
    Επώνυμο Καταχωρήστε ένα επώνυμο.
    Προσδιοριστικά Καταχωρήστε λέξεις-κλειδιά για να αναζητήσετε πρόσωπα με βάση τα προσδιοριστικά τους.
    Επαγγελματικός τίτλος Καταχωρήστε έναν επαγγελματικό τίτλο για να εντοπίσετε τα προφίλ προσώπων που έχουν το ίδιο αντικείμενο ή την ίδια θέση εργασίας.
    Τμήμα Καταχωρήστε μια λέξη-κλειδί για να εντοπίσετε τα προφίλ προσώπων που εργάζονται σε ένα συγκεκριμένο τμήμα.
    Προσωπικές πληροφορίες Καταχωρήστε λέξεις-κλειδιά που περιγράφουν το είδος της εμπειρίας και τις δεξιότητες που σας ενδιαφέρουν.
    Ιστορικό Καταχωρήστε λέξεις-κλειδιά που περιγράφουν το είδος του ιστορικού που σας ενδιαφέρει.
    Οργανισμός ή εταιρεία Καταχωρήστε το όνομα ενός οργανισμού ή μιας εταιρείας για να εντοπίσετε τα προφίλ προσώπων που εργάζονται στον οργανισμό ή στην εταιρεία.
    Πόλη Καταχωρήστε το όνομα μιας πόλης για να εντοπίσετε τα προφίλ προσώπων που εργάζονται σε αυτή την πόλη.
    Νομός Καταχωρήστε το όνομα ενός νομού για να εντοπίσετε τα προφίλ προσώπων που εργάζονται σε αυτό το νομό.
    Χώρα Καταχωρήστε το όνομα μιας χώρας για να εντοπίσετε τα προφίλ προσώπων που εργάζονται σε αυτή τη χώρα.
    Διεύθυνση e-mail Καταχωρήστε μια διεύθυνση e-mail εργασίας. Αυτή η επιλογή δεν είναι διαθέσιμη όταν οι ρυθμίσεις στο περιβάλλον σας δεν επιτρέπουν την εμφάνιση διευθύνσεων e-mail.
    Αριθμός τηλεφώνου Καταχωρήστε έναν αριθμό τηλεφώνου. Η πληροφορία αυτή θα αναζητηθεί και στα πέντε πεδία τηλεφώνου που υπάρχουν σε ένα προφίλ.
  3. Πατήστε Αναζήτηση.
