Βοήθεια για τα Προφίλ

Προβολή επαφών δικτύου

Θέλετε να δείτε ποιοι έχουν συνδεθεί με ποιους; Χρησιμοποιήστε την υπηρεσία Προφίλ για να δείτε τις επαφές δικτύου ενός προσώπου ή για να ελέγξετε αν κάποιο πρόσωπο είναι ήδη μέλος του δικτύου σας.

Σχετικά με αυτή την εργασία
Μπορείτε να δείτε τα πρόσωπα που συμμετέχουν στο δίκτυό σας ή στο δίκτυο άλλων προσώπων από τη σελίδα Επαφές δικτύου.

Για να δείτε τις επαφές δικτύου για το προφίλ σας ή για το προφίλ κάποιου άλλου προσώπου, εκτελέστε την παρακάτω διαδικασία:

  1. Ανοίξτε τη σελίδα προφίλ του προσώπου του οποίου τις επαφές θέλετε να δείτε.
  2. Επιλέξτε Εμφάνιση όλων στην περιοχή Το δίκτυό μου στη δεξιά πλευρά της σελίδας.
  3. Ταξινομήστε τις εμφανιζόμενες επαφές χρησιμοποιώντας μία από τις διαθέσιμες επιλογές στη λίστα Ταξινόμηση κατά:
    Πίνακας 1. Επιλογές εμφάνισης
    Επιλογή Περιγραφή
    Εμφανιζόμενο όνομα Εμφανίζει τις επαφές ταξινομημένες αλφαβητικά.
    Πρόσφατες Εμφανίζει τις επαφές ταξινομημένες χρονολογικά.
