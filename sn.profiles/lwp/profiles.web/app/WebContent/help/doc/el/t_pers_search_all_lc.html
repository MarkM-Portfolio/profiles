Βοήθεια για τα Προφίλ

Αναζήτηση στο Lotus Connections

Χρησιμοποιήστε το εργαλείο αναζήτησης για να αναζητήσετε πληροφορίες σε όλες τις λειτουργίες του IBM Lotus Connections. Το εργαλείο αναζήτησης βρίσκεται στη γραμμή τίτλου της εφαρμογής.

Σχετικά με αυτή την εργασία
Για να αναζητήσετε πληροφορίες σε όλες τις λειτουργίες του Lotus Connections από την υπηρεσία Προφίλ, εκτελέστε την παρακάτω διαδικασία:
  1. Από οποιαδήποτε προβολή της υπηρεσίας Προφίλ, καταχωρήστε τον όρο που θέλετε να αναζητήσετε στο πεδίο Αναζήτηση στη γραμμή τίτλου.
  2. Ορίστε την εμβέλεια αναζήτησης από το μενού που θα ανοίξετε πατώντας στο βέλος του πεδίου Αναζήτηση. Το μενού περιλαμβάνει τις ακόλουθες επιλογές:
    Πίνακας 1. Επιλογές αναζήτησης
    Επιλογή Περιγραφή
    Όλες οι λειτουργίες του Connections Η καθορισμένη συμβολοσειρά κειμένου θα αναζητηθεί σε όλες τις λειτουργίες του Lotus Connections.
    <όνομα_λειτουργίας> Η καθορισμένη συμβολοσειρά κειμένου θα αναζητηθεί στην επιλεγμένη λειτουργία.
    Σύνθετη αναζήτηση Ανοίγει η φόρμα Σύνθετη αναζήτηση. Χρησιμοποιήστε αυτή τη φόρμα για να αναζητήσετε τη συμβολοσειρά κειμένου σε μία, όλες ή μερικές από τις λειτουργίες.
  3. Πατήστε Αναζήτηση.
Γονικό θέμα: Επισκόπηση, γενικές οδηγίες και συχνές ερωτήσεις για τα Προφίλ
