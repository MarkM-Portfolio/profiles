Справка Профайлов

Выполнение расширенного поиска

Пользуйтесь расширенным поиском для того чтобы находить профайлы по контактной информации, расположению, тегам, сведениях об опыте, технических знаниях, обучении и интересах. Выполнить расширенный поиск можно на любой странице в Профайлах.

Об этой задаче

При расширенном поиске результаты отфильтровываются благодаря тому, что поиск ведется по нескольким полям профайла одновременно. Это более направленный поиск по сравнению с быстрым, при котором используется одно поле. Чем больше значений указано в этих полях, тем уже область поиска.

Для расширенного поиска в профайлах выполните следующие действия:

  1. Выберите Расширенный поиск профайлов в управляющем элементе в строке заголовка.
  2. Администратор может при необходимости добавлять и удалять поля с формы поиска Расширенный поиск профайлов. Из-за этой возможности поля, отображаемые в форме, могут отличаться. Для поиска пользователей на основе их контактной информации введите значение в одно или несколько следующих полей:
    Табл. 1. Поля расширенного поиска
    Имя поля Вводимые слова
    Отображаемое имя Введите полное имя искомого пользователя.
    Имя Введите имя.
    Фамилия Введите фамилию.
    Теги Введите ключевые слова для поиска пользователей на основе тегов.
    Должность Введите название должности для поиска профайлов пользователей, занимающих такую должность.
    Отдел Введите ключевое слово для поиска пользователя, работающего в указанном отделе.
    Обо мне Введите ключевые слова, описывающие навыки и профессиональный опыт.
    Биография Введите ключевые слова, описывающие требуемые интересы.
    Организация или компания Введите название организации или компании для поиска профайлов сотрудников.
    Город Введите название города для поиска профайлов пользователей, работающих в этом городе.
    Область Введите название области для поиска профайлов пользователей, работающих в этой области.
    Страна Введите название страны для поиска профайлов пользователей, работающих в этой стране.
    Адрес электронной почты Введите рабочий электронный адрес. Опция недоступна, если конфигурация запрещает отображение электронных адресов.
    Номер телефона Введите номер телефона. Поиск производится по всем пяти полям номеров телефонов профайла пользователя.
  3. Нажмите кнопку Найти.
