Экспорт контактных сведений

Экспортируйте vCard пользователя, чтобы добавить его в качестве нового контакта.

Об этой задаче
Файл виртуальной карточки (vCard) содержит основную контактную информацию согласно спецификации консорциума по почте Internet (Internet Mail Consortium). Можно добавлять новые контакты, используя систему электронной почты для импорта файлов vCard. Также можно загрузить vCard из визитной карточки профайла.

При загрузке vCard необходимо выбрать кодировку символов, которую поддерживает ваша программа чтения адресной книги. По умолчанию используется Международный (UTF-8). Эта кодировка подходит для пользователей IBM Lotus Notes и Microsoft Outlook. Однако если используется Microsoft Outlook Express или Microsoft Windows Address Book, может потребоваться выбор другой кодировки.

Для сохранения vCard коллеги в вашей записи о сотруднике выполните следующие действия.
  1. Откройте страницу профайла для пользователя, которого вы хотите добавить в контакты.
  2. Нажмите Загрузить vCard.
  3. Выберите нужную кодировку символов.
  4. Нажмите кнопку Загрузить, чтобы сохранить vCard пользователя в указанном расположении.
