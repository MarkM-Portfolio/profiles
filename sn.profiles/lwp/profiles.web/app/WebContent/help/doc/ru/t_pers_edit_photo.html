Изменение фотографии в профайле

На странице Изменить профайл можно загрузить вашу новую фотографию.

Об этой задаче

Сделайте ваш профайл лучше, периодически обновляя вашу фотографию. Загружайте подходящие для делового общения фотографии.

Для обновления фотографии в профайле выполните следующие действия:
  1. На странице Изменить мой профайл выберите Фотография.
  2. Чтобы изменить фотографию в профайле, выполните одно из следующих действий:
    • Чтобы удалить существующую фотографию, выберите Удалить изображение.
    • Чтобы загрузить новое изображение:
      1. Нажмите кнопку Обзор для поиска графического файла .jpeg или .gif с вашей фотографией, которую требуется разместить в профайле.
        Прим.: Ограничений для размера файла нет. Однако, поскольку размер изображения автоматически изменяется на сервере Профайлов в соответствии с областью, выделенной в пользовательском интерфейсе, следует иметь в виду, что чем больше размер загруженного файл, тем больше времени потребуется серверу для его обработки.
      2. Выберите файл изображения для просмотра и нажмите кнопку Открыть.
      3. Нажмите кнопку Обновить для сохранения изменений.
    • Для предварительного просмотра изображения:
      1. Нажмите кнопку Обзор и найдите файл .jpeg или .gif для предварительного просмотра, затем нажмите кнопку Открыть.
        Прим.: Ограничений для размера файла нет. Обратите внимание, что вследствие проверки изображения на наличие вирусов и изменения его размера для предварительного просмотра, время предварительного просмотра выбранного файла изображения зависит от его размера.
      2. Выберите Предварительный просмотр изображения для просмотра изображения в том виде, в каком оно появится на странице вашего профайла.
      3. Щелкните внутри окна выбора и переместите его, чтобы выбрать фрагмент изображения, который следует разместить в профайле. Можно также переместить край окна выбора с помощью указателя мыши для изменения размера выбранной области.
      4. Нажмите кнопку Обновить для сохранения изменений.
