Поиск профайлов

Поиск профайлов позволяет найти людей по контактной информации, например, по имени и должности, или по более подробным сведениям, например, по интересам, опыту и рабочим проектам.

Существуют различные типы поиска Профайлов:
  • Для выполнения базового поиска на основе одного поискового поля воспользуйтесь управляющим элементом поиска. Этот элемент расположен в правом верхнем углу на каждой странице Профайлов.
  • Элемент быстрого поиска позволяет производить поиск пользователя по имени или ключевому слову с помощью опций Профайлы по имени и Профайлы по ключевым словам.
  • Выберите Расширенный поиск профайлов для использования опций расширенного поиска и выполнения более направленного поиска на основе нескольких полей.
  • Облако тегов на домашней странице Профайлы показывает все теги профайлов для организации. Щелкните на теге для вывода списка сотрудников с этим тегом, а также связанных тегов для помощи и уточнения поиска. Для поиска профайла, содержащего определенный тег, нажмите Найти теги, введите тег в поле и нажмите Найти профайлы, содержащие данный тег.

Во всех видах поиска поддерживается символ подстановки (*). Например, если ввести искомую строку Ethan * при поиске по имени, то будут возвращены все профайлы пользователей с именем Ethan. Если ввести искомую строку 555* при поиске номера телефона, то будут возвращены все номера, которые начинаются с 555.
