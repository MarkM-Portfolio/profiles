Справка Профайлов

Выполнение быстрого поиска

Для быстрого поиска в верхней части любой страницы Профайлов расположено соответствующее поле.

Об этой задаче

При быстром поиске профайлы в каталоге ищутся по одному полю.

Для выполнения быстрого поиска выполните следующие действия:

  1. Щелкните на стрелке вниз около поля Поиск в панели навигации и выберите одну из следующих опций:
    Табл. 1. Опции поиска профайлов
    Вариант Выберите эту опцию для
    Профайлы по имени Поиск пользователя по имени. Введите имя и фамилию. Буферизация ввода предлагает имена по мере ввода текста. Если искомое имя найдено, нажмите его для перехода в профайл пользователя.
    Прим.: Эта опция является опцией по умолчанию.
    Профайлы по ключевому слову Поиск пользователя по ключевым словам. Ищутся профайлы и все связанное содержимое, например, опыт работы и информация об интересах в профайле.
  2. Введите слово для поиска в текстовом поле. Используйте звездочку (*) в качестве символа-заменителя.
  3. Нажмите кнопку Найти.
