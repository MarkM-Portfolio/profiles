Справка Профайлов

Новое в этом выпуске?

Приведено описание новых и обновленных функций, разработанных для облегчения использования профайлов для взаимодействия с другими пользователями.

Lotus Connections 2.0.1

  • В число усовершенствований защиты входит возможность запрета отображения электронных адресов в визитных карточках профайлов.

Lotus Connections 2.0

  • Усовершенствованная возможность загрузки графических объектов позволяет загружать в профайл изображения большего размера, а также обрезать их в соответствии с необходимостью.
  • Усовершенствования визитной карточки профайла позволяют взаимодействовать с коллегами рядом способов, например проверяя их статус в Sametime, добавляя их в свою сеть и начиная с ними мгновенный разговор.
  • Усовершенствованное добавление тегов позволяет добавлять теги для других людей, используя ключевые слова для определения их связи с вами, а также для облегчения поиска.
  • Информация об отчетности теперь сообщает вам, какие пользователи имеют общего руководителя, а также кто кем управляет.
  • Усовершенствованные функции поиска позволяют выполнять поиск по имени и фамилии.
