Обновление контактной информации

Актуализируйте вашу контактную информацию, чтобы сотрудники могли связываться с вами.

Об этой задаче
Поскольку информация взята из каталога персонала организации, а в других каталогах хранится другая информация, поля вашего профайла могут отличаться от полей по умолчанию. Кроме того, администратор может включить автоматическое заполнение поля, чтобы вы не заполняли его. Также администратор может сделать поле доступным только для чтения, если в нем находится информация, которая не имеет отношения к вашей организации.

Для изменения контактной информации выполните следующие действия:

  1. На странице Изменить мой профайл выберите Контактная информация.
  2. По необходимости измените содержимое следующих полей:
    Табл. 1. Поля контактной информации
    Имя поля Описание
    Дом Номер здания.
    Этаж Этаж.
    Офис Название офиса.
    Номер офиса Номер телефона в офисе.
    Номер IP-телефонии Номер линии прямой связи.
    Номер мобильного телефона Номер мобильного телефона.
    Номер пейджера Номер пейджера.
    Номер факса Номер факса.
    Дополнительный электронный адрес Дополнительный электронный адрес. Эта опция недоступна, если администратор запретил отображение электронных адресов.
    Ссылка на блог Ссылка на личный блог.
    Должность Официальное название вашей должности.
    Помощник Имя вашего помощника.
    Часовой пояс Щелкните на стрелке для выбора часового пояса для местонахождения вашего основного рабочего места.
  3. Нажмите кнопку Обновить.
