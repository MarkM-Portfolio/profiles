Поиск в Lotus Connections

Для поиска информации во всех функциях IBM Lotus Connections воспользуйтесь управляющим элементом в строке заголовка.

Об этой задаче
Для поиска в профайлах всех функций Lotus Connections выполните следующие действия:
  1. В любом представлении в Профайлах введите условие поиска, используемое для поля Поиск в строке заголовка.
  2. Определите область поиска. Для этого щелкните на стрелке рядом с полем Поиск и выберите один из вариантов:
    Табл. 1. Варианты поиска
    Вариант Описание
    Все соединения Поиск текстовой строки во всех функциях Lotus Connections.
    <имя-функции> Поиск текстовой строки в отдельной функции.
    Дополнительно Открывается форма Расширенный поиск. Воспользуйтесь формой для поиска по одиночному условию или по нескольким в одной, всех или подмножестве функций.
  3. Нажмите кнопку Найти.
