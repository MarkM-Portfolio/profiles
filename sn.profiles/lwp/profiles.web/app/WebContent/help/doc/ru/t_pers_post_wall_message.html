Справка Профайлов

Публикация сообщений на доске объявлений

С сотрудниками в сети можно связываться непосредственно, публикуя сообщения и комментарии на Доске объявлений их профайла.

Подготовка
Для отправки сообщений в свой профайл или другие профайлы необходимо войти в профайлы.
Об этой задаче
Для всех пользователей Профайлов на странице профайла расположена область Доска объявлений, на которой показаны последние сообщения о состоянии и обновления функций IBM Lotus Connections. С сотрудниками можно связаться, отправляя сообщения непосредственно в их профайлы.

Для отправки сообщения на доску объявлений пользователя выполните следующие действия:

  1. На странице профайла пользователя в верхней части Доски объявлений нажмите поле сообщений, где показывается приглашение Напишите что-нибудь на этой доске объявлений профайла.
  2. Введите сообщение.
  3. Нажмите кнопку Отправить сообщение.
Результат
Ваше сообщение будет показано в верхней части Доски объявлений, где владелец профайла и другие пользователи могут на это сообщение ответить. Владелец профайла получит уведомление по электронной почте о новом сообщении, отправленном в его профайл.
