Справка Профайлов

Публикация комментариев на доске объявлений

Общайтесь с сотрудниками в сети с помощью оставления комментариев на их сообщение о состоянии, оставленное на Доске объявлений профайла.

Подготовка
Для отправки комментариев в свой профайл или другие профайлы необходимо войти в профайлы.
Об этой задаче
Функция Доска объявлений позволяет быстро взаимодействовать с сотрудниками, отправляя сообщения в их профайлы и комментируя сообщения, оставленные в профайле пользователя. Отвечая на чужие сообщения, можно совместно работать над задачами в сети и создавать цепочки диалоговых сообщений, видимых другим.

Для добавления комментария в область Доска объявлений профайла пользователя выполните следующие действия:

  1. Найдите сообщение или комментарий, на которые необходимо ответить, и выполните следующие действия:
    • Для ответа на сообщение нажмите кнопку Добавить комментарий под сообщением.
    • Для ответа на комментарий нажмите кнопку Добавить еще комментарий под комментарием.
  2. Введите комментарий в предложенном поле и нажмите кнопку Отправить комментарий.
Результат
Комментарий будет показан непосредственно под комментируемым сообщением. Владелец профайла получит уведомление по электронной почте о новом комментарии на свой профайл.
