Использование визитной карточки профайла

С помощью визитной карточки Профайлов можно просмотреть контактную информацию пользователя и связаться с ним.

Об этой задаче
Визитная карточка Профайлов предоставляет удобную моментальную копию информации профайла пользователя. Помимо просмотра должности и контактной информации пользователя, при установленном модуле IBM Lotus Connections для Sametime, в визитной карточке отображается текущий статус пользователя в Sametime.

Визитная карточка содержит ссылки на компоненты Lotus Connections, используемые пользователем, и позволяет вам выполнять ряд действий. Например, если администратор разрешил отображение электронных адресов, то пользователь может отправлять электронные сообщения другим пользователям, а также приглашать их принять участие в мгновенных разговорах (если установлен встраиваемый модуль Lotus Connections для Sametime).

Для доступа к визитной карточке пользователя наведите курсор на его имя в любом из компонентов Lotus Connections и щелкните на появившейся ссылке.

Используя визитную карточку Профайлов, можно выполнить следующие действия:

  1. Обратиться к компонентам Lotus Connections, связанным с данным пользователем, выбирая ссылки на компоненты в верхней части визитной карточки.
  2. Отправить пользователю электронное письмо, выбрав опцию Отправить письмо. Эта опция недоступна, если администратор запретил отображение электронных адресов в Lotus Connections.
  3. Начать мгновенный разговор с пользователем, выбрав опцию Разговор. Эта опция доступна только после установки встраиваемого модуля Lotus Connections для Sametime.
  4. Начать телеконференцию с пользователем, выбрав опцию Позвонить. Эта опция доступна только после установки Sametime Advanced Telephony.
  5. Добавить пользователя как коллегу, выбрав Дополнительные действия > Пригласить в сеть, введя пригласительное сообщение и щелкнув Отправить приглашение.
  6. Загрузить и сохранить vCard пользователя, выбрав Дополнительные действия > Загрузить vCard, нажав Сохранить, выбрав расположение файла и щелкнув Сохранить.
  7. Получить доступ к библиотеке файлов сотрудника в Lotus Quickr можно, выбрав Файлы. Эта опция доступна, только если администратор включил интеграцию с Lotus Quickr и настроил визитную карточку для отображения ссылки на Lotus Quickr.
