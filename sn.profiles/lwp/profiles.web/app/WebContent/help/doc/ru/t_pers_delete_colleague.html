Удаление сотрудников из сети

Если положение пользователя изменилось или он ушел из организации, можно удалить его из сети.

Об этой задаче
Для удаления коллеги из сети выполните следующие действия:
  1. На странице Мой профайл выберите Показать всех в области страницы Моя сеть.
  2. Выберите переключатель напротив пользователя, которого следует удалить.
  3. Нажмите кнопку Удалить из сети.
    Прим.: Пользователь не получает уведомление о его удалении из сети.
