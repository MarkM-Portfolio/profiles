Справка Профайлов

Просмотр профайлов

В профайле указана контактная и рабочая информация о человеке.

При просмотре профайла можно узнать следующую информацию:

Информация о записи адресной книги
Информация, которая обычно указана на визитной карточке, например, имя, должность, компания, местонахождение, номера телефонов и электронные адреса, а также локальное время пользователя и контактные данные помощника, если таковой имеется. Для экспорта контактной информации в формат виртуальной карточки щелкните на значке визитной карточки. Файл виртуальной карточки (vCard) содержит основную контактную информацию согласно спецификации консорциума по почте Internet (Internet Mail Consortium). Можно добавлять новые контакты, используя систему электронной почты для импорта файлы vCard.
Прим.: Администратор может запретить отображение электронных адресов в IBM Lotus Connections. В этом случае в профайле или бизнес-карточке пользователя электронные адреса не отображаются.
Когда включено уведомление Sametime, показывается индикатор присутствия пользователя в сети. Для этого должен быть установлен модуль Lotus Connections для Sametime.
Произношение
Ссылка Произношение присутствует в случаях, когда владелец профайла создал звуковой файл и загрузил его в профайл. Щелкните на этой ссылке для прослушивания звуковой записи правильного произношения имени.
Структура подчинения
Выберите одну из следующих опций:
Табл. 1. Представления структур подчинения
Представление Описание
Вся цепочка подчинения Показывает позицию владельца профайла начиная с владельца профайла и до верхнего звена организационной диаграммы.
Одинаковый менеджер Показывает людей, которые подотчетны тому же руководителю, что и владелец профайла.
Управляемые люди Показывает позицию владельца профайла в полной организационной диаграмме, включая людей, которыми управляет владелец профайла.
Обо мне
Текущий опыт работы, технические навыки, владение языками, интересы, прошлый профессиональный опыт и образование. Это поле для ввода форматированного текста, составленного владельцем профайла и содержащего информацию, которой желает поделиться владелец профайла.
Доска объявлений
Содержит последние публикации на доске объявлений владельца профайла. Здесь же расположены последние сообщения о состоянии и комментарии, оставленные другими пользователями.
Последние публикации
В этом разделе используется представление с вкладками для отображения последних обновлений, внесенных владельцем профайла в другие компоненты Lotus Connections.
Теги профайла
Краткие ключевые слова, заданные владельцем профайла для указания опыта, интересов или видов рабочей деятельности. Эти теги могут быть добавлены профайла или другим лицом.
Инструменты сотрудничества
Если профайлы интегрированы с другими инструментами сотрудничества, например, с IBM Lotus Sametime, то можно узнать о присутствии владельца профайла и его доступности для общения.
Моя сеть
В разделе показан список сотрудников в сети владельца профайла. Для отображения списка всех сотрудников в сети пользователя нажмите кнопку Показать всех.
Мои ссылки
Внешние ссылки, включенные владельцем как часть профайла.
