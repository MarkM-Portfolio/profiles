Справка Профайлов

Создание звукового файла

Запишите, как вы произносите свое имя, чтобы другие смогли услышать его правильное произношение.

Об этой задаче
Для создания звукового файла в системе Microsoft Windows 2000 или Microsoft Windows XP выполните следующие действия:
  1. Выберите Пуск > Программы > Стандартные > Развлечения > Звукозапись.
  2. Нажмите кнопку Запись, произнесите свое полное имя как можно четче и нажмите кнопку Стоп. Нажмите кнопку Воспроизведение для прослушивания записи.
  3. Проверьте размер записи, выбрав опцию меню Файл > Свойства. Размер файла не должен превышать 100 Кб.
  4. Для перезаписи звукового файла нажмите кнопку Файл > Создать и нажмите кнопку Нет в приглашении сохранить запись. Для повторной записи повторите Шаг 2. Для повышения качества записи выполните следующие действия:
    • Если не удается прослушать, убедитесь, что микрофон включен. Выберите Пуск > Программы > Стандартные > Развлечение > Громкость, затем выберите Параметры > Свойства. Выберите Запись, Микрофон и нажмите кнопку OK. Переместите регулятор громкости микрофона вверх на максимальный уровень.
    • Если запись нечеткая или размер файла слишком велик (более 100 Кб), то проведите настройку звукозаписи. Выберите Пуск > Программы > Стандартные > Развлечения > Звукозапись. Выберите Файл > Свойства, а затем Преобразовать. В поле Формат выберите PCM, а в поле Атрибуты выберите 8,000 кГц, 8 бит, Моно. Нажмите кнопку OK, затем нажмите кнопку OK для закрытия окна Свойства.
  5. После создания четкой слышимой записи, размер которой не превышает 100 Кб выберите Файл > Сохранить как для ее сохранения. Введите имя файла и выберите каталог, в который он будет сохранен. Запишите имя и расположение файла, чтобы найти его для загрузки в профайл. По умолчанию в поле Тип файла установлен тип .wav. Примите значение по умолчанию. Нажмите кнопку Сохранить.
Дальнейшие действия
Для создания звукового файла в системе Linux воспользуйтесь командой записи или модулем звукозаписи.
