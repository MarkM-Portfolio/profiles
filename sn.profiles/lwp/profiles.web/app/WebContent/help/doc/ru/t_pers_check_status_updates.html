Проверка обновлений состояния

Будьте в курсе последних изменений в сети, регулярно проверяя обновления состояний.

Об этой задаче
В области Обновления состояния на домашней странице Профайлы показаны последние обновления состояний различных профайлов. Пользователь может отфильтровать отображение: показывать обновления состояний пользователей, подключенных к сети либо собственных обновлений состояния. На этой странице также можно обновить сообщение о личном состоянии и оставить комментарий на сообщения состояний других пользователей.

Для проверки обновлений состояния выполните следующие действия:

  1. Откройте страницу Профайлы.
  2. В области Обновления состояния выполните одно из следующих действий:
    • Выберите Обновления состояния в моей сети для отображения списка последних обновлений пользователей в сети.
    • Выберите Обновления моего состояния для отображения обновлений собственного состояния.
  3. Необязательно: Нажмите кнопку Добавить комментарий для комментирования конкретного сообщения о состоянии.
