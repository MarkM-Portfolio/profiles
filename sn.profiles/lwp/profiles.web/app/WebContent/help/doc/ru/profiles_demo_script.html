Демонстрационный сценарий Профайлов

Демонстрационный текст Профайлов

Об этой задаче

Используйте Профайлы для поиска в организации обладателей нужных навыков и специальностей. Создайте социальную сеть сотрудников и отслеживайте последние публикации.

Этот сценарий демонстрирует выполнение следующих задач:
  • Искать пользователей по имени
  • Добавлять пользователя в сеть
  • Оставлять сообщения на досках объявлений других пользователей
  • Комментировать сообщение на доске объявлений
  • Обновлять сообщение о состоянии
  • Изменять свой профайл
  1. Для быстрого поиска воспользуйтесь управляющим элементом поиска.
  2. Результаты поиска будут показаны с такой информацией, как местонахождение, должность и номер телефона.
  3. Узнайте дополнительные сведения о пользователе, наведя указатель на его имя и щелкнув на его визитной карточке.
  4. Всплывающая визитная карточка содержит фотографию и ссылки на другую информацию о пользователе.
  5. С помощью визитной карточки можно связаться по электронной почте с ее владельцем.
  6. Выберите Дополнительные действия для доступа к дополнительным вариантам контактов с данным пользователем.
  7. Щелкните на ссылке Профайл для отображения полного профайла пользователя.
  8. Страница профайла содержит информацию о пользователе, включая должность, местоположение офиса и контактные сведения.
  9. Если доступна цепочка отчетности, используйте ее для более глубокого изучения структуры организации.
  10. Для добавления пользователя в сеть выберите Пригласить в сеть.
  11. Кроме того, файл vCard пользователя можно сохранить в своем списке контактов.
  12. Поддерживайте связь непосредственно с сотрудниками путем публикации сообщений на их Досках объявлений.
  13. Введите сообщение и нажмите кнопку Отправить сообщение.
  14. Для просмотра последних изменений, внесенных пользователем в другие компоненты, прокрутите область Последние публикации вниз.
  15. Доступ к своему профайлу можно получить в любое время, щелкнув на вкладке Мой профайл.
  16. Выделите время для проверки актуальности вашей информации и для просмотра последних сообщений на своей доске объявлений.
  17. Для ответа на сообщение нажмите кнопку Добавить комментарий.
  18. Обновляя свое сообщение о состоянии, информируйте сотрудников о своих текущих делах.
  19. Выберите Над чем сейчас работаете?, введите сообщение и нажмите кнопку Сохранить.
  20. Для обновления профайла щелкните на вкладке Изменить мой профайл.
  21. Дополните содержимое полей на различных вкладках, а затем нажмите Обновить, чтобы сохранить изменения.
  22. Теперь Вам известны основные сведения, входите в систему и подключайтесь!
  23. Спасибо за просмотр демонстрации. Дополнительная информация приведена на странице справки Профайлов.
