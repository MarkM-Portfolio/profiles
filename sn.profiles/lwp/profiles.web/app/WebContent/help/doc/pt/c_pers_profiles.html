Ajuda dos Perfis

Visualizar perfis

Um perfil faculta informações de contacto e informações relacionadas com o trabalho acerca de uma pessoa.

Visualize o perfil de uma pessoa, para localizar as seguintes informações:

Informações de contacto
O perfil identifica a hora local da pessoa e, se aplicável, as informações de contacto de um assistente, geralmente localizadas num cartão de visita, tal como o nome, cargo, empresa, localização, números de telefone e endereços de correio electrónico. Faça clique no ícone de cartão de visita para exportar as informações de contacto da pessoa como um ficheiro de cartão virtual. Um cartão virtual (vCard) consiste num ficheiro que armazena informações de contacto básicas, seguindo a especificação do Consórcio de Correio da Internet. Pode adicionar novos contactos utilizando o sistema de correio electrónico para importar ficheiros vCard.
Nota: O administrador pode configurar o IBM Lotus Connections para impedir a apresentação de endereços de correio electrónico, fazendo com que estes não sejam apresentados no cartão de visita ou perfil de uma pessoa.
Quando a detecção do Sametime está activada, o estado online da pessoa é apresentado. O Suplemento do Lotus Connections para o Sametime tem de ser instalado para que estas informações sejam apresentadas.
Pronúncia
A ligação Pronunciação é apresentada quando o proprietário do perfil tiver criado um ficheiro áudio e tiver transferido o mesmo para o perfil. Faça clique na ligação para reproduzir uma gravação do nome da pessoa correctamente pronunciado.
Estrutura de relatórios
Seleccione uma das seguintes opções de visualização:
Tabela 1. Vistas de estrutura de relatórios
Ver Descrição
Relatório completo para a cadeia Apresenta a posição do proprietário do perfil a partir deste até ao topo do gráfico da organização.
O mesmo gestor Apresenta as pessoas que também reportam ao gestor do proprietário do perfil.
Pessoas geridas Apresenta a posição do proprietário do perfil no gráfico completo da organização, incluindo as pessoas que este gere.
Acerca de mim
Experiência de trabalho actual, competências técnicas, idiomas falados, interesses, experiência de trabalho anterior e formação. Estas informações correspondem a um campo de rich-text criado pelo proprietário do perfil e que inclui quaisquer informações que este pretenda partilhar.
O Quadro
As últimas publicações no quadro do proprietário do perfil. Estas informações incluem as mensagens de estado recentes do proprietário do perfil e os comentários que os outros utilizadores publicaram no quadro do proprietário do perfil.
Publicações recentes
Esta secção utiliza vistas com separadores para apresentar as actualizações mais recentes que o proprietário do perfil tenha efectuado a outros componentes do Lotus Connections.
Etiquetas de perfis
Palavras-chave curtas utilizadas para identificar as capacidades, áreas de interesse ou actividades profissionais do proprietário do perfil. Estas etiquetas podem ser adicionadas pelo proprietário do perfil ou por outra pessoa.
Ferramentas de colaboração
Quando os perfis são integrados com outras ferramentas de colaboração, tais como o IBM Lotus Sametime, poderá visualizar as informações de presença do proprietário do perfil, que indicam se a pessoa está disponível para iniciar uma conversação.
A minha rede
Esta secção apresenta uma selecção das pessoas existentes na rede do proprietário do perfil. Pode fazer clique em Ver tudo para apresentar uma lista de todas as pessoas na rede dessa pessoa.
As minhas ligações
Ligações externas incluídas pelos proprietários dos perfis como parte dos respectivos perfis.
