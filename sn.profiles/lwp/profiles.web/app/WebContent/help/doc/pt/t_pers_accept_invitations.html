Ajuda dos Perfis

Aceitar convites

Aceite convites para participar em novas redes e adicionar ligações à sua rede pessoal.

Sobre esta tarefa

Se o administrador tiver activado notificações por correio electrónico para o componente Perfis, quando as outras pessoas o adicionarem à respectiva rede, estas irão gerar uma mensagem de correio electrónico a convidá-lo para participar na rede. A mensagem de correio electrónico contém uma ligação para o separador Convites, no qual pode aceitar ou recusar o convite.

Se as notificações por correio electrónico tiverem sido activadas para a implementação, pode aceder a convites directamente a partir do separador Convites na página Contactos de rede.
Nota: Se ignorar um convite, não será enviada qualquer notificação de acção à pessoa que enviou o convite.

Para aceitar um convite, execute uma das seguintes acções:

  1. Se as notificações por correio electrónico estiverem activadas:
    1. Faça clique na ligação fornecida no convite por correio electrónico para abrir o separador Convites na página Contactos de rede. Esta página apresenta todos os convites pendentes.
    2. Faça clique em Aceitar para aceitar o convite e participar na rede da pessoa que enviou o convite.
  2. Se as notificações por correio electrónico não estiverem activadas:
    1. Na página O meu perfil, faça clique em A minha rede.
    2. Faça clique no separador Convites para visualizar todos os convites pendentes.
    3. Faça clique em Aceitar para aceitar um convite e participar na rede de outra pessoa.
