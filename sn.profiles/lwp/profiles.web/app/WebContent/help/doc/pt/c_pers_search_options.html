Ajuda dos Perfis

Procurar perfis

Pode procurar perfis para localizar pessoas pelas respectivas informações de contacto, tais como o nome e o cargo, ou por informações mais avançadas, tais como os respectivos interesses, experiência e projectos de trabalho.

Existem diferentes tipos de procura que poderá efectuar nos Perfis:
  • Utilize o controlo de procura para efectuar uma procura básica, com base num único campo de procura. Este controlo encontra-se disponível na parte superior direita de cada página dos Perfis.
  • Procure uma pessoa por nome ou palavra-chave utilizando as opções Perfis por nome e Perfis por palavra-chave disponíveis a partir do controlo de procura rápida.
  • Faça clique em Procura avançada de perfis para utilizar as opções de procura avançada e efectuar uma procura mais objectiva com base em vários campos de procura.
  • Utilize a nuvem de etiquetas da página Início dos perfis para procurar todas as etiquetas de perfil da organização. Faça clique numa etiqueta para apresentar uma lista das pessoas com essa etiqueta, bem como as etiquetas relacionadas para ajudar a aperfeiçoar a procura.Para procurar perfis que contenham uma etiqueta específica, faça clique em Localizar etiquetas, introduza uma etiqueta no campo fornecido e faça clique em Procurar perfis que contêm esta etiqueta.

O carácter global (*) é suportado em todos os tipos de procura. Por exemplo, se introduzir António * como cadeia de procura para nomes, os resultados apresentarão todos os perfis das pessoas cujo nome próprio é António. Se introduzir 555* como cadeia de procura para números de telefone, os resultados apresentarão todos os números que começam por 555.
