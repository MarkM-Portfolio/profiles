Criando um Arquivo de Áudio

Crie um registro de si mesmo dizendo seu nome para que outros possam ouvi-lo com a pronúncia correta.

Por Que e Quando Desempenhar Esta Tarefa
Para criar um arquivo de áudio em um sistema Microsoft Windows 2000 ou Microsoft Windows XP, execute as etapas a seguir:
  1. Selecione Iniciar > Todos os Programas > Acessórios > Entretenimento > Gravador de Som.
  2. Clique no botão Gravar, diga seu nome completo, falando o mais claro possível e, em seguida, clique no botão Parar. Clique no botão Reproduzir para ouvir a gravação.
  3. Verifique o tamanho da gravação clicando em Arquivo > Propriedades. O tamanho dos dados deve ser menor que 100 KB.
  4. Para regravar o arquivo de áudio, clique em Arquivo > Novo e selecione Não ao ser solicitado para salvar a gravação. Repita a etapa 2 para regravar. Sigas estas dicas para melhorar a qualidade da gravação:
    • Se a gravação estiver inaudível, certifique-se de que o microfone esteja ativado. Selecione Iniciar > Todos os Programas > Acessórios > Entretenimento > Controle de Volume e, em seguida, selecione Opções > Propriedades. Escolha Gravação, selecione Microfone e, em seguida, clique em OK. Arraste o controlador de volume do microfone até o nível máximo.
    • Se a gravação não estiver clara ou se o arquivo produzido for muito grande (mais de 100 KB), configure o gravador de som. Selecione Iniciar > Todos os Programas > Acessórios > Entretenimento > Gravador de Som. Selecione Arquivo > Propriedades e, em seguida, clique em Converter Agora. No campo Formato, selecione PCM e, no campo Atributos, selecione 8.000 kHz, 8 bit, Mono. Clique em OK e, em seguida, clique em OK para fechar a janela Propriedades.
  5. Depois de ter produzido uma gravação clara, audível e menor que 100 KB, selecione Arquivo > Salvar como para salvá-la. Digite um nome para o arquivo e escolha um diretório no qual salvá-lo. Anote o nome e o local para que possa recuperá-lo ao fazer upload do arquivo em seu perfil. O tipo de arquivo padrão no campo Salvar como tipo é .wav. Aceite o padrão. Clique em Salvar.
O que Fazer Depois
Para criar um arquivo de áudio em um sistema Linux, utilize o comando record ou um utilitário de gravação de som.
