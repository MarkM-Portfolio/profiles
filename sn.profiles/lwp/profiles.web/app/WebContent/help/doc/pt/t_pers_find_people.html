Ajuda dos Perfis

Efectuar uma procura rápida

Utilize o controlo de procura na parte superior de qualquer página nos Perfis para efectuar procuras rápidas.

Sobre esta tarefa

Uma procura rápida consiste numa procura efectuada num único campo no directório dos perfis.

Para efectuar uma procura rápida, execute os seguintes passos:

  1. Faça clique na seta para baixo junto ao campo Procurar na barra de navegação e seleccione uma das seguintes opções:
    Tabela 1. Opções de procura nos Perfis
    Opção Seleccione esta opção para
    Perfis por nome Localizar uma pessoa efectuando uma procura pelo respectivo nome. Introduza um nome próprio seguido de um apelido. A memorização de teclas sugere nomes à medida que escreve. Se o nome que pretende for apresentado, faça clique no mesmo para navegar directamente para o perfil dessa pessoa.
    Nota: Esta opção corresponde à predefinição de procura.
    Perfis por palavra-chave Procurar uma pessoa com base numa palavra-chave. Esta opção permite procurar perfis e todo o conteúdo associado, por exemplo, a experiência profissional e as informações de segundo plano num perfil.
  2. No campo de texto, introduza a palavra que pretende procurar. Utilize um asterisco (*) como carácter global.
  3. Faça clique em Procurar.
