Exportando os Detalhes do Contato

Exporte o vCard de uma pessoa para incluí-la como um novo contato.

Por Que e Quando Desempenhar Esta Tarefa
Um vCard (cartão virtual) é um arquivo que armazena informações básicas de contato seguindo a especificação Internet Mail Consortium. É possível incluir novos contatos utilizando o sistema de e-mail para importar arquivos vCard.

Para salvar o vCard de um colega nos registros pessoais, execute as etapas a seguir.

  1. Abra a página de perfil para a pessoa que deseja incluir como um contato.
  2. Clique no ícone do vCard à direita do nome.
  3. Clique em Salvar para salvar o vCard da pessoa em uma localidade especificada.
O que Fazer Depois
Nota: É possível também fazer o download do vCard a partir do cartão de visita do perfil.
