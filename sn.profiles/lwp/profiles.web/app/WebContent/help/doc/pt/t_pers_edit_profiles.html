Ajuda dos Perfis

Editar o perfil

Edite o perfil para adicionar ou actualizar as suas próprias informações, alterar a imagem do perfil ou adicionar um ficheiro de pronunciação.

Sobre esta tarefa

Inicialmente, um administrador cria o perfil com informações do directório do utilizador para a organização. Uma vez que as informações são obtidas a partir do directório de funcionários da organização e directórios diferentes armazenam informações diferentes, os campos que são apresentados no perfil poderão diferir dos campos predefinidos. Para além disso, o administrador pode preencher automaticamente um campo, para que não seja necessário que o utilizador o preencha. O administrador pode definir também um campo como só de leitura, se este recolher informações que não sejam relevantes para a organização.

Para editar o perfil, execute os seguintes passos:
  1. Faça clique em Editar o meu perfil.
  2. Dependendo do tipo de alterações que pretende efectuar, faça clique num dos seguintes separadores:
    Tabela 1. Separadores de edição de perfis
    Separador Seleccione este separador se pretender:
    Informações de contacto Editar a localização do escritório, os detalhes de contacto, o cargo ou seleccionar um fuso horário diferente. Adicionar ou alterar a ligação do blogue pessoal.
    Acerca de mim Actualizar os detalhes do histórico de trabalho e informações de segundo plano.
    Fotografia Actualizar ou remover a imagem actual do perfil.
    Pronúncia Actualizar ou remover um ficheiro áudio do nome do utilizador a ser correctamente pronunciado.
  3. Quando tiver efectuado as alterações necessárias, faça clique em Actualizar para aplicar as mesmas. Se corrigir o conteúdo de um campo e, em seguida, navegar para outra página, ser-lhe-á solicitado que guarde ou cancele as alterações efectuadas.
