Ajuda dos Perfis

Alterar a imagem do perfil

Transfira uma fotografia actualizada para o perfil a partir da página Editar o meu perfil.

Sobre esta tarefa

Pode melhorar o perfil actualizando a sua imagem de tempos a tempos. Certifique-se de que transfere uma imagem adequada ao negócio.

Para transferir a imagem do perfil, execute os seguintes passos:
  1. Na página Editar o meu perfil, faça clique em Fotografia.
  2. Para editar a imagem do perfil, execute uma das seguintes acções:
    • Para remover a fotografia existente, seleccione Remover imagem.
    • Para transferir uma nova imagem:
      1. Faça clique em Pesquisar para localizar um ficheiro .jpeg ou .gif com uma fotografia sua para apresentar no perfil.
        Nota: Não existem restrições do tamanho do ficheiro. No entanto, uma vez que a imagem é automaticamente redimensionada no servidor dos Perfis de modo a ajustar-se ao espaço atribuído na interface do utilizador, deverá ter em atenção que quanto maior for o ficheiro que pretende transferir, mais tempo o servidor irá demorar a processar a imagem.
      2. Seleccione o ficheiro de imagem que pretende apresentar e, em seguida, faça clique em Abrir.
      3. Faça clique em Actualizar para guardar as alterações.
    • Para pré-visualizar uma imagem:
      1. Faça clique em Pesquisar para localizar o ficheiro .jpeg ou .gif que pretende pré-visualizar e, em seguida, faça clique em Abrir.
        Nota: Não existem restrições do tamanho do ficheiro. No entanto, uma vez que a imagem é analisada para detectar qualquer vírus e redimensionada pelo servidor dos Perfis para pré-visualização, tenha em atenção que quanto maior for o ficheiro que seleccionar, mais tempo o servidor irá demorar a pré-visualizar a imagem.
      2. Faça clique em Pré-visualizar imagem para visualizar a imagem tal como esta será apresentada na página do perfil.
      3. Faça clique dentro da caixa de selecção e arraste a mesma para seleccionar a secção da imagem que pretende apresentar no perfil. Pode arrastar a extremidade da caixa de selecção para redimensionar a mesma, caso pretenda modificar o tamanho da área de selecção.
      4. Faça clique em Actualizar para guardar as alterações.
