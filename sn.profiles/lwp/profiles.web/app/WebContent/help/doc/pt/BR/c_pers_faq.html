Ajuda para Perfis

Perguntas Mais Freqüentes

Localize respostas a perguntas que os usuários de Perfis fazem com maior freqüência.

O que são Perfis?
Perfis fornecem um diretório das pessoas de sua organização, fornecendo informações necessárias para entrar em contato com elas e descobrir sobre seus projetos, habilidades e interesses.
Posso alterar meu perfil?
É possível utilizar a página Editar Perfil para fazer alterações no perfil. É possível atualizar os detalhes de contato, informações de segundo plano e carregar uma nova imagem ou arquivo de pronunciação utilizando as opções disponíveis nesta página.
O que são Marcações?
Uma marcação é uma palavra-chave descritiva designada ao seu perfil ou ao perfil de outra pessoa para identificar suas atividades de trabalho, habilidades ou interesses.
Como posso localizar pessoas?
Você pode pesquisar os perfis utilizando uma pesquisa simples para localizar pessoas por suas informações de contato, como nome e cargo, ou utilizando informações mais avançadas para localizar pessoas pesquisando seus interesses, experiência e projetos de trabalho.
Como construo minha própria rede?
Após localizar as pessoas com as quais deseja se conectar, inclua-as como colegas para construir uma rede profissional própria. Ao incluir alguém como um colega, esta pessoa recebe um convite por e-mail, perguntando se ela deseja unir-se à sua rede, podendo aceitar ou recusar.
