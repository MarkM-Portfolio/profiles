Criar um ficheiro áudio

Crie uma gravação dizendo o seu próprio nome, para que os outros utilizadores possam ouvir a pronúncia correcta do mesmo.

Sobre esta tarefa
Para criar um ficheiro áudio num sistema Microsoft Windows 2000 ou Microsoft Windows XP, execute os seguintes passos:
  1. Seleccione Iniciar > Todos os programas > Acessórios > Entretenimento > Gravador de áudio.
  2. Faça clique no botão Gravar, diga o nome completo, falando o mais claramente possível e, em seguida, faça clique no botão Parar. Faça clique no botão Reproduzir para ouvir a gravação.
  3. Verifique o tamanho da gravação ao fazer clique em Ficheiro > Propriedades. O tamanho dos dados tem de ser inferior a 100 KB.
  4. Para gravar novamente o ficheiro de áudio, faça clique em Ficheiro > Novo e seleccione Não quando for solicitado que guarde a gravação. Repita o passo 2 para gravar novamente. Siga estas sugestões para melhorar a qualidade da gravação:
    • Se a gravação for inaudível, certifique-se de que o microfone está activado. Seleccione Iniciar > Todos os programas > Acessórios > Entretenimento > Controlo do volume e, em seguida, seleccione Opções > Propriedades. Seleccione Gravar, Microfone e, em seguida, faça clique em OK. Arraste o cursor de deslocamento do volume do microfone até ao nível máximo.
    • Se a gravação não for compreensível ou se o ficheiro produzido for demasiado grande (superior a 100 KB), configure o gravador de áudio. Seleccione Iniciar > Todos os programas > Acessórios > Entretenimento > Gravador de áudio. Seleccione Ficheiro > Propriedades e, em seguida, faça clique em Converter agora. No campo Formatar, seleccione PCM e no campo Atributos, seleccione 8.000 kHz, 8 bit, Mono. Faça clique em OK e, em seguida, faça clique em OK para fechar a janela Propriedades.
  5. Após produzir uma gravação clara, audível e inferior a 100 KB, seleccione Ficheiro > Guardar como para a guardar. Introduza um nome para o ficheiro e seleccione um directório no qual este será guardado. Anote o nome e a localização, para que a possa obter novamente ao transferir o ficheiro para o perfil. O tipo de ficheiro predefinido no campo Guardar como tipo é .wav. Aceite a predefinição. Faça clique em Guardar.
Como proceder a seguir
Para criar um ficheiro áudio num sistema Linux, utilize o comando de gravação ou um utilitário de gravação de som.
