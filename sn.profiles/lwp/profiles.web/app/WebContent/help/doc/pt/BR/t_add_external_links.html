Incluindo Links Externos ao Perfil

Estenda seu perfil incluindo seus links da Web favoritos ao perfil.

Por Que e Quando Desempenhar Esta Tarefa
É possível incluir seus links da Web favoritos da rede de relacionamento social para a área Meus Links de seu perfil. Por exemplo, é possível que você deseje incluir um link ao perfil nos sites da Web Facebook ou LinkedIn para que você e seus colegas possam acessar facilmente os dados sociais de um único local.

Para incluir um link a seção Meus Links do perfil, conclua as etapas a seguir:

  1. Na página Meu Perfil, clique em Incluir link na área Meus Links.
  2. Digite um nome para o link no campo Nome. Este texto identifica o link na área Meus Links.
  3. Digite a URL do link no campo Link.
  4. Clique em Salvar.
