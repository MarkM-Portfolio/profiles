Ajuda para Perfis

Marcando Pessoas

Utilize palavras-chave para marcar os perfis de pessoas que possuem conhecimentos e habilidades de seu interesse.

Antes de Iniciar
Se o administrador do sistema possibilitou a marcação de pessoas, é possível marcar você mesmo e outros. É necessário efetuar login para criar ou modificar marcações.
Por Que e Quando Desempenhar Esta Tarefa
É possível marcar as pessoas utilizando o formulário de entrada de marcações na página de perfil. As marcações não podem ser ocultas ou particulares.

Para marcar uma pessoa:

  1. Abra a página de perfil para a pessoa que deseja marcar.
  2. Digite uma ou mais marcações no campo de entrada de marcações na barra lateral à esquerda. A digitação antecipada prevê a marcação que está sendo digitada com base na similaridade com as tags existentes.
  3. Clique em Incluir Marcação(ões) a este Perfil para incluir as marcações ao perfil da pessoa.
