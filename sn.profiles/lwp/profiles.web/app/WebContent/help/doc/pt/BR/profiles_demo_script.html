Script de Demo dos Perfis

Este é o texto para o demo de Perfis

Por Que e Quando Desempenhar Esta Tarefa

Perfis é um diretório das pessoas conectadas a você.

Este demo mostra como:
  • Localizar pessoas por nome
  • Acessar o cartão de visita de uma pessoa
  • Visualizar o perfil de alguém
  • Incluir um colega à rede
  • Editar os detalhes do perfil
  1. Utilize a página Pesquisar Perfis para localizar pessoas por cargo, localidade, conhecimento e marcações.
  2. Localize uma pessoa rapidamente utilizando o controle de pesquisa.
  3. Os resultados de pesquisa são retornados com informações como localidade, cargo e número do telefone.
  4. Se sua pesquisa retornar alguma comunidade, clique na guia Comunidades para exibir os resultados.
  5. Saiba mais sobre uma pessoa passando o mouse sobre o nome e clicando neste controle.
  6. O cartão de visita popup mostra uma fotografia e links para outras informações sobre a pessoa.
  7. Utilize o cartão de visita para entrar em contato com a pessoa utilizando o e-mail ou o bate-papo instantâneo.
  8. Clique em Mais Ações para acessar opções adicionais para entrar em contato com a pessoa.
  9. Clique no link Perfil no cartão de visita para exibir o perfil completo da pessoa.
  10. A página de perfil mostra informações sobre uma pessoa como profissão, localidade comercial e contato.
  11. Se uma cadeia de relatórios estiver disponível, utilize-a para navegar na estrutura da organização.
  12. Clique em Convidar Meus Colegas para incluir a pessoa à rede.
  13. É possível clicar no ícone vCard para salvar o vCard de uma pessoa e incluí-lo aos contatos.
  14. Localize mais informações sobre uma pessoa, clicando nos links Estrutura de Relatórios e Colegas.
  15. É possível acessar seu próprio perfil a qualquer momento, clicando na guia Meu Perfil.
  16. Role até a área Postagens Recentes para exibir as atualizações recentes feitas em outros recursos do Lotus Connections.
  17. Para atualizar o perfil, clique na guia Editar meu Perfil.
  18. Corrija o conteúdo do campo nas guias diferentes e, em seguida, clique em Atualizar para salvar suas alterações.
  19. Obrigado por assistir a demo. Para obter mais informações, verifique a página de ajuda dos Perfis.
