Ajuda dos Perfis

Transferir um ficheiro áudio

Edite o perfil para transferir uma gravação com a pronúncia correcta do seu nome.

Sobre esta tarefa

Melhore o perfil através da adição de uma gravação dizendo o seu próprio nome, para que os outros possam ouvir a pronúncia correcta do mesmo.

Para transferir um ficheiro áudio, execute os seguintes passos:
  1. Na página Editar o meu perfil, faça clique em Pronúncia.
  2. Faça clique em Pesquisar para pesquisar uma gravação em ficheiro áudio do nome do utilizador a ser correctamente pronunciado. O tamanho do ficheiro não poderá exceder os 100 KB.
  3. Seleccione o ficheiro que pretende transferir e, em seguida, faça clique em Abrir.
  4. Faça clique em Actualizar para aplicar alterações.
