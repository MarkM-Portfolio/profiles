Ajuda dos Perfis

Efectuar uma procura avançada

Efectue uma procura avançada para localizar perfis por informações de contacto, localização, etiqueta, experiência, conhecimentos técnicos, formação ou interesses. Pode efectuar uma procura avançada a partir de qualquer página nos Perfis.

Sobre esta tarefa

Uma procura avançada filtra os resultados, ao permitir efectuar procuras em vários campos do perfil em simultâneo. Consiste numa procura mais objectiva do que a procura rápida, que efectua a procura num único campo de cada vez. Quanto mais valores facultar nestes campos, mais limitada se torna a procura.

Para efectuar uma procura avançada de perfis, execute os seguintes passos:

  1. Seleccione Procura avançada de perfis a partir do controlo de procura na barra de título.
  2. O administrador pode adicionar ou remover campos do formulário Procura de perfis avançada, conforme necessário. Devido a esta função, os campos apresentados no formulário podem variar. Para procurar pessoas com base nas respectivas informações de contacto, introduza um valor num ou mais dos seguintes campos:
    Tabela 1. Campos de procura avançada
    Nome do campo Palavras a introduzir
    Nome de apresentação Introduza o nome completo da pessoa que pretende localizar.
    Nome próprio Introduza um nome próprio.
    Apelido Introduza um apelido.
    Etiquetas Introduza palavras-chave para procurar pessoas com base nas respectivas etiquetas.
    Cargo Introduza um cargo, para localizar os perfis das pessoas que executam essa função.
    Departamento Introduza uma palavra-chave para localizar uma pessoa que trabalha num departamento específico.
    Acerca de mim Introduza palavras-passe que descrevam o tipo de competências e experiência de que está à procura.
    Experiência profissional Introduza palavras-passe que descrevam o tipo de experiência no qual está interessado.
    Organização ou empresa Introduza o nome de uma organização ou empresa para localizar os perfis dos funcionários.
    Cidade Introduza uma cidade para encontrar os perfis de pessoas que trabalhem nessa localização.
    Estado Introduza um estado para encontrar perfis de pessoas que trabalhem nessa localização.
    País Introduza um país para localizar os perfis das pessoas que trabalham nessa localização.
    Endereço de correio electrónico Introduza um endereço de correio electrónico de negócios. Esta opção não está disponível, se a configuração impedir a apresentação de endereços de correio electrónico.
    Número de telefone Introduza um número de telefone. Este campo efectua uma procura em todos os cinco campos de número de telefone no perfil de uma pessoa.
  3. Faça clique em Procurar.
