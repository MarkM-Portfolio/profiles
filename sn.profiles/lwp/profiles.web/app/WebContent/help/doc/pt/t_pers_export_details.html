Ajuda dos Perfis

Exportar detalhes de contacto

Exporte o vCard de uma pessoa para a adicionar como novo contacto.

Sobre esta tarefa
Um cartão virtual (vCard) consiste num ficheiro que armazena informações de contacto básicas de um utilizador, seguindo a especificação do Consórcio de Correio da Internet. Pode adicionar novos contactos utilizando o sistema de correio electrónico para importar ficheiros vCard. Pode também descarregar o vCard a partir do cartão de visita dos Perfis.

Ao descarregar um vCard, é necessário seleccionar a opção de codificação de caracteres apropriada para o leitor de livro de endereços de que dispõe. A opção predefinida é Internacionalizado (UTF-8), que é adequada para utilizadores do IBM Lotus Notes e do Microsoft Outlook. No entanto, poderá ser necessário seleccionar uma codificação diferente se o cliente de correio electrónico for o Microsoft Outlook Express ou o Livro de endereços do Microsoft Windows.

Para guardar o vCard de um colega nos seus registos pessoais, execute os seguintes passos.
  1. Abra a página do perfil da pessoa que pretende adicionar como contacto.
  2. Faça clique em Descarregar vCard.
  3. Seleccione a codificação de caracteres que pretende utilizar.
  4. Faça clique em Descarregar para guardar o vCard da pessoa numa localização especificada.
