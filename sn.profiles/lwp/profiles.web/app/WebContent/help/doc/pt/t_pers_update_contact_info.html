Ajuda dos Perfis

Actualizar informações de contacto

Mantenha as informações de contacto actualizadas para assegurar que as pessoas podem entrar em contacto consigo.

Sobre esta tarefa
As informações contidas no perfil são obtidas a partir do directório de funcionários da organização e, uma vez que directórios diferentes armazenam informações diferentes, os campos que são apresentados no perfil poderão diferir dos campos predefinidos. Para além disso, o administrador pode preencher automaticamente um campo, para que não seja necessário que o utilizador o preencha. O administrador pode definir também um campo como só de leitura, se este recolher informações que não sejam relevantes para a organização.

Para editar as informações de contacto, execute os seguintes passos:

  1. Na página Editar o meu perfil, faça clique em Informações de contacto.
  2. Edite o conteúdo de qualquer um dos seguintes campos:
    Tabela 1. Campos de informações de contacto
    Nome do campo Descrição
    Edifício O nome do edifício.
    Andar O número do andar.
    Escritório O nome do escritório.
    Número do escritório O número de telefone do escritório.
    Número de telefone do IP Número da extensão.
    Número de telemóvel O número do telemóvel.
    Número do pager O número do pager.
    Número de fax O número do fax.
    Endereço de correio electrónico alternativo Um endereço de correio electrónico alternativo. Esta opção não está disponível se o administrador tiver desactivado a apresentação de endereços de correio electrónico para a sua implementação.
    Ligação do blogue Ligação para o blogue pessoal.
    Cargo O cargo oficial do utilizador.
    Assistente O nome do assistente.
    Fuso horário Faça clique na seta para baixo para seleccionar o fuso horário da localização de trabalho principal.
  3. Faça clique em Actualizar.
