Ajuda para Perfis

Executando uma Pesquisa Rápida

Use o controle de pesquisa no início de qualquer página em Perfis para pesquisas rápidas.

Por Que e Quando Desempenhar Esta Tarefa

Uma pesquisa rápida pesquisa um único campo no diretório de perfil.

Para executar uma pesquisa rápida, conclua as etapas a seguir:

  1. Clique na seta para baixo ao lado do campo Pesquisar na barra de navegação e selecione uma das seguintes opções:
    Tabela 1. Opções de Pesquisa de Perfis
    Opção Selecionar esta opção para
    Perfis por Nome Localizar pessoas pesquisando pelo nome. Digite um nome seguido pelo sobrenome. A digitação antecipada sugere nomes conforme você digita. Caso veja o nome que está procurando, clique nele para navegar diretamente para o perfil dessa pessoa.
    Nota: Essa opção é a configuração de pesquisa padrão.
    Perfis por Palavra-chave Pesquisar alguém com base em uma palavra-chave. Isso pesquisa perfis e todo o conteúdo associado, por exemplo, a experiência profissional e informações de segundo plano em um perfil.
  2. Insira uma palavra para pesquisar no campo de texto. Utilize um asterisco (*) como um caractere curinga.
  3. Clique em Pesquisar.
