Ajuda para Perfis

Visualizando Perfis

Um perfil fornece informações de contato e relacionadas ao trabalho de uma pessoa.

Visualize o perfil de uma pessoa para localizar as seguintes informações:

Informações para contato
Em geral, encontradas em um cartão de visita, como nome, cargo, empresa, localidade, números de telefone e endereços de e-mail, o perfil identifica a hora local da pessoa e, se aplicável, as informações de contato de um assistente. Clique no ícone de cartão de visita para exportar as informações de contato da pessoa como um arquivo de cartão virtual. Um vCard (cartão virtual) é um arquivo que armazena informações básicas de contato seguindo a especificação Internet Mail Consortium. É possível adicionar novos contatos utilizando o sistema de e-mail para importar arquivos vCard.
Nota: Seu administrador pode configurar o IBM® Lotus Connections para impedir a exibição de endereços de e-mail; nesse caso, eles não serão exibidos no perfil ou no cartão de visita de uma pessoa.
Quando o reconhecimento do Sametime está ativado, o status on-line da pessoa é exibido. O Lotus Connections Plug-in para Sametime deve ser instalado para que essa informação seja exibida.
Pronúncia
O link de pronúncia será exibido quando o proprietário do perfil tiver criado um arquivo de áudio e transferido-o por upload para o perfil. Clique nesse link para reproduzir uma gravação do nome da pessoa com a pronúncia correta.
Estrutura de relatório
Escolha uma das seguintes opções de visão:
Tabela 1. Visões de Estrutura de Relatório
Visão Descrição
Cadeia de relatórios integral Mostra a posição do proprietário do perfil, partindo dele até o início do gráfico organizacional.
Mesmo gerenciador Mostra as pessoas que também se reportam ao gerenciador do proprietário do perfil.
Pessoas gerenciadas Mostra a posição do proprietário do perfil no gráfico organizacional completo, incluindo as pessoas gerenciadas por ele.
Sobre Mim
Experiência de trabalho atual, qualificações técnicas, idiomas falados, interesses, experiência de trabalho anterior e educação. Essa informação é um campo Rich Text composto pelo proprietário do perfil e inclui todas as informações que o proprietário do perfil deseja compartilhar.
O Comitê
As postagens mais recentes para o comitê do proprietário do perfil. Essas informações incluem as mensagens de status e os comentários recentes do proprietário do perfil que outros usuários postaram para o comitê do proprietário do perfil.
Postagens Recentes
Esta seção usa visões tabuladas para exibir as atualizações mais recentes que o proprietário do perfil fez em outros recursos do Lotus Connections.
Marcações de perfil
Palavras-chave curtas usadas para identificar qualificações, áreas de interesse ou atividades profissionais do proprietário do perfil. Essas marcações podem ser incluídas pelo proprietário do perfil ou outra pessoa.
Ferramentas colaborativas
Quando os perfis estão integrados a outras ferramentas colaborativas, como o IBM Lotus Sametime, é possível ver informações de presença do proprietário do perfil, indicando se a pessoa está disponível para bate-papo.
Minha Rede
Esta seção mostra uma seleção das pessoas na rede do proprietário do perfil. Você pode clicar em Ver Todos para exibir uma lista de todos na rede da pessoa.
Meus Links
Links externos que os proprietários de perfil incluem como parte de seus perfis.
