Ajuda para Perfis

Alterando a Imagem do Perfil

Faça upload de uma foto atualizada para o perfil na página Editar meu Perfil.

Por Que e Quando Desempenhar Esta Tarefa

Você pode aprimorar seu perfil atualizando sua imagem periodicamente. Assegure-se de carregar uma imagem apropriada aos negócios.

Para atualizar sua imagem do perfil, execute as etapas a seguir:
  1. Na página Editar meu Perfil, clique em Foto.
  2. Para editar a imagem do perfil, faça um dos seguintes:
    • Para remover a foto existente, selecione Remover Imagem.
    • Para fazer upload de uma nova imagem:
      1. Clique em Procurar para localizar um arquivo .jpeg ou .gif contendo uma imagem sua para exibir no perfil.
        Nota: Não há restrições de tamanhos de arquivos. Entretanto, como a imagem é redimensionada automaticamente no servidor Perfis para se ajustar ao espaço alocado na interface com o usuário, observe que quanto maior o arquivo a ser carregado, mais longo será o tempo de processamento da imagem pelo servidor.
      2. Selecione o arquivo de imagem que deseja exibir e, em seguida, clique em Abrir.
      3. Clique em Atualizar para salvar as alterações.
    • Para visualizar uma imagem:
      1. Clique em Procurar para localizar o arquivo .jpeg ou .gif que deseja visualizar e, em seguida, clique em Abrir.
        Nota: Não há restrições de tamanhos de arquivos. Entretanto, como é feita verificação de vírus e redimensionamento da imagem pelo servidor Perfis para visualização, saiba que quanto maior o arquivo selecionado, mais longo será o tempo usado pelo servidor para visualizar a imagem.
      2. Clique em Visualizar Imagem para ver a imagem como ela será exibida na página de perfil.
      3. Clique dentro da caixa de seleção e arraste-a para selecionar a seção da imagem que deseja exibir no perfil. Você pode arrastar borda da caixa de seleção para redimensioná-la, se desejar modificar o tamanho da área de seleção.
      4. Clique em Atualizar para salvar as alterações.
