Modifica dell'immagine del profilo personale

Caricare una foto aggiornata per il profilo dalla pagina Modifica profilo personale.

Informazioni su questa attività

Si può migliorare il proprio profilo aggiornando, di tanto in tanto, l'immagine. Verificare che l'immagine caricata sia appropriata dal punto di vista lavorativo.

Per aggiornare l'immagine del profilo personale, eseguire i passi riportati di seguito:
  1. Dalla pagina Modifica profilo personale, fare clic su Foto.
  2. Per modificare l'immagine del profilo personale, eseguire una delle azioni riportate di seguito:
    • Per rimuovere la foto esistente, selezionare Rimuovi immagine.
    • Per caricare una nuova immagine, eseguire quanto riportato di seguito:
      1. Fare clic su Sfoglia per individuare un file .jpeg o .gif che contenga una fotografia da presentare nel profilo.
        Nota: non vi sono limiti relativi alla dimensione del file. Tuttavia, poiché l'immagine viene ridimensionata automaticamente sul server dei Profili per essere adattata allo spazio assegnato nell'interfaccia utente, tenere presente che maggiori sono le dimensioni del file caricato, più lungo sarà il tempo impiegato dal server per elaborare l'immagine.
      2. Selezionare il file immagine che si desidera visualizzare e fare clic su Apri.
      3. Fare clic su Aggiorna per salvare le modifiche.
    • Per visualizzare l'anteprima di un'immagine, eseguire quanto riportato di seguito:
      1. Fare clic su Sfoglia per individuare il file .jpeg o .gif che si desidera visualizzare in anteprima, quindi, fare clic su Apri.
        Nota: non vi sono limiti relativi alla dimensione del file. Tuttavia, poiché l'immagine viene sottoposta alla scansione antivirus e ridimensionata dal server Profili per l'anteprima, tenere presente che più grande è il file selezionato, maggiore sarà il tempo richiesto dal server per presentare l'anteprima dell'immagine.
      2. Fare clic su Anteprima immagine per visualizzare l'immagine come verrà visualizzata nella pagina del profilo.
      3. Fare clic all'interno della casella di selezione e trascinarla per selezionare la sezione dell'immagine che si desidera visualizzare nel profilo. È possibile trascinare il bordo della casella di selezione per ridimensionarla se si desidera modificare la dimensione dell'area di selezione.
      4. Fare clic su Aggiorna per salvare le modifiche.
