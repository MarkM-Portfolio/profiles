Assegnazione di tag alle persone

Utilizzare le parole chiave per assegnare tag ai profili di persone che hanno competenze e abilità alle quali si è interessati.

Prima di iniziare
Se l'amministratore di sistema ha abilitato l'assegnazione di tag alle persone, è possibile assegnare le tag. Per creare o modificare le tag è necessario aver effettuato il login.
Informazioni su questa attività
È possibile assegnare tag alle persone utilizzando il modulo di immissione delle tag sulla relativa pagina del profilo. Le tag non possono essere private o nascoste. Gli utenti possono verificare chi li ha taggati facendo clic sul numero accanto allo specifico tag per visualizzare un elenco delle persone che hanno aggiunto quel tag al proprio profilo.

Per assegnare un tag ad una persona:

  1. Aprire la pagina del profilo per la persona cui si desidera assegnare tag.
  2. Immettere uno o più tag nel campo Tag nella barra laterale sinistra. La funzione di completamento automatico predetermina la tag che si sta immettendo in base alla somiglianza con le tag esistenti.
  3. Fare clic su Aggiungi tag a questo profilo per aggiungere i tag al profilo della persona.
