Creazione di un file audio

Registrati mentre dici il tuo nome così gli altri sentiranno come si pronuncia correttamente.

Informazioni su questa attività
Per creare un file audio su un sistema Microsoft Windows 2000 o Microsoft Windows XP, eseguire i passi riportati di seguito:
  1. Seleziona Start > Tutti i Programmi > Accessori > Svago > Registratore di suoni.
  2. Fai clic sul bottone Registra, pronuncia il tuo nome completo, dicendolo nel modo più chiaro possibile e poi fai clic sul bottone Stop. Fai clic sul bottone Play per ascoltare la registrazione.
  3. Controlla le dimensioni della registrazione facendo clic su File > Proprietà. La dimensione dei dati deve essere inferiore a 100 KB.
  4. Per registrare nuovamente il file audio, fai clic su File > Nuovo, e seleziona Noquando ti si chiede di salvare la registrazione. Per registrare di nuovo, ripetere il Passo 2. Segui questi suggerimenti per migliorare la qualità della registrazione:
    • Se non riesci a sentire la registrazione, assicurati che il microfono sia acceso. Seleziona Start > Tutti i Programmi > Accessori > Svago > Controllo volume, poi seleziona Opzioni > Proprietà. Scegli Registrazione, seleziona Microfono, e poi fare clic su OK. Sposta il cursore del volume del microfono fino al livello massimo.
    • Se la registrazione non è chiara o il file creato è troppo grande (più di 100 KB), configura il registratore di suoni. Seleziona Start > Tutti i Programmi > Accessori > Svago > Registratore di suoni. Seleziona File > Proprietà, e poi fai clic su Converti. Nel campo Formato, seleziona PCM e nel campo Attributi, seleziona 8.000 kHz, 8 bit, Mono. Fare clic su OK, e poi fare clic su OK per chiudere la finestra delle Proprietà.
  5. Dopo aver creato una registrazione chiara, udibile, e inferiore a 100 KB, seleziona File > Salva con nome per salvarla. Scrivi il nome del file e scegli una cartella in cui salvarlo. Prendi nota del nome e della sua posizione, in modo da poterlo recuperare quando caricherai il file nel tuo profilo. Il tipo di file per difetto nel campo Salva come tipo è .wav. Accetta il tipo per difetto. Fare clic su Salva.
Operazioni successive
Per creare un file audio su un sistema Linux, utilizzare il comando record oppure utilizzare un programma di utilità di registrazione audio.
