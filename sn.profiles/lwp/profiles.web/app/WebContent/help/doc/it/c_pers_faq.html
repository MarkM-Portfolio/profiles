Aiuto sui Profili

Domande frequenti

Vi si trovano le risposte alle domande che gli utenti di Profili formulano più frequentemente.

Che cos'è Profili?
Profili fornisce un elenco delle persone nell'organizzazione, fornendo le informazioni necessarie per contattarle e quelle relative ai loro progetti, abilità e interessi.
È possibile modificare il profilo personale?
È possibile utilizzare la pagina Modifica profilo personale per apportare modifiche al proprio profilo. È possibile aggiornare i propri dettagli di contatto, le informazioni generali e caricare un nuovo file di pronuncia o di immagine utilizzando le opzioni disponibili su questa pagina.
Che cosa sono le tag?
Una tag è una parola chiave descrittiva che si assegna al proprio profilo o a quello di un altro utente per identificare le attività lavorative, le competenze o gli interessi.
Come individuare le persone?
È possibile ricercare i profili utilizzando una ricerca semplice per individuare le persone in base alle relative informazioni di contatto, ad esempio il nome e la qualifica professionale oppure utilizzare informazioni più avanzate per individuare le persone ricercando in base ai relativi interessi, esperienze e progetti di lavoro.
Come si crea una rete personale?
Dopo aver trovato le persone con cui interagire, aggiungerle alla rete per creare una serie di contatti utili. Quando si aggiunge qualcuno alla rete, quella persona riceve un invito mediante e-mail in cui viene richiesto di unirsi alla rete, che può essere accettato o rifiutato.
