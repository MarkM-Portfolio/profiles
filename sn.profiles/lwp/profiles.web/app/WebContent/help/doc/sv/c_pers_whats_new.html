Hjälp för profiler

Nyheter i den här versionen

Läs mer om de nya och uppdaterade funktioner som är utformade för att göra profiler enklare att använda så att du når ut till andra och gör dig själv synlig.

Lotus Connections 2.5

  • Du kan använda anslagstavlan till att hålla kontakten med personer i ditt nätverk och till att kommunicera med dem utan att lämna din profil.
  • Du kan lämna in ett statusmeddelande som används till att hålla personerna i ditt nätverk informerade om vad du gör.
  • I området Företagsetiketter på hemsidan för Profiler visas alla etiketter i Profiler. Du kan leta rätt på personer med en viss expertis genom att bläddra i etikettmolnet och klicka på etiketter som du är intresserad av.
  • genom att ändra visningsspråk kan du åsidosätta språkinställningen för webbläsaren och visa profiler på ett annat språk

Lotus Connections 2.0.1

  • Säkerhetsförbättringar innebär att administratören nu kan förhindra e-postadresser från att visas i i profilernas visitkort.

Lotus Connections 2.0

  • Bättre kapacitet att överföra grafik gör att du kan överföra en större bild till din profil och beskära den så att den blir som du vill ha den.
  • Bättre visitkort för profiler gör att du kan samverka med arbetskamrater på olika sätt, till exempel genom att se deras Sametime-status, lägga dem i ditt nätverk och starta en snabbchatt med dem.
  • Bättre etikettfunktioner gör att du kan tilldela etiketter med nyckelord till andra personer så att du kan definiera deras anknytning till dig och enkelt söka efter dem.
  • Rapportinformation visar vilka som har samma chef och vem olika personer har som chef.
  • Bättre sökfunktioner gör att du kan söka efter förnamn eller efternamn.
