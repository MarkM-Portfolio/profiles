Hjälp för profiler

Skapa en ljudfil

Du kan skapa en inspelning där du uttalar ditt namn som andra kan lyssna på.

Den här uppgiften
Du skapar en ljudfil i ettMicrosoft® Windows® 2000- eller Microsoft Windows XP-system genom att utföra följande steg:
  1. Välj Start > Program > Tillbehör > Underhållning > Ljudspelaren.
  2. Klicka på inspelningsknappen Spela in, tala in ditt fullständiga namn klart och tydligt och klicka sedan på knappen Stopp. Klicka på knappen Spela ut om du vill lyssna på inspelningen.
  3. Kontrollera storleken på inspelningen genom att välja Arkiv > Egenskaper. Datafilen får inte vara större än 100 kB.
  4. Om du vill spela in ljudfilen igen väljer du Arkiv > Nytt och sedan anger du att du inte vill spara inspelningen genom att klicka på Nej. Upprepa steg 2 och du vill göra om inspelningen. Om du vill förbättra kvaliteten på inspelningen följer du följande steg:
    • Om det inte går att höra något kontrollera du att mikrofonen är aktiverad. Välj Start > Program > Tillbehör > Underhållning > Volymkontrollen och välj sedan Alternativ > Egenskaper. Välj Inspelning, markera Mikrofon och klicka sedan på OK. Dra volymskjutreglaget för mikrofonen till den maximala nivån.
    • Om det inte går att höra ljudet eller om filen är för stor (över 100 kB) konfigurerar du Ljudinspelaren. Välj Start > Alla program > Tillbehör > Underhållning > Ljudinspelaren. Välj Arkiv > Egenskaper och klicka sedan på Konvertera. Välj PCM i fältet Format och 8,000 kHz; 8 bitar; Mono i fältet Attribut. Klicka på OK, och klicka sedan på OK så dialogrutan Egenskaper stängs.
  5. Efter att du har skapat en inspelning som är klar, hörbar och mindre än 100 kB sparar du den genom att välja Arkiv > Spara som. Ange ett namn för filen och en mapp att spara den i. Anteckna namnet och platsen, så att du kan hämta den när du ska överföra filen till profilen. Standardfilformatet i fältet Filformat är Ljud (*.wav). Använd standardformatet. Klicka på Spara.
Och sedan då?
Om du vill skapa en ljudfil i ett Linux®-system, använder du inspelningskommandot eller använder ett ljudinspelningsverktyg.
