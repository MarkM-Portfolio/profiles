Hjälp för profiler

Lägga till etiketter till din profil

Lägg till etiketter till din profil om du vill beskriva dina kunskaper, erfarenheter och intressen.

Den här uppgiften

Genom att tilldela etiketter till din profil kan du låta andra i företaget få information om dina kunskaper, din befattning, dina intressen och din utbildning. Du måste vara inloggad för att kunna lägga till etiketter till din profil.

Så här lägger du till etiketter till din profil:

  1. Klicka på fliken Min profil.
  2. Ange nyckelord som beskriver din befattning, din kompetens och dina intressen i fältet Etiketter. Avgränsa etiketterna med kommatecken eller blanktecken. Etiketterna konverteras automatiskt till gemener.
  3. Klicka på Lägg till etiketter till den här profilen.
