Hjälp för profiler

Uppdatera kontaktinformation

Håll din kontaktinformation aktuell så att personer kan få tag i dig.

Den här uppgiften
Informationen i din profil hämtas från företagets användarkatalog, och eftersom olika information lagras i olika användarkataloger kan eventuellt andra fält än standardfälten visas i din profil. Dessutom kan administratören fylla i fält automatiskt så att inte du behöver göra det. Administratören kan också ange fält som skrivskyddade om de inte är relevanta för företaget.

Så här redigerar du din kontaktinformation:

  1. På sidan Redigera min profil klickar du på Kontaktinformation.
  2. Redigera innehållet i följande fält:
    Tabell 1. Kontaktinformationsfält
    Fältnamn Beskrivning
    Byggnad Namnet på byggnaden.
    Våning Våningsnummer.
    Arbete Kontorets namn.
    Telefonnummer (arbete) Telefonnummer till arbetet.
    IP-telefoninummer Telefonnummer för IP-telefoni.
    Mobiltelefonnummer Mobiltelefonnummer.
    Personsökarnummer Personsökarnummer.
    Faxnummer Faxnummer.
    E-postadress (alternativ) En alternativ e-postadress. Alternativet är inte tillgängliga om din administratör har avaktiverat e-postadresser för din installation.
    Blogglänk Länk till din privata blogg.
    Befattning Din officiella befattning.
    Assistent Namnet på din assistent.
    Tidszon Tidszon för din primära arbetsplats. Om du vill välja tidszon klickar du på nedåtpilen.
  3. Klicka på Uppdatera.
