Hjälp för profiler

Söka efter profiler

Du kan söka efter personer efter kontaktinformation, som namn och befattning, eller efter mer avancerad information, som intressen, erfarenhet eller arbetsprojekt.

Det finns olika typer av sökning som du kan utföra i Profiler:
  • Använd sökkontrollen om du vill göra en enkel sökning baserad på ett enda sökfält. Den här kontrollen är tillgänglig längst upp till höger på alla sidor i Profiler.
  • Använd alternativen Profiler efter namn och Profiler efter nyckelord i snabbsökkontrollen till att söka efter en person efter namn eller nyckelord.
  • Om du vill använda de avancerade sökalternativen och utföra en mer avancerad sökning med flera sökfält klickar du på Avancerad sökning efter profiler.
  • Du kan använda etikettmolnet på hemsidan för Profiler till att söka bland alla profiletiketter för företaget. Om du vill visa en lista med personer med en viss etikett och relaterade etiketter som du kan använda till att förfina din sökning klickar du på en etikett. Om du vill söka efter profiler som innehåller en viss etikett klickar du på Sök efter etiketter, anger en etikett i fälten och sedan klickar du på Sök efter profiler som innehåller den här etiketten.

Du kan använda jokertecknet (*) för alla typer av sökningar. Om du till exempel anger namnet Pelle * returneras alla profiler för personer som heter Pelle i förnamn. Om du anger 555* som söksträng för telefonnummer visas alla nummer som börjar med 555.
