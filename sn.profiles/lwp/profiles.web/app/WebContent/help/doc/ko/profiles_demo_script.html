프로파일 데모 스크립트

프로파일 데모용 텍스트입니다.

이 태스크 정보

프로파일을 사용하여 조직에서 기술 및 전문 지식을 검색할 수 있습니다. 사교용 둉료 네트워크를 구성하고 최신 게시물을 추적하십시오.

이 데모에서 보여주는 수행 내용은 다음과 같습니다.
  • 이름별 사용자 찾기
  • 네트워크에 사용자 추가
  • 다른 사용자 게시판에 메시지 게시
  • 게시판 메시지에 댓글 달기
  • 상태 메시지 업데이트
  • 프로파일 편집
  1. 검색 컨트롤을 사용하여 빨리 찾으십시오.
  2. 검색 결과에는 주소, 직위 및 전화 번호 같은 정보가 함께 리턴됩니다.
  3. 개인 사용자의 이름 위에 마우스를 움직인 후 클릭하면 사용자에 대해 자세히 알 수 있는 명함이 표시됩니다.
  4. 팝업 명함에는 해당 개인 사용자의 사진과 기타 정보에 대한 링크가 표시됩니다.
  5. 명함으로 이메일을 사용하여 개인 사용자에게 연락하십시오.
  6. 개인 사용자에게 연락할 추가 옵션에 액세스하려면 자세한 조치를 클릭하십시오.
  7. 개인 사용자의 전체 프로파일을 보려면 프로파일 링크를 클릭하십시오.
  8. 프로파일 페이지에는 직책, 직장 주소, 연락처 세부사항 등 개인 사용자에 대한 정보가 표시됩니다.
  9. 보고 대상 체인을 사용할 수 있는 경우 이를 사용하여 조직의 구조를 더 잘 이해할 수 있습니다.
  10. 네트워크에 개인 사용자를 추가하려면 내 네트워크에 초대를 클릭하십시오.
  11. 사용자의 vCard를 저장하면 연락처에 이를 추가할 수도 있습니다.
  12. 게시판에 메시지를 게시하여 개인 사용자와 직접 의사 소통하십시오.
  13. 메시지를 입력하고 메시지 게시를 클릭하십시오.
  14. 최근 게시물 영역 아래로 이동하여 해당 사용자가 다른 기능에서 작성한 최근 업데이트 사항을 볼 수 있습니다.
  15. 내 프로파일을 클릭하면 언제든지 프로파일에 액세스할 수 있습니다.
  16. 시간을 내어 사용자 정보가 최신 상태인지 점검하고 게시판에서 최신 메시지를 확인하십시오.
  17. 메시지에 응답하려면 댓글 추가를 클릭하십시오.
  18. 상태 메시지를 업데이트하여 다른 사용자에게 사용자의 현재 상태를 알리십시오.
  19. 지금 어떤 작업을 하고 있습니까?를 클릭하고 메시지를 입력한 다음 저장을 클릭하십시오.
  20. 프로파일 세부사항을 업데이트하려면 내 프로파일 편집 탭을 클릭하십시오.
  21. 변경사항을 저장하려면 다른 탭에서 필드 컨텐츠를 수정한 후 업데이트를 클릭하십시오.
  22. 이제 기본사항을 알고 있으므로 로그인하고 연결하십시오.
  23. 데모를 시청해 주셔서 감사합니다. 추가 정보는 프로파일 도움말 페이지를 참조하십시오.
