프로파일 이미지 변경

내 프로파일 편집 페이지에서 프로파일에 대한 최신 사진을 업로드하십시오.

이 태스크 정보

가끔 이미지를 업데이트하여 프로파일을 강화하십시오. 비즈니스에 적합한 이미지로 업로드해야 합니다.

프로파일 이미지를 업데이트하려면 다음 단계를 수행하십시오.
  1. 내 프로파일 편집 페이지에서 사진을 클릭하십시오.
  2. 프로파일 이미지를 편집하려면 다음 중 하나를 수행하십시오.
    • 기존 사진을 제거하려면 이미지 제거를 선택하십시오.
    • 새 이미지를 업로드하려면 다음을 수행하십시오.
      1. 찾아보기를 클릭하여 프로파일에 표시할 사진이 있는 .jpeg 또는 .gif 파일을 찾으십시오.
        주: 파일 크기에 대한 제한은 없습니다. 그러나 이미지의 크기는 프로파일 서버에서 사용자 인터페이스에 할당된 공간에 맞게 자동으로 조정되므로 업로드하는 파일의 크기가 클수록 해당 서버가 이미지를 처리하는 시간이 오래 걸립니다.
      2. 표시하려는 이미지 파일을 선택한 후, 열기를 클릭하십시오.
      3. 변경사항을 저장하려면 업데이트를 클릭하십시오.
    • 이미지를 미리보려면 다음을 수행하십시오.
      1. 찾아보기를 클릭하여 미리 보려는 .jpeg 또는 .gif 파일을 찾고, 열기를 클릭하십시오.
        주: 파일 크기에 대한 제한은 없습니다. 그러나, 미리 보려는 이미지에 대해 프로파일 서버에서 바이러스를 검사하고 크기 조정하므로, 선택한 파일이 클수록 서버에서 이미지를 미리보는 데 시간이 오래 걸린다는 점에 유의하십시오.
      2. 이미지 미리보기를 클릭하여 프로파일 페이지에 표시되는 그대로 이미지를 보십시오.
      3. 선택 상자 안을 클릭하고 끌어다 프로파일에 표시할 이미지의 섹션을 선택하십시오. 선택 영역의 크기를 조정하려는 경우, 선택 상자의 테두리를 끌어 크기를 조정하십시오.
      4. 변경사항을 저장하려면 업데이트를 클릭하십시오.
