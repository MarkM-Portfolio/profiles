프로파일 도움말

Lotus Connections 검색

제목 표시줄의 검색 컨트롤을 사용하여 모든 IBM® Lotus® Connections 기능에서 정보를 검색할 수 있습니다.

이 태스크 정보
프로파일에서 모든 Lotus Connections 기능을 검색하려면 다음 단계를 완료하십시오.
  1. 프로파일의 아무 보기에서 제목 표시줄에 있는 검색 필드에 검색할 용어를 입력하십시오.
  2. 검색 필드 옆에 있는 아래로 화살표를 클릭하고 다음 옵션 중 하나를 선택하여 검색 범위를 정의하십시오.
    표 1. 검색 옵션
    옵션 설명
    모든 Connections 모든 Lotus Connections 기능에서 텍스트 문자열을 검색합니다.
    <feature_name> 개별 기능에서 텍스트 문자열을 검색합니다.
    고급 고급 검색 양식이 열립니다. 앙식을 사용하여 한 기능, 모든 기능 또는 기능 부분집합에서 단일 용어나 복수 용어를 검색할 수 있습니다.
  3. 검색을 클릭하십시오.
