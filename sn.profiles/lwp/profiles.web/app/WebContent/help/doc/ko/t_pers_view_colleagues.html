프로파일 도움말

네트워크 연락처 보기

누가 누구에게 연결되어 있는지 보려는 경우 프로파일을 사용하여 개인 네트워크 연락처를 보거나 다른 사용자가 이미 사용자 네트워크의 회원인지 여부를 확인할 수 있습니다.

이 태스크 정보
네트워크 연락처 페이지에서 사용자의 네트워크와 다른 사용자의 네트워크에 있는 사람들을 볼 수 있습니다.

사용자 프로파일 또는 다른 사용자 프로파일의 네트워크 연락처를 보려면 다음을 수행하십시오.

  1. 보려는 네트워크 연락처를 보유한 사용자의 프로파일 페이지를 여십시오.
  2. 페이지 오른쪽의 내 네트워크 영역에서 모두 표시를 클릭하십시오.
  3. 정렬 기준 목록에서 사용 가능한 다음 옵션을 사용하여 표시 내용을 정렬하십시오.
    표 1. 표시 옵션
    옵션 설명
    표시 이름 알파벳 순서로 동료를 나열합니다.
    최근 최신 순서로 동료를 나열합니다.
