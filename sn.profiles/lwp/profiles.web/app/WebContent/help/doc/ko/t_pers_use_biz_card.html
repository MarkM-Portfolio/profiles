프로파일 명함 사용

프로파일 명함을 사용하여 연락처 세부사항을 보고 해당 사용자와 연락할 수 있습니다.

이 태스크 정보
프로파일 명함은 개인 사용자의 프로파일 정보에 대한 유용한 스냅샷을 제공합니다. 직위 및 연락처 세부사항을 표시함은 물론, Sametime®용 IBM® Lotus® Connections 플러그인을 설치한 경우, 명함에 사용자의 현재 Sametime 상태가 표시됩니다.

해당 사용자가 사용한 Lotus Connections 기능에 대한 링크도 제공되어 여러 가지 조치를 수행할 수 있습니다. 예를 들어, 관리자가 이메일 주소를 표시하도록 구성한 경우 이메일을 보내거나 Sametime용 Lotus Connections 플러그인을 설치한 경우 이들과 인스턴트 대화를 시작할 수 있습니다.

명함에 액세스하려면 Lotus Connections 기능에서 개인 사용자의 이름 위에 마우스를 움직이고 표시되는 링크를 클릭합니다.

프로파일 명함을 사용하면 다음 작업을 수행할 수 있습니다.

  1. 명함의 맨 위에서 Lotus Connections 기능 링크를 클릭하여 사용자에 연관된 기능에 액세스하십시오.
  2. 메일 발송을 클릭하여 이메일을 보내십시오. 관리자가 Lotus Connections를 이메일 주소가 표시되지 않도록 구성한 경우, 이 옵션을 사용할 수 없습니다.
  3. 대화를 클릭하여 즉석 대화를 시작하십시오. 이 옵션이 표시되도록 Sametime용 Lotus Connections 플러그인을 설치해야 합니다.
  4. 통화를 눌러 텔레컨퍼런스 통화를 시작할 수 있습니다. 이 옵션이 표시되도록 Sametime Advanced Telephony를 설치해야 합니다.
  5. 자세한 조치 > 내 네트워크에 초대를 선택하고 초대 메시지를 입력한 후 초대 발송을 클릭하여 해당 사용자를 동료로 추가하십시오.
  6. 자세한 조치 > vCard 다운로드를 선택하고 저장을 클릭한 후 파일 위치를 찾아 저장을 다시 클릭하면 해당 사용자의 vCard를 다운로드하여 저장할 수 있습니다.
  7. Lotus Quickr™에서 개인 사용자 파일 라이브러리에 액세스하려면 파일을 클릭하십시오.
    이 옵션은 관리자가 Lotus Quickr과의 통합을 활성화하고 Lotus Quickr 링크를 표시하도록 명함을 구성한 경우에만 사용할 수 있습니다.
