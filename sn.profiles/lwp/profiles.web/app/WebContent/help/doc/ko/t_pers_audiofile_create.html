오디오 파일 작성

다른 사용자가 올바른 발음을 들을 수 있도록 이름을 직접 녹음합니다.

이 태스크 정보
Microsoft® Windows® 2000 또는 Microsoft Windows XP 시스템에서 오디오 파일을 작성하려면 다음 단계를 수행하십시오.
  1. 시작 > 모든 프로그램 > 보조 프로그램 > 엔터테인먼트 > 녹음기를 선택하십시오.
  2. 녹음 단추를 클릭하고, 이름을 정확하게 말한 후, 중지 단추를 클릭하십시오. 녹음을 들으려면 재생 단추를 클릭하십시오.
  3. 파일 > 속성을 클릭하여 녹음 내용의 크기를 선택하십시오. 데이터 크기는 100KB보다 작아야 합니다.
  4. 오디오 파일을 다시 녹음하려면 파일 > 새로 만들기를 클릭한 후 녹음 내용을 저장할 것인지 확인할 때 아니오를 선택하십시오. 2단계를 반복하여 다시 녹음하십시오. 녹음 품질을 향상시키려면 다음 팁을 따르십시오.
    • 녹음을 들을 수 없는 경우, 마이크로폰을 사용할 수 있는지 확인하십시오. 시작 > 모든 프로그램 > 보조 프로그램 > 엔터테인먼트 > 볼륨 조절을 선택한 후 옵션 > 속성을 선택하십시오. 녹음을 선택하고 마이크로폰을 선택한 후 확인을 클릭하십시오. 마이크로폰의 볼륨 슬라이더를 최대 레벨로 끄십시오.
    • 녹음이 분명하지 않거나 생성된 파일이 너무 크면(100KB를 넘음) 녹음기를 구성하십시오. 시작 > 모든 프로그램 > 보조 프로그램 > 엔터테인먼트 > 녹음기를 선택하십시오.파일 > 속성을 선택한 후 지금 변환을 클릭하십시오. 형식 필드에서 PCM을 선택하고 특성 필드에서 8.000kHz, 8비트, 모노를 선택하십시오. 확인을 클릭한 후 확인을 클릭하여 속성 창을 닫으십시오.
  5. 분명하게 들을 수 있고 100KB보다 작은 녹음을 생성한 후에 파일 > 다른 이름으로 저장을 선택하여 저장하십시오. 파일 이름을 입력하고 이를 저장할 디렉토리를 선택하십시오. 프로파일에 파일을 업로드할 때 검색할 수 있도록 이름과 위치를 적어 두십시오. 저장 형식 필드의 기본 파일 유형은 .wav입니다. 기본값을 승인하십시오. 저장을 클릭하십시오.
