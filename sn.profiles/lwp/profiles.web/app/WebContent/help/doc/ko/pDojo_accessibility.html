내게 필요한 옵션 기능

내게 필요한 옵션 기능을 사용하면 몸을 움직이기 힘들거나 시각 장애와 같은 신체적 장애가 있는 사용자가 정보 기술 제품을 제한없이 사용할 수 있게 됩니다.
