상태 업데이트 확인

상태 업데이트를 정기적으로 확인하여 네트워크 상태에 대한 최신 정보를 유지할 수 있습니다.

이 태스크 정보
프로파일 홈 페이지의 상태 업데이트 영역에 다양한 프로파일의 최신 상태 업데이트가 표시됩니다. 사용자와 연결된 다른 사용자의 상태 업데이트 또는 자신의 상태 업데이트를 볼 수 있도록 표시 내용을 필터링할 수 있습니다. 이 페이지에서 개인 상태 메시지를 업데이트하거나 다른 사용자 상태 메시지에 댓글을 추가할 수도 있습니다.

상태 업데이트를 확인하려면 다음 단계를 완료하십시오.

  1. 프로파일 홈 페이지를 여십시오.
  2. 상태 업데이트 영역에서 다음 중 하나를 수행하십시오.
    • 네트워크에 있는 다른 사용자의 최신 업데이트 목록을 표시하려면 내 네트워크의 상태 업데이트를 클릭하십시오.
    • 사용자 자신의 상태 업데이트를 표시하려면 내 상태 업데이트를 클릭하십시오.
  3. 옵션: 특정 상태 메시지에 댓글을 추가하려면 댓글 추가를 클릭하십시오.
