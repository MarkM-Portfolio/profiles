프로파일 도움말

네트워크에 사용자 추가

프로파일을 사용하여 작업 동료의 네트워크 및 유용한 연락처를 작성할 수 있습니다.

이 태스크 정보
다른 사용자에게 초대를 발송하여 사용자의 네트워크에 참여하도록 초대할 수 있습니다. 초대받은 사람은 메시지를 수신하면 동의 또는 무시할 수 있습니다.
주: 네트워크에 사용자를 추가하기 위해 프로파일 안에 있어야 하는 것은 아닙니다. 모든 IBM® Lotus® Connections 기능의 프로파일 명함에서 직접 사용자에게 연결할 수 있습니다. 예를 들어, 검색 결과로 개인의 프로파일이 리턴된 경우나 사용자가 같은 커뮤니티에 있는 회원을 사용자의 네크워크에 추가하려는 경우 명함을 사용하여 추가할 수 있습니다.

네트워크에 다른 사용자를 추가하려면 다음을 수행하십시오.

  1. 네트워크에 추가할 사용자의 프로파일 페이지를 여십시오.
  2. 내 네트워크에 초대를 클릭하십시오. 아직 프로파일 기능에 로그인하지 않은 경우 로그인 안내가 표시됩니다.
  3. 옵션: 네트워크에 참여하도록 초대하는 메시지를 입력하십시오.
  4. 초대 발송을 클릭하십시오.
