상태 메시지 업데이트

개인 상태 메시지를 업데이트하여 네트워크에 있는 다른 사용자에게 사용자의 현재 상태를 알릴 수 있습니다.

시작하기 전에
상태 메시지를 업데이트하려면 프로파일에 로그인되어 있어야 합니다.
이 태스크 정보
프로파일 사진 아래에 표시되는 개인 상태 메시지를 정기적으로 업데이트하여 네트워크에 있는 다른 사용자에게 사용자의 현재 상태를 알릴 수 있습니다. 예를 들어, 다른 사용자에게 미팅 준비로 바쁘거나 휴가 예정임을 알리기 위해 메시지를 게시할 수 있습니다.

상태 메시지는 사용자의 프로파일 페이지를 볼 수 있는 모든 사람에게 표시됩니다. 프로파일의 게시판 영역에 이전 상태 메시지도 표시되며, 다른 사용자가 이에 대해 댓글을 달 수 있습니다.

상태 메시지를 업데이트하려면 다음 단계를 완료하십시오.

  1. 내 프로파일 페이지에서 지금 어떤 작업을 하고 있습니까?를 클릭하십시오.
  2. 제공된 필드에 간단한 상태 메시지를 입력하십시오. 최대 500자를 입력할 수 있습니다.
  3. 상태 게시를 클릭하십시오.
