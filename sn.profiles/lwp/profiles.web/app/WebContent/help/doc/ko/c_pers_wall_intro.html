프로파일 도움말

게시판의 개념

모든 사용자의 프로파일 페이지에 사용자 네트워크의 최근 업데이트 항목을 표시하는 게시판이라는 영역이 있습니다. 다른 사용자가 개인의 게시판에 메시지와 댓글을 게시할 수 있으므로 프로파일을 벗어나지 않고 서로 손쉽게 직접 의사 소통할 수 있습니다.

게시판은 프로파일 소유자의 상태 메시지에 대한 최신 업데이트와 모든 IBM® Lotus® Connections 기능의 최근 업데이트 사항을 표시합니다. 이러한 직접적인 의사 소통은 네트워크의 특정 사용자와 최신 상태를 유지하려고 할 때 유용합니다. 프로파일 페이지로 이동하여 게시판을 살펴보면 Lotus Connections에서 해당 사용자의 최신 조치를 대략 알 수 있습니다. 프로파일 소유자와 다른 사용자가 업데이트 항목 또는 메시지에 대한 응답으로 댓글을 게시할 수 있습니다.

관리자는 조직의 사용자에 대해 게시판 기능을 사용할 것인지 여부를 선택할 수 있습니다. 기본적으로 게시판이 사용 가능합니다. 또한 관리자는 게시할 수 있는 사용자를 제어하는 게시판 설정을 구성할 수도 있습니다. 예를 들어, 관리자는 사용자 네트워크에 있는 사용자만 게시판에 댓글을 게시할 수 있도록 게시판 액세스를 제한할 수 있습니다. 이 설정은 글로벌로 구성됩니다.
