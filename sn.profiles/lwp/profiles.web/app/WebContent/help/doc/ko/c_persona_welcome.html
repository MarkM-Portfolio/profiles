프로파일 시작

프로파일에서 제공하는 조직의 사용자 디렉토리를 통해 이들에게 연락하는 방법과 프로젝트, 기술 및 관심사를 알아보는 방법을 얻을 수 있습니다.

프로파일은 적임자를 신속히 찾아 연락할 수 있도록 하여 협업을 촉진합니다.

표시 여부 늘리기. 프로파일을 사용하여 조직의 사용자가 당신이 누구이고 직업은 무엇이며, 작업 중인 프로젝트는 어떤 것이고 사용 중인 도구와 기술, 현재 해결하려고 노력 중인 문제점 및 무엇을 읽거나 쓰고 있는지 알게 하십시오. 프로파일에 내용을 입력하는 방법으로 시작하십시오. 관심 분야, 취미 같은 다양한 사교 정보와 전문 분야를 나타내는 태그를 추가하면 다른 사람들이 당신에 대해 더 잘 알 수 있습니다.

관여하기. 프로파일을 사용하여 조직에서 당신과 유사한 프로젝트를 작업 중이거나 비슷한 문제점에 대한 해결책을 찾고 있는 사람 및 이들이 작업을 단순화하기 위해 사용 중인 자원을 찾으십시오. 동료가 누구이고 어떤 관심사를 갖고 있으며 보유한 기술은 무엇인지, 공개 또는 애독하는 내용은 무엇인지를 확인하고 연락을 취해 보십시오. 연결 대상이 누구인지 확인하고 네트워크를 더욱 확장하십시오.

사용자 자신의 네트워크 구성. 프로파일을 사용하여 유용한 연결을 만들 수 있습니다. 필요한 기술과 전문 지식을 갖춘 사용자를 찾아 네트워크에 추가하십시오. 네트워크를 작성한 경우 프로파일 홈 페이지에 로그인하여 다른 사용자의 상태 업데이트를 확인함으로써 연결된 사용자들과 최신 상태를 유지할 수 있습니다.

프로파일을 사용하여 조직의 사용자에 대한 집단 지식을 활용하여 다른 사용자와 공동으로 작업할 수 있습니다.

프로파일 기능에 액세스

프로파일 기능에 로그인하려면, 제품을 구성한 관리자가 기능 홈 페이지로 지정한 웹 사이트로 이동하십시오. 관리자가 작성해 준 사용자 이름과 비밀번호를 제공하십시오. 기능에 사용자 액세스를 추가로 지정하려면 관리자에게 문의하십시오. 로그인에 성공하려면 먼저 적절한 사용자 디렉토리에 사용자를 추가해야 합니다.
