프로파일 도움말

태그 사용

태그는 작업 활동, 기술 또는 관심 분야를 식별하기 위해 사용자 프로파일에 지정하는 설명 키워드입니다. 시스템 관리자가 사용 가능하게 한 경우 다른 사람의 프로파일에 태그를 지정할 수도 있습니다.

사용자의 프로파일에 지정하는 태그는 내 프로파일 페이지의 태그 영역에 표시됩니다. 태그는 수행하는 일과 관심을 갖는 사항을 설명합니다. 태그를 작성할 때 관심 분야나 전문 기술 분야를 표시하는 키워드를 사용하십시오. 태그는 한 단어로 입력하거나 하이픈이나 밑줄로 연결된 여러 단어로 입력해야 합니다. 예를 들어, payroll, event-planning 및 human_resources는 모두 유효한 태그입니다.

프로파일 태깅 이유

기술 항목 공표

프로파일에 태그를 지정하여 기술 항목, 작업 역할, 관심사 및 교육을 조직의 다른 사용자에게 설명합니다.

사용자 찾기

관심사를 공유하거나 유사한 작업을 수행 중인 다른 사용자를 신속히 찾으려면 자체 프로파일 태그 중 하나를 클릭하십시오. 공통 태그를 공유하는 사용자는 좋은 자원일 수 있습니다. 프로파일에서 연락처 정보를 가져와 협업을 시작할 수 있습니다.

귀하를 태그 지정한 사용자를 확인하십시오. 특정 태그의 빈도 수를 클릭하면 해당 용어로 귀하를 태그 지정한 사용자의 목록이 표시됩니다. 빈도를 다시 클릭하면 정보가 숨겨집니다.

다른 사람의 프로파일에 태그를 지정하는 이유

연결 정의

동료의 프로파일에 태그를 지정하면 그 사용자와의 연결 상태를 확인할 수 있습니다. 예를 들어 특정 프로젝트에 대해 함께 작업하는 경우 해당 프로젝트의 이름으로 작업 동료에 태그를 지정하도록 선택할 수 있습니다. 다른 사람의 프로파일에 태그를 지정할 때는 그 프로파일을 보는 모든 사용자에게 해당 태그가 표시될 수 있습니다.

사용자 프로파일과 다른 사용자 프로파일에 태그를 지정하면 사용자와 동료가 조직에서 전문 기술 정보를 손쉽게 찾을 수 있습니다. 프로파일에 추가된 모든 태그가 프로파일 홈 페이지의 조직 태그 영역에 표시됩니다. 태그 클라우드를 찾아보고 관심있는 태그를 클릭하여 특정 영역에 대한 전문 기술 정보를 빠르게 찾을 수 있습니다. 또한 태그 찾기를 클릭하고 찾으려는 태그를 입력한 후 이 태그가 포함된 프로파일 검색을 클릭하여 고유 태그가 포함된 프로파일을 검색할 수 있습니다.
