프로파일 도움말

빠른 검색 수행

프로파일의 모든 페이지 맨 위에 있는 검색 컨트롤을 사용하여 빠른 검색을 수행할 수 있습니다.

이 태스크 정보

빠른 검색을 사용하면 프로파일 디렉토리에서 단일 필드를 검색합니다.

빠른 검색을 실행하려면 다음 단계를 완료하십시오.

  1. 탐색줄에서 검색 필드 옆에 있는 아래로 화살표를 클릭하고 다음 옵션 중 하나를 선택하십시오.
    표 1. 프로파일 검색 옵션
    옵션 수행 작업
    Profiles by Name 이름을 검색하여 사용자를 찾습니다. 이름 뒤에 성을 입력하십시오. 자동 검색은 사용자가 입력할 때 이름을 제안합니다. 찾고 있는 이름을 발견하면 해당 이름을 클릭하여 프로파일로 직접 탐색하십시오.
    주: 이 옵션은 기본 검색 설정입니다.
    Profiles by Keyword 키워드에 기반하여 사용자를 검색합니다. 이 옵션은 프로파일과 연관된 모든 컨텐츠(예: 프로파일에 있는 작업 경험과 경력 정보)를 검색합니다.
  2. 텍스트 필드에 검색할 단어를 입력하십시오. 와일드카드 문자로 별표(*)를 사용하십시오.
  3. 검색을 클릭하십시오.
