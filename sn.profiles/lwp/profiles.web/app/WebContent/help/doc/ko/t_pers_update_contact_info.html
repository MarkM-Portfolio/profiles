프로파일 도움말

연락처 정보 업데이트

다른 사용자가 연락할 수 있도록 사용자의 연락처 정보를 최신 상태로 유지합니다.

이 태스크 정보
사용자 프로파일의 정보는 조직의 인사 디렉토리에서 비롯되었고 디렉토리마다 다른 정보가 저장되어 있으므로 사용자 프로파일에 표시되는 필드는 기본 필드와 다를 수 있습니다. 또한 관리자가 필드의 내용을 자동으로 입력하므로 사용자가 내용을 입력할 필요가 없습니다. 관리자는 사용자 조직에 관련이 없는 정보를 수집하는 필드의 경우 읽기 전용으로 설정할 수도 있습니다.

연락처 정보를 편집하려면 다음 단계를 완료하십시오.

  1. 내 프로파일 편집 페이지에서 연락처 정보를 클릭하십시오.
  2. 다음 필드의 컨텐츠를 편집하십시오.
