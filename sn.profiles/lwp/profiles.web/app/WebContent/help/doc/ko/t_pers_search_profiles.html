프로파일 도움말

고급 검색 수행

고급 검색을 수행하면 연락처 정보, 주소, 태그, 경험 기술 지식, 교육 훈련 또는 관심 분야에 따라 프로파일을 찾을 수 있습니다. 프로파일의 모든 페이지에서 고급 검색을 수행할 수 있습니다.

이 태스크 정보

고급 검색의 경우 한 번에 여러 프로파일 필드를 검색할 수 있으므로 결과가 세분화되어 표시됩니다. 빠른 검색보다 대상이 지정된 검색으로, 한 번에 하나의 필드를 검색합니다. 이 필드에 입력한 값이 많을수록 검색 결과는 더 세분화됩니다.

고급 프로파일 검색을 수행하려면 다음 단계를 완료하십시오.

  1. 제목 표시줄의 검색 컨트롤에서 프로파일 고급 검색을 선택하십시오.
  2. 관리자는 고급 프로파일 검색 양식에서 필요에 따리 필드를 추가 또는 제거할 수 있습니다. 이 기능 때문에 양식에 표시되는 필드는 다양합니다. 연락처 정보에 기반하여 사용자를 검색하려면 다음 필드 중 하나 이상에 값을 입력하십시오.
    표 1. 고급 검색 필드
    필드 이름 입력할 단어
    표시 이름 찾을 사용자의 전체 이름을 입력하십시오.
    이름 이름을 입력하십시오.
    성 성을 입력하십시오.
    태그 태그에 기반하여 사용자를 검색하려면 키워드를 입력하십시오.
    직위 해당 작업 기능을 수행하는 사용자의 프로파일을 찾으려면 직위를 입력하십시오.
    부서 특정 부서에 소속된 사용자를 찾으려면 키워드를 입력하십시오.
    내 정보 찾고 있는 경험 및 기술 항목의 유형을 설명하는 키워드를 입력하십시오.
    경력 관심있는 경력의 유형을 설명하는 키워드를 입력하십시오.
    조직 또는 회사 직원 프로파일을 찾을 조직 또는 회사의 이름을 입력하십시오.
    구/군/시 찾을 프로파일의 사용자가 위치한 구/군/시를 입력하십시오.
    시/도 찾을 프로파일의 사용자가 위치한 시/도를 입력하십시오.
    국가 해당 위치에서 작업하는 사용자의 프로파일을 찾으려면 국가를 입력하십시오.
    이메일 주소 비즈니스 이메일 주소를 입력하십시오. 이 옵션은 이메일 주소가 표시되지 않도록 구성한 경우 사용할 수 없습니다.
    전화번호 전화번호를 입력하십시오. 사용자 프로파일에서 5개 전화번호 필드를 모두 검색합니다.
  3. 검색을 클릭하십시오.
