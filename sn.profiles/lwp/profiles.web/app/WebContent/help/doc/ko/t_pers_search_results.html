프로파일 도움말

검색 결과 보기

검색 결과가 너무 많은 경우에는 다음 작업을 수행해야 합니다.

시작하기 전에
프로파일에서 검색을 수행하면 검색 결과가 사용자 보기에 표시됩니다. 프로파일 검색 결과는 기본적으로 250개로 제한됩니다. 이 수치는 관리자에 의해 변경될 수 있습니다.
주: 프로파일 태그는 검색 결과에 포함되지 않습니다.
이 태스크 정보
검색 결과를 검토하려면 다음 단계를 완료하십시오.
  1. 개인 사용자의 이름 위에 마우스를 움직인 후 표시되는 팝업 대화 상자를 클릭하면 검색 결과로 리턴된 사용자의 명함을 볼 수 있습니다. 명함을 본 후 더 자세한 내용을 알고 싶을 때는 해당 사용자의 이름을 클릭하여 프로파일을 여십시오.
  2. 왼쪽의 태그 클라우드에서 태그를 클릭하여 결과를 필터링하면 특정 키워드로 태그 지정된 사용자만 표시됩니다.
  3. 보기의 맨 위 및 아래에서 페이지 번호 단추, 다음 또는 이전 단추를 클릭하여 검색 결과를 흝어볼 수 있습니다. 결과가 너무 많으면 다시 검색할 수 있습니다. 후속 검색에서는 검색 필드를 더 많이 채워 규정자를 더 많이 제공하십시오.
