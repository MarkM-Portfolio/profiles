Modification de la langue d'affichage

Si votre administrateur a activé cette fonction, vous pouvez modifier la langue de l'interface utilisateur afin que toutes les fonctions d'IBM Lotus Connections s'affichent dans votre langue de prédilection.

Avant de commencer
La fonction qui vous permet de modifier le paramètre de langue dans Lotus Connections est désactivée par défaut. Votre administrateur doit l'activer. Pour savoir si cette fonction est activée, recherchez votre nom dans la barre de navigation. Si une langue s'affiche à côté de votre nom, cela signifie que la fonction est activée.
Remarque : La modification de la langue d'affichage utilisée dans la fonction Profils n'a pas d'incidence sur la langue du contenu s'affichant dans les Profils. Par exemple, si la zone Informations personnelles de votre profil est rédigée en anglais et que vous remplacez la langue d'affichage des Profils par le français, le contenu de la zone Informations personnelles continue de s'afficher en anglais.
Pourquoi et quand exécuter cette tâche
Pour modifier la langue d'affichage, procédez comme suit :
  1. Cliquez sur la flèche en regard de la langue indiquée à côté de votre nom dans la barre de navigation.
  2. Sélectionnez une langue dans la liste.
    Remarque : Si la liste ne comprend pas votre langue préférée et que votre langue préférée est définie dans les paramètres d'environnement local du navigateur Web que vous utilisez, sélectionnez Paramètre du navigateur.
Résultats
La page est actualisée et l'interface utilisateur s'affiche dans la langue sélectionnée. Si vous ne voyez aucun changement, actualisez la page.
