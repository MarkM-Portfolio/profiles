Mise à jour de votre message de statut

Mettez à jour votre message de statut personnel afin que les personnes de votre réseau soient au courant des tâches sur lesquelles vous travaillez actuellement.

Avant de commencer
Vous devez être connecté aux Profils pour mettre à jour votre message de statut.
Pourquoi et quand exécuter cette tâche
Vous pouvez informer les personnes de votre réseau des tâches sur lesquelles vous travaillez actuellement, en mettant régulièrement à jour le message de statut personnel qui s'affiche sur votre image de profil. Par exemple, vous pouvez poster un message pour prévenir que vous êtes occupé à préparer une réunion ou que vous allez être absent toute la journée.

Votre message de statut est visible pour toute personne habilitée à afficher votre page de profil. Vos mises à jour de statut précédentes s'affichent également dans la zone Panneau de votre profil, où d'autres utilisateurs peuvent y ajouter des commentaires.

Pour mettre à jour les informations concernant votre message de statut, procédez comme suit :

  1. Sur la page Mon profil, cliquez sur Sur quel sujet travaillez-vous ?
  2. Entrez un bref message de statut dans la zone indiquée. Vous pouvez entrer un maximum de 500 caractères.
  3. Cliquez sur Poster un statut.
