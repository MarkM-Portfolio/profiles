Aide de Profils

Affichage des profils

Le profil d'une personne contient des informations de contact et des renseignements liés à son travail.

Affichez le profil d'une personne pour consulter les informations suivantes :

Informations de contact
Informations figurant généralement sur une carte de visite, par exemple le nom, l'intitulé du poste, l'entreprise, l'emplacement, les numéros de téléphone et les adresses électroniques. Le profil permet d'identifier l'heure locale de la personne et, le cas échéant, les informations de contact de son assistant. Cliquez sur l'icône Carte de visite pour exporter les informations de contact de la personne sous forme de fichier vCard (carte de visite virtuelle). Ce dernier permet de stocker les informations de contact essentielles conformément à la spécification du consortium IMC (Internet Mail Consortium). Vous pouvez ajouter de nouveaux contacts avec votre système de messagerie permettant d'importer ces fichiers vCard.
Remarque : Votre administrateur peut configurer IBM Lotus Connections pour que les adresses électroniques ne s'affichent pas : dans ce cas, les adresses électroniques n'apparaissent ni dans le profil ni sur la carte de visite de la personne.
Lorsque la compatibilité Sametime est activée, le statut en ligne de la personne s'affiche. Le plug-inLotus Connections pour Sametime doit être installé pour que cette information s'affiche.
Prononciation
Le lien Prononciation s'affiche si le propriétaire du profil a créé un fichier audio et qu'il l'a placé dans son profil. Cliquez sur ce lien pour écouter un enregistrement du nom de la personne prononcé correctement.
Structure hiérarchique
Choisissez l'une des options d'affichage suivantes :
Tableau 1. Vues de la structure hiérarchique
Vue Description
Chaîne hiérarchique complète Affiche le poste du propriétaire du profil en remontant jusqu'au sommet de l'organigramme.
Responsable identique Affiche les personnes gérées par le responsable du propriétaire du profil.
Personnes gérées Affiche le poste du propriétaire du profil au sein de l'organigramme complet, y compris les personnes sous ses ordres.
Informations personnelles
Expérience professionnelle actuelle, compétences techniques, langues parlées, centres d'intérêt, expérience professionnelle passée et formation. Il s'agit d'un champ de texte enrichi créé par le propriétaire du profil et qui contient toutes les informations qu'il souhaite partager.
Le Panneau
Regroupe les derniers articles postés sur le panneau du propriétaire du profil. Ces informations comprennent les messages d'état récents du propriétaire du profil et les commentaires que les autres utilisateurs ont posté sur le panneau du propriétaire du profil.
Articles récents
Cette section utilise des vues répertoriées dans des onglets et contenant les dernières mises à jour des autres fonctions de Lotus Connections.
Etiquettes de profil
Mots clés permettant d'identifier les compétences, les centres d'intérêt ou les activités professionnelles du propriétaire du profil. Ces étiquette peuvent être ajoutées par le propriétaire du profil ou par une autre personne.
Outils de collaboration
Si les profils sont intégrés à d'autres outils de collaboration, comme IBM Lotus Sametime, vous pouvez afficher les informations de présence du propriétaire du profil qui indiquent s'il est disponible pour une discussion en ligne.
Mon réseau
Cette section affiche une sélection des personnes dans le réseau du propriétaire du profil. Vous pouvez cliquer sur Afficher tout pour afficher la liste de toutes les personnes figurant dans le réseau de cette personne.
Mes liens
Liens externes que les propriétaires de profil incluent dans leur profil.
