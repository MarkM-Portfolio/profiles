Exportation des détails de contact

Exporter la vCard (carte de visite virtuelle) d'une personne pour l'ajouter comme nouveau contact.

Pourquoi et quand exécuter cette tâche
Une carte de visite virtuelle (vCard) est un fichier qui permet de stocker les informations essentielles permettant de contacter un utilisateur conformément à la spécification du consortium IMC (Internet Mail Consortium. Vous pouvez ajouter de nouveaux contacts en vous servant de votre système de messagerie pour importer les fichiers de carte de visite virtuelle (vCard). Il est également possible de télécharger la vCard depuis la carte de visite de Profils.

Lorsque vous téléchargez une vCard, vous devez sélectionner l'option de codage de caractères qui est appropriée pour votre programme de lecture du carnet d'adresses. L'option par défaut est Internationalisé (UTF-8) qui convient aux utilisateurs d'IBM Lotus Notes et Microsoft Outlook. Toutefois, il se peut que vous ayez besoin de sélectionner un codage différent si votre client de messagerie est Microsoft Outlook Express ou Microsoft Windows Address Book.

Pour enregistrer la vCard d'un(e) collègue dans vos dossiers personnels, procédez comme suit.
  1. Ouvrez la page de profil de la personne que vous souhaitez ajouter comme contact.
  2. Cliquez sur Télécharger la vCard.
  3. Sélectionnez le codage de caractères que vous voulez utiliser.
  4. Cliquez sur Télécharger pour enregistrer la vCard de la personne à l'emplacement indiqué.
