Recherche dans les profils

Vous pouvez effectuer une recherche dans les profils pour identifier des personnes grâce à leurs informations de contact, comme leur nom et l'intitulé de leur poste, ou grâce à des informations plus spécifiques, comme leurs centres d'intérêt, leur expérience et leurs projets professionnels.

Plusieurs types de recherche sont possibles dans Profils :
  • Utilisez la commande de recherche pour effectuer une recherche simple basée sur seule zone de recherche. Cette commande est disponible dans l'angle supérieur droit de toutes les pages de Profils.
  • Recherchez une personne par son nom ou par mot clé avec les options Profils par nom et Profils par mot clé accessibles via l'option de recherche rapide.
  • Cliquez sur Recherche avancée des profils pour utiliser les options de recherche avancée et effectuer une recherche plus ciblée basée sur plusieurs zones de recherche.
  • Utilisez le nuage d'étiquettes dans la page d'accueil de Profils pour rechercher toutes les étiquette de profil de votre organisation. Cliquez sur une étiquette pour afficher la liste des personnes ayant cette étiquette et afficher les étiquette apparentées afin d'affiner votre recherche. Pour rechercher des profils contenant une étiquette spécifique, cliquez sur Rechercher des étiquettes, entrez une étiquette dans la zone affichée et cliquez sur Rechercher des profils contenant cette étiquette.

Le caractère générique (*) est pris en charge pour tous les types de recherche. Par exemple, si vous saisissez Ethan * comme chaîne de recherche de noms, vous obtiendrez tous les profils des personnes dont le prénom est Ethan. Si vous saisissez 555* comme chaîne de recherche de numéros téléphoniques, vous obtiendrez tous les numéros commençant par 555.
