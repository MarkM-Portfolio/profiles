Aide de Profils

Affichage des contacts réseau

Vous souhaitez savoir avec qui sont connectés vos collègues ? Utilisez les Profils pour afficher les contacts réseau d'une personne ou vérifier si une personne fait déjà partie de votre réseau.

Pourquoi et quand exécuter cette tâche
Vous pourrez voir les personnes de votre réseau ou du réseau d'autres personnes à l'aide de la page Contacts réseau.

Pour afficher les contacts réseau de votre profil ou le profil d'une autre personne :

  1. Ouvrez la page de profil de la personne dont vous souhaitez voir les contacts réseau.
  2. Cliquez sur Afficher tout dans la zone Mon réseau dans la partie droite de la page.
  3. Triez l'affichage à l'aide des options suivantes apparaissant dans la liste Trier par :
    Tableau 1. Options d'affichage
    Option Description
    Nom affiché Affiche la liste des collègues dans l'ordre alphabétique.
    Récent Affiche la liste des collègues dans l'ordre du plus récent.
