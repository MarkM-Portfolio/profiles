Aide de Profils

Utilisation de la recherche avancée

Effectuer une recherche avancée pour trouver des profils par information de contact, étiquette, expérience, connaissances techniques, formation ou centre d'intérêt. Vous pouvez effectuer une recherche avancée à partir de n'importe quelle page dans les Profils.

Pourquoi et quand exécuter cette tâche

Une recherche avancée affine les résultats de recherche en vous permettant d'analyser plusieurs zones de profils simultanément. Il s'agit donc d'une recherche plus ciblée qu'une recherche rapide dont l'analyse se réduit à une zone unique à la fois. Plus vous indiquerez de valeurs dans ces zones, plus la recherche sera ciblée.

Pour effectuer une recherche de profil avancée, procédez comme suit :

  1. Sélectionnez Recherche avancée des profils dans la commande de recherche dans la barre de titre.
  2. L'administrateur peut ajouter ou supprimer des zones du formulaire Recherche avancée de profils selon les besoins. Grâce à cette fonction, les zones s'affichant dans le formulaire peuvent varier. Pour rechercher des personnes en fonction de leurs informations de contact, tapez une valeur dans une ou plusieurs des zones suivantes :
    Tableau 1. Zones de recherche avancée
    Nom de zone Termes à saisir
    Nom affiché Tapez le nom complet de la personne recherchée.
    Prénom Tapez un prénom.
    Nom Tapez un nom de famille.
    Etiquettes Entrez des mots clés pour rechercher des personnes en fonction de leurs étiquettes.
    Intitulé de poste Saisissez un intitulé de poste pour identifier les profils des personnes qui exercent cette fonction.
    Service Entrez un mot clé pour rechercher une personne travaillant dans un service précis.
    Informations personnelles Saisissez les mots clés décrivant le type de compétences et d'expérience recherchées.
    Formation Saisissez les mots clés décrivant le type de formation qui vous intéresse.
    Organisation ou entreprise Saisissez le nom d'une organisation ou d'une entreprise pour trouver les profils des employés.
    Ville Saisissez le nom d'une ville pour trouver les profils des personnes qui y travaillent.
    Etat Saisissez le nom d'un Etat pour trouver les profils des personnes qui y travaillent.
    Pays Saisissez le nom d'un pays pour identifier les profils des personnes qui y travaillent.
    Adresse électronique Saisissez une adresse de messagerie électronique professionnelle. Cette option n'est pas disponible lorsque la configuration empêche l'affichage des adresses électroniques.
    Numéro de téléphone Saisissez un numéro de téléphone. Cette zone recherche les cinq numéros de téléphone du profil d'une personne.
  3. Cliquez sur Rechercher.
