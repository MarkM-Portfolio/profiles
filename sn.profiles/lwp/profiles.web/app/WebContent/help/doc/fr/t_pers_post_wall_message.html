Aide de Profils

Poster des messages sur le Panneau

Vous pouvez communiquer directement avec les personnes de votre réseau en postant des messages ou des commentaires dans la zone Panneau de leur profil.

Avant de commencer
Vous devez être connecté aux Profils pour poster un message sur votre propre profil ou celui d'une autre personne.
Pourquoi et quand exécuter cette tâche
Tous les utilisateurs des profils disposent d'une zone Panneau qui affiche les les mises à jours et les messages de statut les plus récents dans toutes les fonctions d'IBM Lotus. Vous pouvez communiquer avec vos collègues en postant un message directement dans leur profil.

Pour poster un message sur le panneau d'une personne, procédez comme suit :

  1. Sur la page de profil d'une personne, cliquez dans la zone du message en haut de la zone Panneau où s'affiche Ecrire un message sur ce panneau de profil.
  2. Entrez votre message.
  3. Cliquez sur Poster un message.
Résultats
Votre message s'affiche en haut de la zone Panneau, où le propriétaire du profil et les autres utilisateurs peuvent y répondre. Le propriétaire du profil reçoit un e-mail l'avertissant qu'un nouveau message a été posté pour son profil.
