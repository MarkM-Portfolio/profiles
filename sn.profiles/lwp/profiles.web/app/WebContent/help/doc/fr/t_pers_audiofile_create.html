Création d'un fichier audio

Créer un enregistrement de la prononciation correcte de votre nom.

Pourquoi et quand exécuter cette tâche
Pour créer un fichier audio avec un ordinateur Microsoft Windows 2000 ou Microsoft Windows XP, procédez comme suit :
  1. Sélectionnez Démarrer > Tous les programmes > Accessoires > Divertissement > Magnétophone.
  2. Cliquez sur le bouton Enregistrer, énoncez votre nom complet en parlant le plus clairement possible, puis cliquez sur le bouton Arrêter. Cliquez sur le bouton Lecture pour écouter l'enregistrement.
  3. Vérifiez la taille de l'enregistrement en cliquant sur Fichier > Propriétés. Le volume de données doit être inférieur à 100 Ko.
  4. Pour réenregistrer le fichier audio, cliquez sur Fichier > Nouveau, puis sélectionnez Non lorsque vous êtes invité à sauvegarder l'enregistrement. Répétez l'étape 2 pour réenregistrer. Suivez les conseils suivants pour améliorer la qualité de l'enregistrement :
    • Si l'enregistrement est inaudible, assurez-vous que le micro est activé. Sélectionnez Démarrer > Tous les programmes > Accessoires > Divertissement > Contrôle du volume, puis sélectionnez Options > Propriétés. Choisissez Enregistrement, sélectionnez Microphone, puis cliquez sur OK. Montez au maximum le volume du micro.
    • Si la qualité de l'enregistrement est insuffisante ou si le fichier généré est trop volumineux (plus de 100 Ko), configurez le magnétophone. Sélectionnez Démarrer > Tous les programmes > Accessoires > Divertissement > Magnétophone. Sélectionnez Fichier > Propriétés, puis cliquez sur Convertir maintenant. Dans le champ Format, sélectionnez PCM et dans le champ Attributs, sélectionnez 8 000 kHz, 8 bits, Mono. Cliquez sur OK, puis cliquez une nouvelle fois sur OK pour fermer la fenêtre Propriétés.
  5. Une fois que vous avez créé un enregistrement dont le son est clair, audible et d'une taille inférieure à 100 Ko, sélectionnez Fichier > Enregistrer sous pour l'enregistrer. Attribuez un nom au fichier et choisissez un répertoire d'enregistrement. Notez son nom et son emplacement, afin de pouvoir le retrouver lorsque vous le chargerez dans votre profil. Le type de fichier par défaut dans la zone Enregistrer sous le type est .wav. Acceptez le format par défaut. Cliquez sur Enregistrer.
Que faire ensuite
Pour créer un fichier audio avec un ordinateur Linux, utilisez la commande d'enregistrement ou un utilitaire d'enregistrement sonore.
