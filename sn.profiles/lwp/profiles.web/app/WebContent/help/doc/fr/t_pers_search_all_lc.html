Recherche dans tout Lotus Connections

Utilisez la commande de recherche dans la barre de titre pour rechercher des informations dans toutes les fonctions d'IBM Lotus Connections.

Pourquoi et quand exécuter cette tâche
Pour rechercher toutes les fonctions Lotus Connections à partir des Profils, procédez comme suit :
  1. A partir de n'importe quel vue dans les Profils, saisissez le terme que vous voulez rechercher dans la zone Rechercher dans la barre de titre.
  2. Définissez la portée de la recherche en cliquant sur la flèche de défilement vers le bas en regard de la zone Rechercher, puis sélectionnez l'une des options suivantes :
    Tableau 1. Options de recherche
    Option Description
    Toutes les connexions Recherche une chaîne de texte dans tous les fonctions de Lotus Connections.
    <nom_fonction> Recherche une chaîne de texte dans une fonction individuelle.
    Avancé Ouvre le formulaire Recherche avancée. Utilisez le formulaire pour rechercher un seul ou plusieurs termes dans une fonction, toutes les fonctions ou un sous-ensemble de fonctions.
  3. Cliquez sur Rechercher.
