Hilfe zu "Profile"

Ihre Profilgrafik ändern

Laden Sie ein aktuelles Foto für Ihr Profil von der Seite Mein Profil bearbeiten hoch.

Warum und wann dieser Vorgang ausgeführt wird

Sie können Ihr Profil erweitern, indem Sie Ihre Grafik von Zeit zu Zeit aktualisieren. Achten Sie darauf, dass Sie eine für geschäftliche Zwecke angemessene Grafik hochladen.

Gehen Sie wie folgt vor, um Ihre Profilgrafik zu aktualisieren:
  1. Klicken Sie auf der Seite Mein Profil bearbeiten auf Foto.
  2. Führen Sie eine der folgenden Aktionen aus, um Ihre Profilgrafik zu bearbeiten:
    • Um das vorhandene Foto zu entfernen, wählen Sie Grafik entfernen aus.
    • Gehen Sie wie folgt vor, um eine neue Grafik hochzuladen:
      1. Klicken Sie auf Durchsuchen, um nach einer .jpeg- oder .gif-Datei mit einem Bild von Ihnen zu suchen, das im Profil angezeigt werden soll.
        Anmerkung: Es gibt keine Einschränkungen bezüglich der Dateigröße. Da die Größe der Grafik jedoch auf dem Profile-Server automatisch an den zugeordneten Bereich in der Benutzerschnittstelle angepasst wird, sollten Sie bedenken, dass sich mit zunehmender Größe der Datei, die Sie hochladen, auch die Zeit erhöht, die der Server zum Verarbeiten der Grafikbenötigt.
      2. Wählen Sie die Grafikdatei aus, die Sie anzeigen möchten, und klicken Sie anschließend auf Öffnen.
      3. Klicken Sie auf Aktualisieren, um Ihre Änderungen zu speichern.
    • Gehen Sie wie folgt vor, um eine Voranzeige für eine Grafik aufzurufen:
      1. Klicken Sie auf Durchsuchen, um nach der .jpeg- oder .gif-Datei zu suchen, von der Sie eine Voranzeige aufrufen möchten, und klicken Sie anschließend auf Öffnen.
        Anmerkung: Es gibt keine Einschränkungen bezüglich der Dateigröße. Da die Grafik jedoch auf Viren überprüft wird und von dem Profilserver die Größe der Grafik zum Aufrufen einer Voranzeige geändert wird, sollten Sie bedenken, dass sich mit zunehmender Größe der Datei, die Sie auswählen, auch die Zeitdauer erhöht, die der Server zum Aufrufen einer Voranzeige für eine Grafik benötigt.
      2. Klicken Sie auf die Option für die Voranzeige der Grafik, um die Grafik so anzuzeigen, wie sie auf Ihrer Profilseite angezeigt wird.
      3. Klicken Sie in das Auswahlfenster und ziehen Sie es, um einen Bereich der Grafik auszuwählen, der in Ihrem Profil angezeigt werden soll. Sie können den Rand des Auswahlfensters so ziehen, dass sich die Größe ändert, wenn Sie die Größe des Auswahlbereichs ändern möchten.
      4. Klicken Sie auf Aktualisieren, um Ihre Änderungen zu speichern.
