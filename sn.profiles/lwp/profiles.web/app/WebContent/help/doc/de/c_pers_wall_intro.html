Hilfe zu "Profile"

Was ist das Board?

Jeder Benutzer verfügt über einen Bereich auf seiner Profile-Seite, der als Board bezeichnet wird und in dem die letzten Aktualisierungen für das Netzwerk dieses Benutzers angezeigt werden. Andere Benutzer können im Board einer Person Nachrichten und Kommentare veröffentlichen. Dies ermöglicht es ihnen, direkt mit dieser Person zu kommunizieren, ohne "Profile" zu verlassen.

Im Board werden die letzten Aktualisierungen der Statusnachrichten des Profileigentümers sowie die letzten Aktualisierungen für alle IBM Lotus Connections-Komponenten angezeigt. Diese direkte Kommunikation ist sehr nützlich, wenn Sie bezüglich einer bestimmten Person im Netzwerk auf dem Laufenden bleiben möchten. Sie können die Profilseiten der Person aufrufen und schnell ihr Board überprüfen, wo sie eine Zusammenfassung ihrer letzten Aktionen in Lotus Connections sehen können. Der Profileigentümer und andere Benutzer können Kommentare als Antwort auf alle Aktualisierungseinträge oder Nachrichten veröffentlichen.

Der Administrator kann auswählen, ob die Komponente "Board" für Benutzer in Ihrem Unternehmen aktiviert oder inaktiviert wird. Standardmäßig ist das Board aktiviert. Der Administrator kann auch Einstellungen für das Board konfigurieren, um zu steuern, wer darin etwas veröffentlichen kann. Der Administrator kann z. B. auswählen, den Zugriff auf das Board so einzuschränken, dass nur die Personen im Netzwerk eines Benutzers Kommentare im Board dieses Benutzers veröffentlichen können. Diese Einstellungen werden global konfiguriert.
