Hilfe zu "Profile"

Externe Links zu Ihrem Profil hinzufügen

Erweitern Sie Ihr Profil, indem Sie Ihre Favoriten unter den Web-Links zur Profilseite hinzufügen.

Warum und wann dieser Vorgang ausgeführt wird
Sie können Ihre Favoriten unter den Web-Links zur sozialen Vernetzung zum Bereich Meine Links Ihres Profils hinzufügen. So möchten Sie möglicherweise einen Link zu Ihrem Profil auf der Website "Facebook" oder "LinkedIn" aufnehmen, so dass Sie und Ihre Kollegen von einer einzelnen Position aus einfach auf Ihre sozialen Daten zugreifen können.

Gehen Sie wie folgt vor, um einen Link zum Bereich "Meine Links" Ihres Profils hinzuzufügen:

  1. Klicken Sie auf der Seite Mein Profil im Bereich Meine Links auf Link hinzufügen.
  2. Geben Sie im Feld Name einen Namen für den Link ein. Durch diesen Text wird der Link im Abschnitt Meine Links gekennzeichnet.
  3. Geben Sie im Feld Link den URL für den Link ein.
  4. Klicken Sie auf Speichern.
