Hilfe zu "Profile"

Schnellsuche durchführen

Verwenden Sie das Suchsteuerelement oben auf den einzelnen Seiten in "Profile" für Schnellsuchen.

Warum und wann dieser Vorgang ausgeführt wird

Bei einer Schnellsuche wird im Profilverzeichnis nach einem einzelnen Feld gesucht.

Gehen Sie wie folgt vor, um eine Schnellsuche durchzuführen:

  1. Klicken Sie auf den Abwärtspfeil neben dem Feld Suchen in der Navigationsleiste und wählen Sie eine der folgenden Optionen aus:
    Tabelle 1. Suchoptionen in "Profile"
    Option Wählen Sie diese Option für den folgenden Zweck aus
    Profile nach Namen Suchen einer Person anhand ihres Namens. Geben Sie einen Vornamen gefolgt von einem Nachnamen ein. Während der Eingabe werden von der Schnelladressierung Namen vorgeschlagen. Wenn der Name, nach dem Sie suchen, angezeigt wird, klicken Sie darauf, um direkt zum Profil dieser Person zu navigieren.
    Anmerkung: Diese Option ist die Standardsucheinstellung.
    Profile nach Schlüsselwort Suchen nach einer Person anhand eines Schlüsselworts. Dadurch werden Profile und der gesamte zugeordnete Inhalt, z. B. Berufserfahrung und Hintergrundinformationen in einem Profil, durchsucht.
  2. Geben Sie im Textfeld ein Wort ein, nach dem gesucht werden soll. Verwenden Sie einen Stern (*) als Platzhalterzeichen.
  3. Klicken Sie auf Suche.
