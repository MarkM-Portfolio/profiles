Hilfe zu "Profile"

Häufig gestellte Fragen

In diesem Abschnitt erhalten Sie Antworten auf die Fragen, die Benutzer von "Profile" am häufigsten stellen.

Was ist "Profile"?
"Profile" ist ein Verzeichnis von Mitarbeitern in Ihrer Organisation. Sie finden dort Kontaktinformationen sowie Informationen zu Projekten, Know-how und Interessen dieser Personen.
Kann ich mein Profil ändern?
Sie können über die Seite Mein Profil bearbeiten Änderungen an Ihrem Profil vornehmen. Sie können Ihre Kontakt- und Hintergrundinformationen aktualisieren sowie eine neue Grafik- oder Aussprachedatei über die Optionen auf dieser Seite hochladen.
Was sind Tags?
Bei einem Tag handelt es sich um ein beschreibendes Schlüsselwort, das Sie Ihrem Profil oder dem Profil einer anderen Person zuordnen, um berufliche Aktivitäten, Know-how oder Interessen zu kennzeichnen.
Wie kann ich Personen suchen?
Sie können über eine einfache Suche nach Profilen suchen, um Personen anhand ihrer Kontaktinformationen, wie z. B. Name und Jobbezeichnung, oder anhand von erweiterten Informationen, wie z. B. Interessen, Erfahrung und beruflichen Projekten, zu suchen.
Wie richte ich mein eigenes Netzwerk ein?
Nachdem Sie die Personen gefunden haben, mit denen Sie in Kontakt treten möchten, fügen Sie sie zu Ihrem Netzwerk hinzu, um eine Reihe nützlicher Kontakte aufzubauen. Wenn Sie jemanden zu Ihrem Netzwerk hinzufügen, erhält derjenige per E-Mail eine Einladung in Ihr Netzwerk, die er annehmen oder ablehnen kann.
