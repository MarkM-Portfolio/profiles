Hilfe zu "Profile"

Audiodatei hochladen

Bearbeiten Sie Ihr Profil, um eine Aufnahme mit der richtigen Aussprache Ihres Namens hochzuladen.

Warum und wann dieser Vorgang ausgeführt wird

Erweitern Sie Ihr Profil, indem Sie eine Aufnahme hinzufügen, in der Sie Ihren Nachnamen aussprechen, so dass andere hören können, wie er richtig ausgesprochen wird.

Gehen Sie wie folgt vor, um eine Audiodatei in Ihr Profil hochzuladen:
  1. Klicken Sie auf der Seite Mein Profil bearbeiten auf Aussprache.
  2. Klicken Sie auf Durchsuchen, um nach einer Audiodateiaufnahme Ihres Namens, wie er in der richtigen Aussprache deutlich ausgesprochen wird, zu suchen. Die Datei darf nicht größer als 100 KB sein.
  3. Wählen Sie die Datei aus, die Sie hochladen möchten, und klicken Sie anschließend auf Öffnen.
  4. Klicken Sie auf Aktualisieren, um Ihre Änderungen zu übernehmen.
