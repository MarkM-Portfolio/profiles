Audiodatei erstellen

Erstellen Sie eine Aufnahme, in der Sie Ihren Nachnamen aussprechen, so dass andere hören können, wie er richtig ausgesprochen wird.

Warum und wann dieser Vorgang ausgeführt wird
Gehen Sie wie folgt vor, um eine Audiodatei auf einem Microsoft Windows 2000- oder Microsoft Windows XP-System zu erstellen:
  1. Wählen Sie Start > Alle Programme > Zubehör > Unterhaltungsmedien > Audiorecorder aus.
  2. Klicken Sie auf die Schaltfläche zum Aufnehmen, sprechen Sie Ihren vollständigen Namen so deutlich wie möglich aus, und klicken Sie dann auf die Schaltfläche zum Stoppen. Klicken Sie auf die Schaltfläche zur Wiedergabe, um die Aufnahme anzuhören.
  3. Prüfen Sie die Größe der Aufnahme, indem Sie auf Datei > Eigenschaften klicken. Die Datei muss kleiner als 100 KB sein.
  4. Um die Audiodatei erneut aufzuzeichnen, klicken Sie auf Datei > Neu, und wählen Sie Nein aus, wenn Sie gefragt werden, ob die Aufnahme gespeichert werden soll. Wiederholen Sie Schritt 2, um die Aufnahme erneut vorzunehmen. Beachten Sie die folgenden Hinweise, um die Qualität der Aufnahme zu verbessern:
    • Wenn die Aufnahme nicht hörbar ist, stellen Sie sicher, dass das Mikrofon aktiviert ist. Wählen Sie die Optionen Start > Alle Programme > Zubehör > Unterhaltungsmedien > Lautstärkereglung und dort die Optionen Optionen > Eigenschaften aus. Wählen Sie die Option Aufnahme aus, wählen Sie Mikrofon aus, und klicken Sie dann auf OK. Stellen Sie den Lautstärkeregler für das Mikrofon auf Maximum ein.
    • Wenn die Aufnahme undeutlich oder die erstellte Datei zu groß (über 100 KB) ist, konfigurieren Sie den Audiorecorder. Wählen Sie Start > Alle Programme > Zubehör > Unterhaltungsmedien > Audiorecorder aus. Wählen Sie die Optionen Datei > Eigenschaften aus, und klicken Sie dann auf Jetzt konvertieren. Wählen Sie im Feld Format den Eintrag PCM und im Feld Attribute den Eintrag 8.000 kHz, 8 Bit, Mono aus. Klicken Sie auf OK, und klicken Sie dann auf OK, um das Fenster "Eigenschaften" zu schließen.
  5. Wenn Sie eine Aufnahme erstellt haben, die hörbar, deutlich ausgesprochen und kleiner als 100 KB ist, wählen Sie die Optionen Datei > Speichern unter aus, um sie zu speichern. Geben Sie einen Namen für die Datei ein, und wählen Sie ein Verzeichnis aus, in dem sie gespeichert werden soll. Notieren Sie sich Name und Position, so dass Sie die Datei beim Upload in Ihr Profil abrufen können. Der Standarddateityp im Feld Speichern unter ist ".wav". Akzeptieren Sie die Standardeinstellung. Klicken Sie auf Speichern.
Nächste Maßnahme
Um eine Audiodatei auf einem Linux-System zu erstellen, verwenden Sie den Befehl "record" oder verwenden Sie ein Dienstprogramm zum Erstellen von Audioaufnahmen.
