Profile-Visitenkarte verwenden

Verwenden Sie die Profile-Visitenkarte, um die Kontaktinformationen zu einer Person anzuzeigen und um in Kontakt mit ihr zu treten.

Warum und wann dieser Vorgang ausgeführt wird
Die Profile-Visitenkarte bietet eine nützliche Kurzdarstellung der Profilinformationen zu einer Person. Wenn das IBM Lotus Connections-Plug-in für Sametime installiert ist, können Sie neben der Jobbezeichnung und den Kontaktinformationen zu einer Person über die Visitenkarte den aktuellen Sametime-Status dieser Person anzeigen.

Zudem bietet die Visitenkarte Links zu den Lotus Connections-Komponenten, die die Person verwendet, sowie die Möglichkeit, eine Reihe von Aktionen auszuführen. Wenn z. B. Ihr Administrator konfiguriert hat, dass E-Mail-Adressen angezeigt werden, können Sie eine E-Mail senden oder Sie können einen Instant-Messaging-Chat mit ihnen starten, wenn Sie das Lotus Connections-Plug-in für Sametime installiert haben.

Sie können auf die Visitenkarte einer Person zugreifen, indem Sie in einer beliebigen Lotus Connections-Komponente mit dem Mauszeiger über ihren Namen fahren und auf den angezeigten Link klicken.

Verwenden Sie die "Profile"-Visitenkarte, um die folgenden Tasks auszuführen:

  1. Greifen Sie auf die Lotus Connections-Komponenten, die einer Person zugeordnet sind, zu, indem Sie auf die Komponentenlinks oben in der Visitenkarte klicken.
  2. Senden Sie der Person eine E-Mail, indem Sie auf Mail senden klicken. Wenn Ihr Administrator Lotus Connections so konfiguriert hat, dass E-Mail-Adressen nicht angezeigt werden, ist diese Option nicht verfügbar.
  3. Starten Sie mit der Person einen Instant-Chat, indem Sie auf Chat klicken. Das Lotus Connections Plug-in für Sametime muss installiert sein, damit diese Option angezeigt wird.
  4. Starten Sie mit der Person eine Telefonkonferenz, indem Sie auf Anruf klicken. Sametime Advanced Telephony muss installiert sein, damit diese Option angezeigt wird.
  5. Fügen Sie die Person als Kollegen hinzu, indem Sie Weitere Aktionen > In mein Netzwerk einladen auswählen, eine Einladung eingeben und auf Einladung senden klicken.
  6. Laden Sie die vCard der Person herunter, und speichern Sie sie, indem Sie Weitere Aktionen > vCard herunterladen auswählen, auf Speichern klicken, eine Dateiadresse angeben und erneut auf Speichern klicken.
  7. Greifen Sie auf die Dateibibliothek einer Person in Lotus Quickr zu, indem Sie auf Dateien klicken. Diese Option ist nur verfügbar, wenn Ihr Administrator die Kombination mit Lotus Quickr aktiviert hat und für die Visitenkarte die Anzeige eines Links zu Lotus Quickr konfiguriert hat.
