Hilfe zu "Profile"

Personen zu Ihrem Netzwerk hinzufügen

Verwenden Sie "Profile", um ein Netzwerk aus Kollegen und nützlichen Kontakten zu erstellen.

Warum und wann dieser Vorgang ausgeführt wird
Sie können Personen zur Teilnahme an Ihrem Netz einladen, indem Sie ihnen eine Einladung senden. Wenn die betreffende Person Ihre Einladung erhält, kann sie sie annehmen oder ignorieren.
Anmerkung: Sie müssen sich nicht in "Profile" befinden, um jemanden zu Ihrem Netzwerk hinzuzufügen. Sie können über die Profile-Visitenkarte in jeder der IBM Lotus Connections-Komponenten direkt einen Kontakt zu einer Person herstellen. Wenn z. B. das Profil einer Person als Teil von Suchergebnissen zurückgegeben wird oder wenn Sie zur selben Community wie eine andere Person gehören, die Sie zu Ihrem Netzwerk hinzufügen möchten, können Sie dies über die Visitenkarte tun.

Gehen Sie wie folgt vor, um eine Person zu Ihrem Netzwerk hinzuzufügen:

  1. Öffnen Sie die Profilseite der Person, die Sie zu Ihrem Netzwerk hinzufügen möchten.
  2. Klicken Sie auf In mein Netzwerk einladen. Wenn Sie noch nicht bei der Komponente "Profile" angemeldet sind, werden Sie zur Anmeldung weitergeleitet.
  3. Optional: Geben Sie eine Nachricht ein, um die gewünschte Person einzuladen, an Ihrem Netzwerk teilzunehmen.
  4. Klicken Sie auf Einladung senden.
