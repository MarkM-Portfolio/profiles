Statusaktualisierungen überprüfen

Bleiben Sie über die Aktivitäten in Ihrem Netzwerk auf dem Laufenden, indem Sie die Statusaktualisierungen regelmäßig überprüfen.

Warum und wann dieser Vorgang ausgeführt wird
Im Bereich Statusaktualisierungen auf der Profile-Homepage werden die aktuellen Statusaktualisierungen aus verschiedenen Profilen angezeigt. Sie können die Anzeige filtern, um die Statusaktualisierungen von Personen anzuzeigen, mit denen Sie in Kontakt sind, oder um Ihre eigenen Statusaktualisierungen anzuzeigen. Sie können von dieser Seite aus auch Ihre persönlichen Statusnachrichten aktualisieren und die Statusnachrichten anderer Personen kommentieren.

Gehen Sie wie folgt vor, um Statusaktualisierungen zu überprüfen:

  1. Öffnen Sie die Profile-Homepage.
  2. Führen Sie im Bereich Statusaktualisierungen eine der folgenden Aktionen aus:
    • Klicken Sie auf Statusaktualisierungen in meinem Netzwerk, um die neueste Liste von Aktualisierungen von Personen in Ihrem Netzwerk anzuzeigen.
    • Klicken Sie auf Meine Statusaktualisierungen, um Ihre eigenen Statusaktualisierungen anzuzeigen.
  3. Optional: Klicken Sie auf Kommentar hinzufügen, um eine bestimmte Statusnachricht zu kommentieren.
