Hilfe zu "Profile"

Tags zu Personen hinzufügen

Verwenden Sie Schlüsselwörter, um zu Profilen von Personen mit für Sie interessantem Know-how und Fachwissen Tags hinzuzufügen.

Vorbereitungen
Wenn Ihr Systemadministrator das Hinzufügen von Tags zu Personen aktiviert hat, können Sie dies in Bezug auf sich selbst und andere Personen tun. Zum Erstellen oder Ändern von Tags müssen Sie angemeldet sein.
Warum und wann dieser Vorgang ausgeführt wird
Sie können Tags zu Personen hinzufügen, indem Sie das Tageintragsformular auf der entsprechenden Profilseite verwenden. Tags können nicht ausgeblendet werden oder privat sein. Benutzer können feststellen, wer sie getaggt hat, indem sie auf die Zahl neben einem bestimmten Tag klicken, um eine Liste der Personen anzuzeigen, die diesen Tag zu ihrem Profil hinzugefügt haben.

Gehen Sie wie folgt vor, um zu einer Person Tags hinzuzufügen:

  1. Öffnen Sie die Profilseite für die Person, zu der Sie Tags hinzufügen möchten.
  2. Geben Sie mindestens einen Tag in das Feld Tags in der linken Seitenleiste ein. Durch die Worterkennungsfunktion kann der Tag, den Sie eingeben, basierend auf der Ähnlichkeit zu bereits vorhandenen Tags erkannt werden.
  3. Klicken Sie auf die Option zum Hinzufügen von Tags zu diesem Profil, um die Tags zu dem Profil der Person hinzuzufügen.
