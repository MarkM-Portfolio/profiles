Hilfe zu "Profile"

Tags verwenden

Bei einem Tag handelt es sich um ein beschreibendes Schlüsselwort, das Sie Ihrem Profil zuordnen, um berufliche Aktivitäten, Know-how oder Interessen zu kennzeichnen. Wenn Ihr Systemadministrator das Hinzufügen von Tags zu Personen aktiviert hat, können Sie auch den Profilen anderer Personen Tags zuordnen.

Die Tags, die Sie Ihrem Profil zuordnen, werden im Bereich Tags Ihrer Seite Mein Profil angezeigt. Die Tags beschreiben Ihre Tätigkeiten und Interessen. Verwenden Sie beim Erstellen von Tags Schlüsselwörter, die Ihre Interessen oder Fachgebiete darstellen. Tags müssen als einzelnes Wort oder als mehrere durch einen Bindestrich oder ein Unterstreichungszeichen verbundene Wörter eingegeben werden. Gültige Tags sind z. B. Gehalt, Planung_von_Ereignissen und Personal-Abteilung.

Warum sollten Sie zu Ihrem Profil Tags hinzufügen?

Know-how bekannt machen

Wenn Sie Ihr Profil mit Tags versehen, beschreiben Sie Ihr Know-how, Ihren Aufgabenbereich, Ihre Interessen und Ihre Ausbildung für andere in Ihrer Organisation.

Personen suchen

Klicken Sie auf einen Ihrer eigenen Tags, um schnell andere Personen zu suchen, die ähnliche Interessen oder eine ähnliche Arbeit haben. Personen mit denselben Tags sind wahrscheinlich gute Ressourcen für Sie. Sie können den Profilen ihre Kontaktinformationen entnehmen und eine Zusammenarbeit mit diesen Kollegen beginnen.

Sie können feststellen, wer Sie getaggt hat. Klicken Sie auf die Häufigkeitszahl für einen bestimmten Tag, um eine Liste der Personen anzuzeigen, die Sie mit diesem Begriff getaggt haben. Klicken Sie erneut auf die Häufigkeit, um die Informationen auszublenden.

Warum sollten Sie zu dem Profil einer anderen Person Tags hinzufügen?

Ihre Verbindung definieren

Durch das Hinzufügen von Tags zu dem Profil eines Kollegen können Sie für sich kennzeichnen, welche Verbindung sie zu dieser Person haben. Wenn Sie z. B. mit einer Person an einem bestimmten Projekt zusammengearbeitet haben, möchten Sie zu dieser Person möglicherweise den Namen des Projekts als Tag hinzufügen. Wenn Sie zu dem Profil einer anderen Person einen Tag hinzufügen, ist dieser Tag für jeden sichtbar, der das Profil anzeigt.

Indem Sie zu Ihrem eigenen Profil und zu Profilen anderer Personen Tags hinzufügen, erleichtern Sie sich selbst und Ihren Kollegen das Suchen von Fachwissen in Ihrem Unternehmen. Alle in der Komponente "Profile" hinzugefügten Tags werden im Bereich Organisationstags auf der Seite Profile-Homepage angezeigt. Sie können auf einfache Weise Fachleute für ein bestimmtes Gebiet suchen, indem Sie die Tagwolke durchsuchen und auf Tags klicken, die Sie interessieren. Sie können auch nach Profilen suchen, die einen bestimmten Tag enthalten, indem Sie auf Tags suchen klicken, den Tag eingeben, nach dem Sie suchen, und dann auf Profile suchen, die diesen Tag beinhalten klicken.
