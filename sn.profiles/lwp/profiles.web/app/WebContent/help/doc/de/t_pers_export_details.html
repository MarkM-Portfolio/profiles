Hilfe zu "Profile"

Kontaktinformationen exportieren

Exportieren Sie die vCard einer Person, um diese als neuen Kontakt hinzuzufügen.

Warum und wann dieser Vorgang ausgeführt wird
Bei einer vCard (virtual Card, virtuelle Karte) handelt es sich um eine Datei, in der grundlegende Kontaktinformationen zu einem Benutzer nach der Spezifikation des Internet Mail Consortium angegeben sind. Sie können neue Kontakte hinzufügen, indem Sie über Ihr E-Mail-System vCard-Dateien importieren. Sie können die vCard auch von der Profile-Visitenkarte herunterladen.

Beim Herunterladen einer vCard müssen Sie die Zeichencodierung auswählen, die für Ihr Adressbuchleseprogramm geeignet ist. Die Standardoption ist International (UTF-8). Sie ist für Benutzer von IBM Lotus Notes und Microsoft Outlook geeignet. Möglicherweise müssen Sie jedoch eine andere Zeichencodierung auswählen, wenn es sich bei Ihrem E-Mail-Client um Microsoft Outlook Express oder Microsoft Windows Address Book handelt.

Gehen Sie wie folgt vor, um die vCard eines Kollegen in Ihren persönlichen Datensätzen zu speichern.
  1. Öffnen Sie die Profilseite für die Person, die Sie als Kontakt hinzufügen möchten.
  2. Klicken Sie auf vCard herunterladen.
  3. Wählen Sie die Zeichencodierung aus, die Sie verwenden möchten.
  4. Klicken Sie auf Herunterladen, um die vCard der Person an einer angegebenen Position zu speichern.
