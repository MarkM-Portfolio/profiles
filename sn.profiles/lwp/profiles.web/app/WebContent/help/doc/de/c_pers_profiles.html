Hilfe zu "Profile"

Profile anzeigen

Ein Profil enthält Kontaktinformationen und arbeitsbezogene Informationen zu einer Person.

Im Profil einer Person finden Sie die folgenden Informationen:

Kontaktinformationen
Die Informationen, die in der Regel auf einer Visitenkarte zu finden sind, wie z. B. Name, Jobbezeichnung, Unternehmen, Standort, Telefonnummern und E-Mail-Adressen. Zudem sind im Profil die Ortszeit der Person sowie ggf. die Kontaktinformationen eines Assistenten angegeben. Klicken Sie auf das Visitenkartensymbol, um die Kontaktinformationen der Person als vCard-Datei zu exportieren. Bei einer vCard (virtual Card, virtuelle Karte) handelt es sich um eine Datei, in der grundlegende Kontaktinformationen nach der Spezifikation des Internet Mail Consortium angegeben sind. Sie können neue Kontakte hinzufügen, indem Sie über Ihr E-Mail-System vCard-Dateien importieren.
Anmerkung: Ihr Administrator kann IBM Lotus Connections so konfigurieren, dass E-Mail-Adressen nicht angezeigt werden. In diesem Fall werden E-Mail-Adressen nicht im Profil oder auf der Visitenkarte einer Person angezeigt.
Wenn die Sametime- Anwesenheitsanzeige aktiviert ist, wird der Onlinestatus der Person angezeigt. Das Lotus Connections-Plug-in für Sametime muss installiert sein, damit diese Information angezeigt wird.
Aussprache
Der Link "Aussprache" wird angezeigt, wenn der Profileigentümer eine Audiodatei erstellt und sie in das Profil hochgeladen hat. Klicken Sie auf den Link, um eine Aufnahme des Namens der Person mit der richtigen Aussprache wiederzugeben.
Berichtsstruktur
Wählen Sie eine der folgenden Anzeigeoptionen aus:
Tabelle 1. Berichtsstrukturanzeigen
Anzeige Beschreibung
Vollständige Berichtskette Zeigt die Position des Eigentümers des Profils vom Profileigentümer bis zur Spitze des Organigramms an.
Derselbe Manager Zeigt die Personen an, die ebenfalls an den Manager des Profileigentümers berichten.
Unterstellte Personen Zeigt die Position des Profileigentümers innerhalb des gesamten Organigramms, einschließlich der Personen, die dem Profileigentümer unterstellt sind, an.
Über mich
Zeigt Informationen zur derzeitigen Berufserfahrung, zur fachlichen Qualifikation, zu Sprachen, zu Interessen, zur bisherigen Berufserfahrung und zur Ausbildung an. Hierbei handelt es sich um ein vom Profileigentümer zusammengestelltes Rich-Text-Feld, das alle Informationen enthält, die der Profileigentümer für die gemeinsame Nutzung freigeben möchte.
Board
Die letzten Beiträge zum Board des Profileigentümers. Diese Informationen enthalten die letzten Statusnachrichten des Profileigentümers sowie Kommentare, die andere Benutzer im Board des Profileigentümers veröffentlicht haben.
Neue Beiträge
Dieser Abschnitt verwendet Ansichten mit Registerkarten zum Anzeigen der letzten Aktualisierungen, die der Profileigentümer an anderen Lotus Connections-Komponenten vorgenommen hat.
Profiltags
Kurze Schlüsselwörter zur Angabe von Know-how, Interessengebieten oder beruflichen Aktivitäten des Profileigentümers. Diese Tags können vom Profileigentümer oder von einer anderen Person hinzugefügt werden.
Tools für Onlinezusammenarbeit
Wenn Profile in andere Tools für Onlinezusammenarbeit, wie z. B. IBM Lotus Sametime, integriert sind, können Sie die Präsenzdaten des Profileigentümers sehen, in denen angegeben ist, ob die Person zum Chatten verfügbar ist.
Mein Netzwerk
In diesem Abschnitt wird eine Auswahl der Personen im Netzwerk des Profileigentümers angezeigt. Sie können auf die Option Alle anzeigen klicken, um eine Liste aller Personen im Netzwerk einer Person anzuzeigen.
Meine Links
Externe Links, die von Profileigentümern in ihr Profil aufgenommen werden.
