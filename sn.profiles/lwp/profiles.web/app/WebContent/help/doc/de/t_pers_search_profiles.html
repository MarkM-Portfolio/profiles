Hilfe zu "Profile"

Erweiterte Suche durchführen

Führen Sie eine erweiterte Suche durch, um Profile anhand von Kontaktinformationen, Standort, Tags, Erfahrung, Fachwissen, Schulungen oder Interessen zu suchen. Sie können von einer beliebigen Seite in "Profile" aus eine erweiterte Suche durchführen.

Warum und wann dieser Vorgang ausgeführt wird

Bei einer erweiterten Suche werden die Ergebnisse dadurch gefiltert, dass gleichzeitig nach mehreren Profilfeldern gesucht werden kann. Diese Suche ist zielgerichteter als die Schnellsuche, bei der jeweils nur ein Feld gesucht wird. Je mehr Werte Sie in diesen Feldern eingeben, desto mehr werden die Suchergebnisse eingegrenzt.

Gehen Sie wie folgt vor, um eine erweiterte Suche in "Profile" durchzuführen:

  1. Wählen Sie aus dem Suchsteuerelement in der Titelleiste die Option Erweiterte Suche in "Profile" aus.
  2. Der Administrator kann Felder zum Formular Erweiterte Profilsuche hinzufügen oder daraus entfernen, je nach Bedarf. Aufgrund dieser Funktionalität können die im Formular angezeigten Felder variieren. Wenn Sie basierend auf den Kontaktinformationen nach Personen suchen möchten, geben Sie einen Wert in ein oder mehrere der folgenden Felder ein:
    Tabelle 1. Felder für erweiterte Suche
    Feldname Einzugebende Wörter
    Anzeigename Geben Sie den vollständigen Namen der Person, nach der Sie suchen möchten, ein.
    Vorname Geben Sie einen Vornamen ein.
    Nachname Geben Sie einen Nachnamen ein.
    Tags Geben Sie Schlüsselwörter ein, um basierend auf Tags nach Personen zu suchen.
    Jobbezeichnung Geben Sie eine Jobbezeichnung ein, um Profile von Personen mit diesem Aufgabenbereich zu finden.
    Abteilung Geben Sie ein Schlüsselwort ein, um nach einer Person, die in einer bestimmten Abteilung arbeitet, zu suchen.
    Über mich Geben Sie Schlüsselwörter ein, die den Typ von Know-how und Erfahrung beschreiben, nach dem Sie suchen.
    Hintergrund Geben Sie Schlüsselwörter ein, die die Art von Hintergrund beschreiben, an dem Sie interessiert sind.
    Organisation oder Unternehmen Geben Sie den Namen einer Organisation oder eines Unternehmens ein, um Profile von Mitarbeitern zu finden.
    Stadt Geben Sie eine Stadt ein, um die Profile von Personen zu suchen, die an diesem Standort arbeiten.
    Bundesland Geben Sie ein Bundesland ein, um die Profile von Personen zu suchen, die an diesem Standort arbeiten.
    Land Geben Sie ein Land ein, um Profile von Personen zu finden, die an diesem Standort arbeiten.
    E-Mail-Adresse Geben Sie eine geschäftliche E-Mail-Adresse ein. Diese Option ist nicht verfügbar, wenn aufgrund der Konfiguration keine E-Mail-Adressen angezeigt werden.
    Telefonnummer Geben Sie eine Telefonnummer ein. Durch dieses Feld werden alle fünf Telefonnummernfelder im Profil einer Person durchsucht.
  3. Klicken Sie auf Suche.
