Hilfe zu "Profile"

Neuerungen in dieser Version

In diesem Abschnitt erhalten Sie Informationen zu den neuen und aktualisierten Funktionen, die Ihnen die Verwendung von Profilen erleichtern, damit Sie auf einfache Weise andere Personen erreichen und sich ihnen vorstellen können.

Lotus Connections 2.0.1

  • Durch Verbesserungen im Sicherheitsbereich kann Ihr Administrator für die "Profile"-Visitenkarte die Option E-Mail-Adressen nicht mehr anzeigen festlegen.

Lotus Connections 2.0

  • Durch die erweiterte Funktionalität für das Hochladen von Grafiken können Sie eine größere Grafik in Ihr Profil hochladen und diese Grafik so zuschneiden, dass sie Ihren Vorstellungen entspricht.
  • Durch Verbesserungen bei der "Profil"-Visitenkarte erhalten Sie eine Reihe von Möglichkeiten, um mit Ihren Kollegen zu interagieren. So können Sie z. B. den Sametime-Status Ihrer Kollegen überprüfen, sie zu Ihrem Netz hinzufügen und einen Instant-Chat mit ihnen starten.
  • Durch verbessertes Tagging können Sie zu anderen Personen Tags mit Schlüsselwörtern hinzufügen. Auf diese Weise können Sie deren Verbindung mit Ihnen definieren und diese Personen einfach finden.
  • Über die Informationen zur Berichtskette erfahren Sie nun, welche Mitarbeiter ein- und demselben Manager haben und wer wessen Manager ist.
  • Durch die verbesserte Suchfunktionalität können Sie in der Reihenfolge nach dem Vornamen oder dem Nachnamen suchen.
