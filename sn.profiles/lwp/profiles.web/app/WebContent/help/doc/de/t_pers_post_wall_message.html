Hilfe zu "Profile"

Nachrichten im Board veröffentlichen

Sie können direkt mit den Personen in Ihrem Netzwerk kommunizieren, indem Sie Nachrichten oder Kommentare im Bereich Board ihres Profils veröffentlichen.

Vorbereitungen
Sie müssen bei "Profile" angemeldet sein, um eine Nachricht in Ihrem eigenen Profil oder im Profil einer anderen Person zu veröffentlichen.
Warum und wann dieser Vorgang ausgeführt wird
Alle Benutzer von "Profile" verfügen auf ihrer Profilseite über den Bereich Board, in dem die letzten Statusnachrichten sowie Aktualisierungen für die IBM Lotus Connections-Komponenten angezeigt werden. Sie können mit Ihren Kollegen kommunizieren, indem Sie direkt in deren Profil eine Nachricht veröffentlichen.

Gehen Sie wie folgt vor, um eine Nachricht im Board einer anderen Person zu veröffentlichen:

  1. Klicken Sie auf der Profilseite der Person in das Nachrichtenfeld oben im Bereich Board, in dem der Text Nachricht in diesem Profilboard schreiben angezeigt wird.
  2. Geben Sie Ihre Nachricht ein.
  3. Klicken Sie auf Nachricht veröffentlichen.
Ergebnisse
Ihre Nachricht wird oben im Bereich Board angezeigt, in dem der Profileigentümer und andere Benutzer darauf antworten können. Der Profileigentümer empfängt eine E-Mail, die ihn darüber informiert, dass eine neue Nachricht in seinem Profil veröffentlicht wurde.
