Personen aus Ihrem Netzwerk entfernen

Wenn eine Person in Ihrer Organisation versetzt wurde oder die Organisation verlassen hat, können Sie sie aus Ihrem Netzwerk entfernen.

Warum und wann dieser Vorgang ausgeführt wird
Gehen Sie wie folgt vor, um einen Kollegen aus Ihrem Netzwerk zu löschen:
  1. Klicken Sie auf der Seite Mein Profil im Bereich Mein Netzwerk auf Alle anzeigen.
  2. Wählen Sie das Kontrollkästchen neben der Person aus, die Sie entfernen möchten.
  3. Klicken Sie auf Aus "Mein Netzwerk" entfernen.
    Anmerkung: Die Person erhält keine Benachrichtigung darüber, dass sie aus Ihrem Netzwerk entfernt wurde.
