Hilfe zu "Profile"

Profile durchsuchen

Sie können Profile durchsuchen, um Personen anhand ihrer Kontaktinformationen, wie z. B. Name und Jobbezeichnung, oder anhand von erweiterten Informationen, wie z. B. Interessen, Erfahrung und beruflichen Projekten, zu suchen.

Es gibt eine Reihe von unterschiedlichen Suchtypen, die in "Profile" zur Auswahl stehen:
  • Verwenden Sie das Suchsteuerelement, um eine Basissuche auf der Grundlage eines einzelnen Suchfeldes durchzuführen. Dieses Steuerelement ist oben rechts auf jeder Seite in "Profile" verfügbar.
  • Verwenden Sie die Optionen Profile nach Namen und Profile nach Schlüsselwort, die über das Steuerelement für die Schnellsuche verfügbar sind, um eine Person anhand ihres Namens oder anhand eines Schlüsselworts zu suchen.
  • Klicken Sie auf Erweiterte Suche in "Profile", um die erweiterten Suchoptionen zu verwenden und eine zielgerichtetere Suche, basierend auf der Grundlage mehrerer Suchfeldern, durchzuführen.
  • Verwenden Sie die Tagwolke auf der Profile-Homepage, um alle Profiltags für Ihre Organisation zu durchsuchen. Klicken Sie auf einen Tag, um eine Liste der Personen mit diesem Tag sowie zugehörige Tags anzuzeigen, die Ihnen beim Optimieren der Suche helfen. Um nach Profilen zu suchen, die einen bestimmten Tag enthalten, klicken Sie auf Tags suchen, geben Sie einen Tag in das bereitgestellte Feld ein und klicken Sie auf Profile suchen, die diesen Tag beinhalten.

Bei allen Suchtypen wird das Platzhalterzeichen (*) unterstützt. Wenn Sie z. B. die Zeichenfolge Ethan* als Suchbegriff für Namen eingeben, erhalten Sie alle Profile von Personen mit dem Vornamen Ethan als Ergebnis angezeigt. Wenn Sie 555* als Suchbegriff für Telefonnummern eingeben, erhalten Sie alle Telefonnummern, die mit 555 beginnen.
