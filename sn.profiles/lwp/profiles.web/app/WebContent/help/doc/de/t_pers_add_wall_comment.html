Hilfe zu "Profile"

Kommentare im Board veröffentlichen

Stellen Sie einen Kontakt zu Personen in Ihrem Netzwerk her, indem Sie eine Statusnachricht einer anderen Person oder eine Nachricht, die im Bereich Board des Profils einer Person veröffentlicht wurde, kommentieren.

Vorbereitungen
Sie müssen bei "Profile" angemeldet sein, um einen Kommentar zu Ihrem eigenen Profil oder zum Profil einer anderen Person zu veröffentlichen.
Warum und wann dieser Vorgang ausgeführt wird
Die Funktion "Board" ermöglicht es Ihnen, dynamisch mit den Personen in Ihrem Netzwerk zu interagieren, indem Sie Nachrichten in deren Profilen veröffentlichen sowie Nachrichten, die in Ihrem eigenen Profil veröffentlicht wurden, kommentieren. Indem Sie auf Nachrichten einer anderen Person antworten, können Sie bei Ereignissen in Ihrem Netzwerk zusammenarbeiten und einen Konversationsthread, der für andere sichtbar ist, erstellen.

Gehen Sie wie folgt vor, um einen Kommentar zum Bereich Board des Profils einer Person hinzuzufügen:

  1. Suchen Sie die Nachricht oder den Kommentar, auf die oder den Sie antworten möchten, und führen Sie eine der folgenden Aktionen aus:
    • Wenn Sie auf eine Nachricht antworten, klicken Sie unter der Nachricht auf Kommentar hinzufügen.
    • Wenn Sie auf einen Kommentar antworten, klicken Sie unter dem Kommentar auf Weiteren Kommentar schreiben.
  2. Geben Sie Ihren Kommentar im bereitgestellten Feld ein und klicken Sie dann auf Kommentar veröffentlichen.
Ergebnisse
Ihr Kommentar wird direkt unter der Nachricht, die Sie kommentiert haben, angezeigt. Der Profileigentümer empfängt eine E-Mail, die ihn darüber informiert, dass ein neuer Kommentar in seinem Profil veröffentlicht wurde.
